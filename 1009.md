# EOJ-1009 字串排序 - 完整解析

---

## 一、题目理解

### 问题描述

实现一个特殊的字符串比较函数 `strnumcmp`，规则如下：

1. **含数字的字符串 > 不含数字的字符串**
   - 例如：`"abc1" > "abcd"`（因为 abc1 含数字，abcd 不含）

2. **都含数字时，比较数字大小**
   - 例如：`"abc2" > "abc10"`？不对！
   - 实际上比较的是**数字部分的大小**，不是字典序
   - 所以：`"abc10" > "abc2"`（因为 10 > 2）

3. **数字部分相同，或都不含数字时，按原 strcmp 比较**
   - 例如：`"abc" < "abc#"`（按 ASCII 比较）

4. **数字在一个字符串中是连续的**
   - 例如：`"abc123def"` 中的数字是连续的 `123`

### 关键区别

| 比较方式 | "abc2" vs "abc10" | 结果 |
|---------|-------------------|------|
| 普通 strcmp | 逐字符比较：'2' < '1'? 实际上 '2' > '1' | "abc10" < "abc2" |
| strnumcmp | 提取数字：2 vs 10 | "abc2" < "abc10" |

---

## 二、比较规则详解

### 规则优先级

```
1. 是否含数字
   - 含数字的字符串 > 不含数字的字符串

2. 都含数字 → 比较数字大小
   - 数字大的字符串 > 数字小的字符串
   - 注意：数字部分可能有多位，要整体比较

3. 数字相同 或 都不含数字 → 按普通 strcmp 比较
   - 使用字典序（ASCII 码顺序）
```

### 示例分析

```
输入：abc# abc1 abc10 abcd abc2 abc
输出：abc abc# abcd abc1 abc2 abc10
```

**排序过程：**

| 字符串 | 含数字？ | 提取数字 | 分类 |
|--------|---------|---------|------|
| abc | ❌ | - | 无数字组 |
| abc# | ❌ | - | 无数字组 |
| abcd | ❌ | - | 无数字组 |
| abc1 | ✅ | 1 | 有数字组 |
| abc2 | ✅ | 2 | 有数字组 |
| abc10 | ✅ | 10 | 有数字组 |

**排序：**
1. 无数字组按字典序：`abc < abc# < abcd`
2. 有数字组按数字大小：`1 < 2 < 10` → `abc1 < abc2 < abc10`
3. 有数字组整体大于无数字组

结果：`abc abc# abcd abc1 abc2 abc10` ✓

## 三、解题思路

### 核心步骤

```
1. 读取所有字符串
2. 对每个字符串：
   a. 判断是否含数字
   b. 如果有数字，提取连续的数字作为整体
3. 排序：
   a. 优先按 是否含数字 排序（不含数字 < 含数字）
   b. 都含数字：按数字大小排序
   c. 数字相同或都不含数字：按 strcmp 排序
4. 输出
```

### 提取数字的方法

```cpp
string extractNumber(const string& s) {
    string num = "";
    for (char ch : s) {
        if (ch >= '0' && ch <= '9') {
            num += ch;
        }
    }
    return num;
}
```

**注意：** 题目说"数字在一个字符串中是连续的"，所以提取所有数字字符就是完整的数字。

## 四、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

// 提取字符串中的数字部分
string extractNumber(const string& s) {
    string num = "";
    for (char ch : s) {
        if (ch >= '0' && ch <= '9') {
            num += ch;
        }
    }
    return num;
}

// 判断字符串是否包含数字
bool hasDigit(const string& s) {
    for (char ch : s) {
        if (ch >= '0' && ch <= '9') {
            return true;
        }
    }
    return false;
}

// 自定义比较函数
bool strnumcmp(const string& a, const string& b) {
    bool hasA = hasDigit(a);
    bool hasB = hasDigit(b);
    
    // 规则1：含数字的 > 不含数字的
    if (hasA != hasB) {
        return hasB;  // 不含数字的排在前面
    }
    
    // 都含数字：比较数字大小
    if (hasA && hasB) {
        string numA = extractNumber(a);
        string numB = extractNumber(b);
        
        // 数字长度不同：长度长的数字更大（去前导零后）
        // 或者直接用数值比较（用 long long 防止溢出）
        if (numA.size() != numB.size()) {
            return numA.size() < numB.size();
        }
        // 长度相同：字典序比较（这时等同于数值比较）
        if (numA != numB) {
            return numA < numB;
        }
    }
    
    // 规则3：数字相同 或 都不含数字 → 普通 strcmp
    return a < b;
}

int main() {
    vector<string> strings;
    string s;
    
    // 读取所有字符串（直到 EOF）
    while (cin >> s) {
        strings.push_back(s);
    }
    
    // 排序
    sort(strings.begin(), strings.end(), strnumcmp);
    
    // 输出
    for (int i = 0; i < (int)strings.size(); i++) {
        if (i > 0) cout << " ";
        cout << strings[i];
    }
    cout << endl;
    
    return 0;
}
```

## 六、特殊情况处理

### 1. 数字长度为 8 位

```cpp
// 数字可能长达 8 位，用字符串比较更安全
// 而不是用 stoi 转成整数
```

### 2. 字符串包含前导零

```cpp
// 例如："a0" 和 "a00"
// 提取数字："0" 和 "00"
// 长度不同："0" < "00"
// 但实际数值 0 = 0
// 不过题目中数字不会以 0 开头（除了单独一个 0）
// 所以可以忽略这个问题
```

### 3. 只含数字的字符串

```cpp
// 例如："123" 和 "45"
// hasDigit 都为 true
// 提取数字："123" 和 "45"
// 比较："123" > "45"（因为长度 3 > 2）
// 正确：123 > 45
```

## 七、可读性更好的写法（使用结构体）

```cpp
#include <bits/stdc++.h>
using namespace std;

struct StringInfo {
    string str;
    string num;   // 提取的数字
    bool hasNum;  // 是否含数字
};

bool compare(const StringInfo& a, const StringInfo& b) {
    // 规则1：含数字的 > 不含数字的
    if (a.hasNum != b.hasNum) {
        return b.hasNum;  // 不含数字的排在前面
    }
    
    // 规则2：都含数字，比较数字大小
    if (a.hasNum && b.hasNum) {
        if (a.num.size() != b.num.size()) {
            return a.num.size() < b.num.size();
        }
        if (a.num != b.num) {
            return a.num < b.num;
        }
    }
    
    // 规则3：普通 strcmp
    return a.str < b.str;
}

int main() {
    vector<StringInfo> arr;
    string s;
    
    while (cin >> s) {
        StringInfo info;
        info.str = s;
        info.num = "";
        info.hasNum = false;
        
        for (char ch : s) {
            if (ch >= '0' && ch <= '9') {
                info.hasNum = true;
                info.num += ch;
            }
        }
        arr.push_back(info);
    }
    
    sort(arr.begin(), arr.end(), compare);
    
    for (int i = 0; i < (int)arr.size(); i++) {
        if (i > 0) cout << " ";
        cout << arr[i].str;
    }
    cout << endl;
    
    return 0;
}
```

## 八、关键点总结

| 要点 | 说明 |
|------|------|
| **含数字 > 不含数字** | 最高优先级 |
| **数字比较** | 整体比较，不是逐字符 |
| **数字长度比较** | 长度长的数字大 |
| **数字相同/都不含** | 普通 strcmp |
| **读取方式** | `while(cin >> s)` 读到 EOF |
| **数据类型** | 数字用字符串存储，避免溢出 |
