## 完整代码（先贴出来方便对照）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;

    int ans = 0;
    int i = 0;
    int n = s.size();

    while (i < n) {
        // 跳过非数字
        if (!isdigit(s[i])) {
            i++;
            continue;
        }

        // 找到数字子串的起始位置
        int start = i;
        while (i < n && isdigit(s[i])) {
            i++;
        }
        // 数字子串范围：[start, i)

        // 去掉前导0，找到第一个非零数字的位置
        int j = start;
        while (j < i && s[j] == '0') {
            j++;
        }

        // 有效长度 = i - j（从第一个非零数字到末尾）
        int len = i - j;
        ans = max(ans, len);
    }

    cout << ans << endl;
    return 0;
}
```

### 6. 跳过非数字字符（字母）
```cpp
        if (!isdigit(s[i])) {
            i++;
            continue;
        }
```
- `isdigit(s[i])`：判断 `s[i]` 是否是数字字符（'0'~'9'）
- `!isdigit(s[i])`：如果不是数字（即字母），就跳过
- `i++`：移动到下一个字符
- `continue`：跳过本次循环剩下的代码，继续下一次循环

**示例**：
```
s = "abc1234abc12"
i=0, s[0]='a' → 不是数字 → i=1
i=1, s[1]='b' → 不是数字 → i=2
i=2, s[2]='c' → 不是数字 → i=3
i=3, s[3]='1' → 是数字 → 进入下面的代码
```

---

### 7. 记录数字子串的起始位置
```cpp
        int start = i;
```
- 当遇到数字时，记录当前数字子串的起始位置
- `start` 指向数字子串的第一个字符

**示例**：
```
s = "abc1234abc12"
i=3, s[3]='1' → start=3
```

---

### 8. 找到数字子串的结束位置
```cpp
        while (i < n && isdigit(s[i])) {
            i++;
        }
```
- 循环条件：`i` 没有超出范围，且 `s[i]` 是数字
- 循环体：`i++` 移动到下一个字符
- 循环结束后，`i` 指向数字子串**后面第一个非数字字符**
- 数字子串范围是 `[start, i)`

**示例**：
```
s = "abc1234abc12"
start=3, i=3
循环：
  i=3, s[3]='1' → 是数字 → i=4
  i=4, s[4]='2' → 是数字 → i=5
  i=5, s[5]='3' → 是数字 → i=6
  i=6, s[6]='4' → 是数字 → i=7
  i=7, s[7]='a' → 不是数字 → 退出循环
此时：i=7, 数字子串范围 [3, 7) = "1234"
```

### 9. 注释：数字子串范围
```cpp
        // 数字子串范围：[start, i)
```
- 这是一个左闭右开区间 `[start, i)`
- 包含 `s[start]`，不包含 `s[i]`

---

### 10. 去掉前导 0
```cpp
        int j = start;
        while (j < i && s[j] == '0') {
            j++;
        }
```
- `j` 从 `start` 开始
- 循环条件：`j` 还没有超出数字子串范围，且 `s[j]` 是 `'0'`
- 每遇到一个 `'0'`，`j++` 向右移动
- 循环结束后，`j` 指向**第一个非零数字**的位置
- 如果全是 0，`j` 会等于 `i`

**示例1**：`"1234"`
```
start=3, i=7
j=3, s[3]='1' → 不是0 → 退出循环
j=3（指向 '1'）
有效数字从位置3开始
```

**示例2**：`"00123"`
```
s = "ABC00123ABC12"
找到数字子串 "00123"：start=3, i=8
j=3, s[3]='0' → 是0 → j=4
j=4, s[4]='0' → 是0 → j=5
j=5, s[5]='1' → 不是0 → 退出循环
j=5（指向 '1'）
有效数字从位置5开始
```

**示例3**：`"000"`
```
找到数字子串 "000"：start=0, i=3
j=0, s[0]='0' → j=1
j=1, s[1]='0' → j=2
j=2, s[2]='0' → j=3
j=3, 不满足 j < i → 退出循环
j=3，等于 i
有效数字从位置3开始（没有有效数字）
```

---

### 11. 计算有效长度
```cpp
        int len = i - j;
```
- `len = i - j`
- `i`：数字子串的结束位置
- `j`：第一个非零数字的位置
- 有效长度 = 从第一个非零数字到子串末尾的字符数

**示例1**：`"1234"`
```
i=7, j=3
len = 7-3 = 4
```

**示例2**：`"00123"`
```
i=8, j=5
len = 8-5 = 3
```

**示例3**：`"000"`
```
i=3, j=3
len = 3-3 = 0
```

### 12. 更新答案
```cpp
        ans = max(ans, len);
```
- `max(ans, len)`：取 `ans` 和 `len` 中的较大值
- 更新 `ans` 为当前最大值
