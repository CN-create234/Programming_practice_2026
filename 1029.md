# EOJ-1029 括号字符串的分数 - 栈方法逐行详解

---

## 一、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;

    stack<int> st;
    st.push(0);

    for (char ch : s) {
        if (ch == '(') {
            st.push(0);
        } else {
            int inner = st.top();
            st.pop();

            if (inner == 0) {
                st.top() += 1;
            } else {
                st.top() += 2 * inner;
            }
        }
    }

    cout << st.top() << endl;

    return 0;
}
```

## 二、逐行详细解释

### 第一部分：头文件和主函数

```cpp
#include <bits/stdc++.h>
using namespace std;
```

| 代码 | 含义 |
|------|------|
| `#include <bits/stdc++.h>` | 万能头文件，包含所有 C++ 标准库 |
| `using namespace std` | 使用标准命名空间 |

---

```cpp
int main() {
    string s;

```

| 行 | 代码 | 含义 |
|----|------|------|
| 1 | `int main()` | 主函数入口 |
| 2 | `string s` | 声明字符串变量 `s` |
| 3 | `cin >> s` | 从标准输入读取括号字符串 |

---

### 第二部分：初始化栈

```cpp
    stack<int> st;
    st.push(0);
```

| 行 | 代码 | 含义 |
|----|------|------|
| 1 | `stack<int> st` | 创建一个整数栈 `st` |
| 2 | `st.push(0)` | 压入初始值 0，作为最终结果的容器 |

**为什么初始压入 0？**

栈的底层元素用来存储**最终的总分数**。在计算过程中，我们通过修改 `st.top()` 来累加分数。

---

### 第三部分：遍历字符串

```cpp
    for (char ch : s) {
```

- 遍历字符串 `s` 中的每个字符
- `ch` 依次取 `'('` 或 `')'`

---

### 第四部分：遇到左括号

```cpp
        if (ch == '(') {
            st.push(0);
        }
```

| 代码 | 含义 |
|------|------|
| `if (ch == '(')` | 如果当前字符是左括号 |
| `st.push(0)` | 压入 0，表示新的括号对开始，分数暂为 0 |

**栈状态示例：**

```
输入：(())

处理第一个 '(' → 栈：[0, 0]
处理第二个 '(' → 栈：[0, 0, 0]
```

---

### 第五部分：遇到右括号

```cpp
        } else {
            int inner = st.top();
            st.pop();
```

| 行 | 代码 | 含义 |
|----|------|------|
| 1 | `} else {` | 当前字符是右括号 `')'` |
| 2 | `int inner = st.top()` | 获取栈顶元素（当前括号对内部的分数） |
| 3 | `st.pop()` | 弹出栈顶元素，移除当前括号对 |

**示例：**

```
栈：[0, 0, 0]
遇到 ')'
inner = 0
弹出后栈：[0, 0]
```

---

### 第六部分：计算括号对分数

```cpp
            if (inner == 0) {
                st.top() += 1;
            } else {
                st.top() += 2 * inner;
            }
```

| 代码 | 含义 |
|------|------|
| `if (inner == 0)` | 空括号对 `()` → 分数为 1 |
| `st.top() += 1` | 将 1 加到外层括号的累计分数上 |
| `else` | 嵌套括号 `(A)` → 分数为 `2 × inner` |
| `st.top() += 2 * inner` | 将 `2 × inner` 加到外层括号的累计分数上 |

**重点理解：** `st.top()` 现在指向的是**外层括号**的累计分数。我们将当前括号对的分数加到外层。

---

### 第七部分：输出结果

```cpp
    cout << st.top() << endl;

    return 0;
}
```

- 遍历结束后，栈中只剩一个元素：总分数
- `st.top()` 即为结果

## 三、完整执行示例：`(())`

### 初始状态

```
字符串: ( ( ) )
栈: [0]
```

### 第1步：处理第一个字符 `'('`

```
ch = '('
执行：st.push(0)
栈: [0, 0]
```

### 第2步：处理第二个字符 `'('`

```
ch = '('
执行：st.push(0)
栈: [0, 0, 0]
```

### 第3步：处理第三个字符 `')'`

```
ch = ')'
inner = st.top() = 0
st.pop() → 栈: [0, 0]
inner == 0 → st.top() += 1
栈: [0, 1]
```

**图解：**

```
     栈            说明
    [0, 0, 0]   初始
         ↑
         inner=0（空括号 ()）
    [0, 1]       加1到外层
```

### 第4步：处理第四个字符 `')'`

```
ch = ')'
inner = st.top() = 1
st.pop() → 栈: [0]
inner != 0 → st.top() += 2 * 1 = 2
栈: [2]
```

**图解：**

```
     栈            说明
    [0, 1]      当前状态
       ↑
       inner=1（嵌套括号 (())）
    [2]          2 × 1 = 2
```

### 输出结果

```
st.top() = 2
输出：2 ✓
```

## 四、完整执行示例：`(()(())())`

### 步骤图解

| 步骤 | 字符 | 操作 | 栈状态 | 说明 |
|------|------|------|--------|------|
| 初始 | - | - | `[0]` | 初始总分 |
| 1 | `(` | push 0 | `[0, 0]` | 新括号开始 |
| 2 | `(` | push 0 | `[0, 0, 0]` | 嵌套开始 |
| 3 | `)` | inner=0，加1 | `[0, 1]` | `()` → 1 |
| 4 | `(` | push 0 | `[0, 1, 0]` | 新括号开始 |
| 5 | `(` | push 0 | `[0, 1, 0, 0]` | 嵌套开始 |
| 6 | `)` | inner=0，加1 | `[0, 1, 1]` | `()` → 1 |
| 7 | `)` | inner=1，2×1=2 | `[0, 3]` | `(())` → 2 |
| 8 | `(` | push 0 | `[0, 3, 0]` | 新括号开始 |
| 9 | `)` | inner=0，加1 | `[0, 4]` | `()` → 1 |
| 10 | `)` | inner=4，2×4=8 | `[8]` | `(4)` → 8 |

**最终结果：8 ✓**

## 五、栈的变化可视化（`(()(())())`）

```
初始: [0]
第1步 '(' : [0, 0]
第2步 '(' : [0, 0, 0]
第3步 ')' : [0, 1]          ← () → 1
第4步 '(' : [0, 1, 0]
第5步 '(' : [0, 1, 0, 0]
第6步 ')' : [0, 1, 1]       ← () → 1
第7步 ')' : [0, 3]          ← (()) → 2×1=2，累加到外层
第8步 '(' : [0, 3, 0]
第9步 ')' : [0, 4]          ← () → 1，累加到外层
第10步 ')' : [8]            ← (4) → 2×4=8
```

## 六、核心思想总结

| 操作 | 含义 |
|------|------|
| `st.push(0)` | 遇到 `(`，新建一个括号对，分数暂为 0 |
| `st.top()` | 获取当前括号对内部的分数 |
| `st.pop()` | 遇到 `)`，关闭当前括号对 |
| `st.top() += 1` | 空括号 `()` 得 1 分 |
| `st.top() += 2 * inner` | 嵌套括号 `(A)` 得 `2 × inner` 分 |

**栈的本质：** 栈中每个元素代表一个**未闭合的括号对**的累计分数。遇到左括号新建，遇到右括号闭合并累加到外层。
