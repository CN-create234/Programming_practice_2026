# EOJ-1026 0-1背包问题 - 完整解析

---

## 一、题目理解

### 问题描述

有 n 个物品，每个物品有重量 w_i 和价值 v_i，背包容量为 C。每个物品只能选择 **取** 或 **不取**（0-1背包），求在不超过背包容量的情况下，能装下的最大总价值。

### 输入格式

- 第一行：整数 T，表示测试组数
- 每组测试数据：
  - 第一行：n 和 C（物品数量，背包容量）
  - 接下来 n 行：每行两个整数 w_i 和 v_i（重量，价值）

### 样例分析

```
输入：
1
5 10
2 6
2 3
6 5
5 4
4 6

计算：
物品1: 重量2, 价值6
物品2: 重量2, 价值3
物品3: 重量6, 价值5
物品4: 重量5, 价值4
物品5: 重量4, 价值6

最优选择：物品1(2,6) + 物品2(2,3) + 物品5(4,6) = 总重量8, 总价值15
输出：15 ✓
```

---

## 二、解题思路

### 核心思想：动态规划

用 `dp[j]` 表示容量为 j 时能获得的最大价值。

### 状态转移

```
对于每个物品 (w, v)：
    从容量 C 向下遍历到 w：
        dp[j] = max(dp[j], dp[j - w] + v)
```

### 为什么从大到小遍历？

因为每个物品只能用一次（0-1背包），从大到小遍历可以避免物品被重复使用。

## 三、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;

    while (T--) {
        int n, C;
        cin >> n >> C;

        vector<int> dp(C + 1, 0);

        for (int i = 0; i < n; i++) {
            int w, v;
            cin >> w >> v;

            for (int j = C; j >= w; j--) {
                dp[j] = max(dp[j], dp[j - w] + v);
            }
        }

        cout << dp[C] << endl;
    }

    return 0;
}
```

### 6. 动态规划状态转移

```cpp
            for (int j = C; j >= w; j--) {
                dp[j] = max(dp[j], dp[j - w] + v);
            }
        }
```

**解释：**

| 代码 | 含义 |
|------|------|
| `j = C; j >= w; j--` | 从容量 C 向下遍历到 w（倒序） |
| `dp[j]` | 不取当前物品的价值 |
| `dp[j - w] + v` | 取当前物品的价值（腾出 w 空间，加上 v） |
| `max(...)` | 取两者中较大的 |

**为什么倒序？**

倒序保证每个物品只被考虑一次。如果正序，`dp[j]` 可能会被当前物品多次更新。

## 六、算法流程图

```
开始
  ↓
读取 T
  ↓
循环 T 次
  ↓
读取 n, C
  ↓
创建 dp[0..C] = 0
  ↓
循环 n 个物品
  ↓
读取 w, v
  ↓
从 j = C 到 w（倒序）
  ↓
dp[j] = max(dp[j], dp[j-w] + v)
  ↓
输出 dp[C]
  ↓
结束
```

## 七、复杂度分析

| 操作 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 每组测试 | O(n × C) | O(C) |
| 总体 | O(T × n × C) | O(C) |

由于 `n ≤ 1000`，`C ≤ 1000`，单组最多 1,000,000 次运算，可以接受。

## 八、关键点总结

| 要点 | 说明 |
|------|------|
| **核心算法** | 动态规划（0-1背包） |
| **状态定义** | `dp[j]` = 容量 j 时的最大价值 |
| **状态转移** | `dp[j] = max(dp[j], dp[j-w] + v)` |
| **遍历顺序** | 从大到小（防止重复使用物品） |
| **初始化** | 全为 0 |
| **答案** | `dp[C]` |

---

## 九、常见错误

| 错误 | 正确做法 |
|------|----------|
| 正序遍历容量 | 必须从大到小遍历 |
| 不判断 `j >= w` | 循环条件 `j >= w` |
| 二维 DP 数组 | 可用一维数组优化空间 |
| 忘记读取物品数据 | 每次循环都要读取 |

## 十、二维 DP 版本（便于理解）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;

    while (T--) {
        int n, C;
        cin >> n >> C;

        vector<int> w(n), v(n);
        for (int i = 0; i < n; i++) {
            cin >> w[i] >> v[i];
        }

        // dp[i][j] 表示前 i 个物品，容量 j 时的最大价值
        vector<vector<int>> dp(n + 1, vector<int>(C + 1, 0));

        for (int i = 1; i <= n; i++) {
            for (int j = 0; j <= C; j++) {
                if (j >= w[i - 1]) {
                    dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - w[i - 1]] + v[i - 1]);
                } else {
                    dp[i][j] = dp[i - 1][j];
                }
            }
        }

        cout << dp[n][C] << endl;
    }

    return 0;
}
```

## 十一、总结

**核心思想：** 使用一维 DP 数组，倒序遍历容量，对每个物品更新状态：`dp[j] = max(dp[j], dp[j-w] + v)`。倒序保证每个物品只被使用一次。
