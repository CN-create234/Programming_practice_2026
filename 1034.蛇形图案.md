## 完整代码（先贴出来方便对照）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;

    // 方向数组：右，下，左，上
    int dr[] = {0, 1, 0, -1};
    int dc[] = {1, 0, -1, 0};

    for (int caseNum = 0; caseNum < T; caseNum++) {
        int n;
        cin >> n;

        vector<vector<int>> grid(n, vector<int>(n, 0));

        int r = 0, c = 0;      // 当前位置
        int dir = 0;           // 当前方向（0=右，1=下，2=左，3=上）
        int num = 1;

        while (num <= n * n) {
            grid[r][c] = num++;

            int nr = r + dr[dir];
            int nc = c + dc[dir];

            // 如果下一个位置越界或已填过，转向
            if (nr < 0 || nr >= n || nc < 0 || nc >= n || grid[nr][nc] != 0) {
                dir = (dir + 1) % 4;
                nr = r + dr[dir];
                nc = c + dc[dir];
            }

            r = nr;
            c = nc;
        }

        cout << "case #" << caseNum << ":" << endl;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (j > 0) cout << " ";
                cout << grid[i][j];
            }
            cout << endl;
        }
    }

    return 0;
}
```

### 4. 方向数组定义
```cpp
    // 方向数组：右，下，左，上
    int dr[] = {0, 1, 0, -1};
    int dc[] = {1, 0, -1, 0};
```
- `dr` 和 `dc` 是方向数组，配合使用表示四个方向的移动。
- 索引 0：向右 → `(0, 1)`
- 索引 1：向下 → `(1, 0)`
- 索引 2：向左 → `(0, -1)`
- 索引 3：向上 → `(-1, 0)`

```
方向顺序：右 → 下 → 左 → 上 → 右 → ...
```
这是螺旋填数的核心：碰到边界或已填过的格子就**顺时针转向**。

- `caseNum` 从 0 开始，对应输出格式中的 `case #0:`、`case #1:` 等。

### 6. 读取当前 N

        int n;

- 读取当前测试用例的方阵大小 N。

### 7. 创建 N×N 的二维数组
```cpp
        vector<vector<int>> grid(n, vector<int>(n, 0));
```
- 创建一个 `n × n` 的二维向量，所有元素初始化为 0。
- `vector<vector<int>>` 表示二维数组，每个元素是一个 `vector<int>`。
- `grid[n][n]` 的索引范围是 `[0, n-1]`。

**为什么初始化为 0？**  
- 0 表示该位置尚未被填数。
- 在判断是否需要转向时，检查下一个位置是否为 0，如果是 0 说明还没填过，可以继续走；如果不是 0（已填过），则需要转向。

---

### 8. 初始化位置、方向和数字
```cpp
        int r = 0, c = 0;      // 当前位置
        int dir = 0;           // 当前方向（0=右，1=下，2=左，3=上）
        int num = 1;
```
- `r = 0, c = 0`：从方阵的左上角 `(0, 0)` 开始。
- `dir = 0`：初始方向向右（索引 0）。
- `num = 1`：从数字 1 开始填充。

---

### 9. 主循环：填数直到填满所有格子
```cpp
        while (num <= n * n) {
```
- 循环条件：当前要填的数字 `num` 小于等于总格子数 `n * n`。
- 当 `num` 变成 `n*n + 1` 时，说明所有格子已填满，循环结束。

---

### 10. 填充当前位置
```cpp
            grid[r][c] = num++;
```
- 将当前数字 `num` 赋值给当前位置 `(r, c)`。
- `num++` 表示赋值后将 `num` 加 1，准备填下一个数字。

---

### 11. 计算下一个位置
```cpp
            int nr = r + dr[dir];
            int nc = c + dc[dir];
```
- 根据当前方向 `dir`，计算下一个位置 `(nr, nc)`。
- 例如当前方向是右（`dir=0`），则 `dr[0]=0, dc[0]=1`，所以 `nr = r, nc = c + 1`，即向右移动一格。

---

### 12. 判断是否需要转向
```cpp
            if (nr < 0 || nr >= n || nc < 0 || nc >= n || grid[nr][nc] != 0) {
                dir = (dir + 1) % 4;
                nr = r + dr[dir];
                nc = c + dc[dir];
            }
```
- 如果下一个位置满足以下**任一条件**，就需要转向：
  1. `nr < 0`：行号越界（向上超出）
  2. `nr >= n`：行号越界（向下超出）
  3. `nc < 0`：列号越界（向左超出）
  4. `nc >= n`：列号越界（向右超出）
  5. `grid[nr][nc] != 0`：下一个位置已经填过数字（不是 0）

- **转向操作**：
  - `dir = (dir + 1) % 4`：方向顺时针旋转 90 度（0→1→2→3→0）
  - 重新计算转向后的下一个位置：`nr = r + dr[dir]`, `nc = c + dc[dir]`

**为什么要先判断再转向？**  
- 因为我们要沿着当前方向一直走，直到走不通（越界或遇到已填格子）才转向。
- 转向后，新的方向就是下一步要走的方向。

---

### 13. 更新当前位置
```cpp
            r = nr;
            c = nc;
```
- 将当前位置更新为计算出的下一个位置（可能是原方向的下一个位置，也可能是转向后的下一个位置）。
- 下一次循环会填充这个新位置。

- 外层循环 `i` 遍历行。
- 内层循环 `j` 遍历列。
- `if (j > 0) cout << " ";`：在每行的数字之间输出一个空格，但行首不加空格，行尾也没有多余空格。
- 每行结束后输出换行符 `endl`。
