## 一、核心思想

想象你在一个矩形区域内行走，每走完一条边，就把这条边"收缩"进去，缩小未填充区域的范围。

### 形象比喻
就像剥洋葱一样：
1. 先填最外层（上边 → 右边 → 下边 → 左边）
2. 然后把外层"剥掉"，进入内层
3. 继续填下一层（上边 → 右边 → 下边 → 左边）
4. 直到所有格子填满

---

## 二、四个边界变量

```cpp
int top = 0;       // 上边界（当前未填充区域的最上行）
int bottom = n - 1; // 下边界（当前未填充区域的最下行）
int left = 0;      // 左边界（当前未填充区域的最左列）
int right = n - 1; // 右边界（当前未填充区域的最右列）
```

### 初始状态（N=5）

```
top=0  →  [0,0] [0,1] [0,2] [0,3] [0,4]
           [1,0] [1,1] [1,2] [1,3] [1,4]
           [2,0] [2,1] [2,2] [2,3] [2,4]
           [3,0] [3,1] [3,2] [3,3] [3,4]
bottom=4 → [4,0] [4,1] [4,2] [4,3] [4,4]
           ↑                    ↑
         left=0              right=4
```

**未填充区域**：整个矩阵 (0,0) 到 (4,4)

## 五、边界判断的重要性

### 为什么需要 `if (top <= bottom)` 和 `if (left <= right)`？

考虑 N=3 的情况：

```
第1轮后：
top=1, bottom=1, left=1, right=1

第2轮：
  上边：填充 (1,1) = 9
  top++ → top=2
  
  右边：top=2, bottom=1
  因为 top > bottom，所以 if (top <= bottom) 为 false，跳过
  
  下边：跳过
  
  左边：跳过
```

**如果没有边界判断**：
- 右边循环会执行，尝试填充 `(2,1)`（已越界）❌
- 或者覆盖已填的数字 ❌

## 六、完整代码详细注释

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;

    for (int caseNum = 0; caseNum < T; caseNum++) {
        int n;
        cin >> n;

        vector<vector<int>> grid(n, vector<int>(n, 0));
        int num = 1;

        // 四个边界
        int top = 0, bottom = n - 1;
        int left = 0, right = n - 1;

        while (num <= n * n) {
            // 1. 从左到右填充上边
            for (int j = left; j <= right; j++) {
                grid[top][j] = num++;
            }
            top++;  // 上边界向下收缩一行

            // 2. 从上到下填充右边
            for (int i = top; i <= bottom; i++) {
                grid[i][right] = num++;
            }
            right--;  // 右边界向左收缩一列

            // 3. 从右到左填充下边（需要检查是否还有行）
            if (top <= bottom) {
                for (int j = right; j >= left; j--) {
                    grid[bottom][j] = num++;
                }
                bottom--;  // 下边界向上收缩一行
            }

            // 4. 从下到上填充左边（需要检查是否还有列）
            if (left <= right) {
                for (int i = bottom; i >= top; i--) {
                    grid[i][left] = num++;
                }
                left++;  // 左边界向右收缩一列
            }
        }

        // 输出结果
        cout << "case #" << caseNum << ":" << endl;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (j > 0) cout << " ";
                cout << grid[i][j];
            }
            cout << endl;
        }
    }

    return 0;
}
```

## 八、与方向数组法的对比

| 特性 | 边界收缩法 | 方向数组法 |
|------|-----------|-----------|
| 核心思想 | 缩小边界，逐层填充 | 沿着方向走，碰到边界转向 |
| 边界控制 | 显式四边界变量 | 隐式（通过判断越界） |
| 代码直观性 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 适用场景 | 矩形螺旋遍历 | 任意形状的螺旋遍历 |
| 边界判断 | 需要 `if` 判断 | 不需要额外判断 |

## 九、常见变体

### 变体1：先判断再填充（更安全）

```cpp
while (num <= n * n) {
    // 上边
    for (int j = left; j <= right && num <= n * n; j++) {
        grid[top][j] = num++;
    }
    top++;
    
    // 右边
    for (int i = top; i <= bottom && num <= n * n; i++) {
        grid[i][right] = num++;
    }
    right--;
    
    // 下边
    for (int j = right; j >= left && num <= n * n; j--) {
        grid[bottom][j] = num++;
    }
    bottom--;
    
    // 左边
    for (int i = bottom; i >= top && num <= n * n; i--) {
        grid[i][left] = num++;
    }
    left++;
}
```

### 变体2：使用 while 循环逐条边

```cpp
while (left <= right && top <= bottom) {
    // 上边
    for (int j = left; j <= right; j++) grid[top][j] = num++;
    top++;
    
    // 右边
    for (int i = top; i <= bottom; i++) grid[i][right] = num++;
    right--;
    
    // 下边（如果有）
    if (top <= bottom) {
        for (int j = right; j >= left; j--) grid[bottom][j] = num++;
        bottom--;
    }
    
    // 左边（如果有）
    if (left <= right) {
        for (int i = bottom; i >= top; i--) grid[i][left] = num++;
        left++;
    }
}
```

## 总结

**边界收缩法的核心**：
1. 用 `top, bottom, left, right` 四个变量维护当前未填充区域
2. 每轮按 **上→右→下→左** 的顺序填充四条边
3. 填充完一条边后，收缩对应的边界
4. 用 `if` 判断避免重复填充或越界

这种方法**直观、易理解、不易出错**，是螺旋填数问题的最佳入门解法之一！
