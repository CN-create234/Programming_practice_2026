# EOJ-1002 进制转换 - 详细逐行解释

---

## 一、题目回顾

**输入：** 一个 R 进制数字（整数或浮点数）
**输出：** 对应的十进制数，用**最简带分数**表示

---

## 二、完整代码（先贴完整版，后面逐行分解）

```cpp
#include <bits/stdc++.h>
using namespace std;

int charToDigit(char ch) {
    if (ch >= '0' && ch <= '9') return ch - '0';
    if (ch >= 'A' && ch <= 'Z') return ch - 'A' + 10;
    if (ch >= 'a' && ch <= 'z') return ch - 'a' + 10;
    return -1;
}

long long intRto10(const string& s, int base) {
    long long value = 0;
    for (char ch : s) {
        int digit = charToDigit(ch);
        value = value * base + digit;
    }
    return value;
}

long long power(int base, int exp) {
    long long result = 1;
    for (int i = 0; i < exp; i++) {
        result *= base;
    }
    return result;
}

long long gcd(long long a, long long b) {
    while (b != 0) {
        long long t = a % b;
        a = b;
        b = t;
    }
    return a;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int base;
    string s;
    cin >> base >> s;

    int sign = 1;
    size_t pos = 0;
    if (s[0] == '-') {
        sign = -1;
        pos = 1;
    }

    size_t dotPos = s.find('.');
    string intPart, fracPart;

    if (dotPos == string::npos) {
        intPart = s.substr(pos);
        fracPart = "";
    } else {
        intPart = s.substr(pos, dotPos - pos);
        fracPart = s.substr(dotPos + 1);
    }

    long long intValue = 0;
    if (!intPart.empty()) {
        intValue = intRto10(intPart, base);
    }

    long long numerator = 0;
    long long denominator = 1;

    if (!fracPart.empty()) {
        numerator = intRto10(fracPart, base);
        denominator = power(base, (int)fracPart.size());

        long long g = gcd(abs(numerator), denominator);
        numerator /= g;
        denominator /= g;
    }

    if (sign == -1) cout << '-';

    if (numerator == 0) {
        cout << intValue << endl;
    } else if (intValue == 0) {
        cout << numerator << " " << denominator << endl;
    } else {
        cout << intValue << " " << numerator << " " << denominator << endl;
    }

    return 0;
}
//简化
int toDigit(char c) {
    if (c <= '9') return c - '0';
    return toupper(c) - 'A' + 10;
}

long long toDec(const string& s, int base) {
    long long v = 0;
    for (char c : s) v = v * base + toDigit(c);
    return v;
}
```

## 三、逐行详细解释

### 第一部分：头文件和命名空间

```cpp
#include <bits/stdc++.h>
using namespace std;
```

- `bits/stdc++.h`：万能头文件，包含所有标准库
- `using namespace std`：使用标准命名空间

---

### 第二部分：字符转数值函数

```cpp
int charToDigit(char ch) {
    if (ch >= '0' && ch <= '9') return ch - '0';
    if (ch >= 'A' && ch <= 'Z') return ch - 'A' + 10;
    if (ch >= 'a' && ch <= 'z') return ch - 'a' + 10;
    return -1;
}
```

**逐行解释：**

| 代码 | 含义 |
|------|------|
| `charToDigit(char ch)` | 函数名，输入字符，输出数值 |
| `if (ch >= '0' && ch <= '9')` | 判断是否是数字字符 |
| `return ch - '0'` | ASCII码相减得到数值：'5'-'0'=5 |
| `if (ch >= 'A' && ch <= 'Z')` | 判断是否是大写字母 |
| `return ch - 'A' + 10` | 如 'B'-'A'+10 = 1+10=11 |
| `if (ch >= 'a' && ch <= 'z')` | 判断是否是小写字母 |
| `return ch - 'a' + 10` | 同上，处理小写 |
| `return -1` | 非法字符，返回-1 |

### 第三部分：R进制转十进制（整数部分）

```cpp
long long intRto10(const string& s, int base) {
    long long value = 0;
    for (char ch : s) {
        int digit = charToDigit(ch);
        value = value * base + digit;
    }
    return value;
}
```

**逐行解释：**

| 代码 | 含义 |
|------|------|
| `const string& s` | **常量引用**：不拷贝字符串，只读访问 |
| `long long value = 0` | 存储转换结果，用 long long 防溢出 |
| `for (char ch : s)` | 范围for循环，从高位到低位遍历 |
| `int digit = charToDigit(ch)` | 字符转数值 |
| `value = value * base + digit` | **霍纳法则核心**：累乘累加 |

**示例：** `intRto10("FFFF", 16)`

```
初始 value=0
ch='F' → digit=15 → value=0×16+15=15
ch='F' → digit=15 → value=15×16+15=255
ch='F' → digit=15 → value=255×16+15=4095
ch='F' → digit=15 → value=4095×16+15=65535
返回 65535 ✓
```

### 第四部分：幂运算函数

```cpp
long long power(int base, int exp) {
    long long result = 1;
    for (int i = 0; i < exp; i++) {
        result *= base;
    }
    return result;
}
```

**逐行解释：**

| 代码 | 含义 |
|------|------|
| `long long power(int base, int exp)` | 计算 base^exp |
| `long long result = 1` | 初始值为1 |
| `for (int i = 0; i < exp; i++)` | 循环 exp 次 |
| `result *= base` | 每次乘以 base |
| `return result` | 返回结果 |

**示例：** `power(16, 4) = 16×16×16×16 = 65536`

**为什么不用 `pow()`？**
- `pow()` 返回 double，可能有精度误差
- 自定义函数用整数运算，精确无误

### 第五部分：最大公约数（约分）

```cpp
long long gcd(long long a, long long b) {
    while (b != 0) {
        long long t = a % b;
        a = b;
        b = t;
    }
    return a;
}
```

**逐行解释：**

| 代码 | 含义 |
|------|------|
| `long long gcd(long long a, long long b)` | 欧几里得算法求最大公约数 |
| `while (b != 0)` | 当 b 不为0时循环 |
| `long long t = a % b` | 取余数 |
| `a = b; b = t` | 辗转相除 |
| `return a` | 返回最大公约数 |

**示例：** `gcd(65534, 65536)`

```
a=65534, b=65536
t = 65534 % 65536 = 65534
a = 65536, b = 65534

t = 65536 % 65534 = 2
a = 65534, b = 2

t = 65534 % 2 = 0
a = 2, b = 0

返回 2 ✓
```

### 第八部分：分割整数和小数部分

```cpp
    size_t dotPos = s.find('.');

```

```
dotPos == string::npos

fracPart = ""
```

**示例3：** `s = "-0.7772"`，`pos=1`，`dotPos=2`

```
intPart = s.substr(1, 2-1) = s.substr(1, 1) = "0"
fracPart = s.substr(3) = "7772"
```

### 第十部分：转换小数部分（核心！）

```cpp
    long long numerator = 0;
    long long denominator = 1;

    if (!fracPart.empty()) {
        numerator = intRto10(fracPart, base);
        denominator = power(base, (int)fracPart.size());

        long long g = gcd(abs(numerator), denominator);
        numerator /= g;
        denominator /= g;
    }
```

**逐行解释：**

| 代码 | 含义 |
|------|------|
| `long long numerator = 0` | 分子，初始为0 |
| `long long denominator = 1` | 分母，初始为1 |
| `if (!fracPart.empty())` | 如果有小数部分 |
| `numerator = intRto10(fracPart, base)` | 将小数部分当作整数转换 |
| `denominator = power(base, (int)fracPart.size())` | 分母 = R^(小数位数) |
| `long long g = gcd(abs(numerator), denominator)` | 求最大公约数（约分） |
| `numerator /= g; denominator /= g` | 约分 |

**核心公式：**
```
R进制小数 0.abcde... 转十进制分数：
分子 = abcde... (R进制转十进制)
分母 = R^(小数位数)
```

**示例1：** `fracPart = "FFFE"`, `base = 16`

```
numerator = intRto10("FFFE", 16)
         = 15×16³ + 15×16² + 15×16¹ + 14
         = 61440 + 3840 + 240 + 14
         = 65534

denominator = power(16, 4) = 65536

g = gcd(65534, 65536) = 2
numerator = 65534/2 = 32767
denominator = 65536/2 = 32768
```

**示例2：** `fracPart = "7772"`, `base = 8`

```
numerator = intRto10("7772", 8)
         = 7×8³ + 7×8² + 7×8¹ + 2
         = 3584 + 448 + 56 + 2
         = 4090

denominator = power(8, 4) = 4096

g = gcd(4090, 4096) = 2
numerator = 4090/2 = 2045
denominator = 4096/2 = 2048
```


## 四、关键公式总结

### 小数部分转换公式

```
R进制小数: 0.d₁d₂d₃...dₖ

转十进制分数:
分子 = d₁d₂d₃...dₖ (R进制转十进制)
分母 = Rᵏ

然后约分
```

### 完整数值公式

```
结果 = ±(整数部分 + 小数部分)
     = ±(intValue + numerator/denominator)
```

## 六、易错点提醒

| 易错点 | 正确做法 |
|--------|---------|
| 分母是 2 的幂 | ❌ 分母应该是 **R 的幂** |
| 忘记约分 | ✅ 必须用 gcd 约分到最简 |
| 符号处理错误 | ✅ 负号只输出一次，分数取绝对值 |
| 整数部分为0时 | ✅ 输出 `B C` 而不是 `0 B C` |
| 小数部分为空 | ✅ `denominator = 1`，输出纯整数 |

---

这样，EOJ-1002 的每一个细节都解释清楚了！核心思想就是：**小数部分当作整数转十进制，再除以 R 的位数次方，最后约分。**
