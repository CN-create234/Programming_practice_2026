# EOJ-1039 围栏 - 完整解析

---

## 一、题目理解

### 什么是围栏？

一个字符串是围栏，当且仅当它由 **交替出现** 的 `|` 和 `-` 组成。

**围栏示例：**
- `|-|-|-|` ✅（交替出现）
- `-|-|` ✅（交替出现）
- `|` ✅（单独一个字符）
- `-` ✅（单独一个字符）

**非围栏示例：**
- `|-||-|` ❌（连续两个 `|`）
- `--` ❌（连续两个 `-`）
- `||` ❌（连续两个 `|`）

### 本质

围栏 = 字符串中**相邻字符都不相同**。

---

## 二、解题思路

### 核心思想

遍历字符串，统计最长的**相邻字符都不相同**的连续子串长度。

### 判断条件

如果 `s[i] != s[i-1]`，当前字符与前一个字符不同，围栏长度 +1。

如果 `s[i] == s[i-1]`，围栏断开，重新开始计数。

---

## 三、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;

    int n = s.size();
    int maxLen = 1;   // 至少为1（题目保证非空）
    int curLen = 1;

    for (int i = 1; i < n; i++) {
        if (s[i] != s[i - 1]) {
            curLen++;
            maxLen = max(maxLen, curLen);
        } else {
            curLen = 1;
        }
    }

    cout << maxLen << endl;

    return 0;
}
```

## 四、逐行详细解释

### 1. 头文件

```cpp
#include <bits/stdc++.h>

```

万能头文件。

---

### 2. 读取输入

```cpp
int main() {
    string s;

```

读取只包含 `|` 和 `-` 的字符串。

---

### 3. 初始化变量

```cpp
    int n = s.size();
    int maxLen = 1;
    int curLen = 1;
```

| 变量 | 含义 |
|------|------|
| `n` | 字符串长度 |
| `maxLen` | 最长围栏长度，初始为 1 |
| `curLen` | 当前围栏长度，初始为 1 |

---

### 4. 遍历字符串

```cpp
    for (int i = 1; i < n; i++) {
        if (s[i] != s[i - 1]) {
            curLen++;
            maxLen = max(maxLen, curLen);
        } else {
            curLen = 1;
        }
    }
```

**逻辑：**

| 条件 | 操作 |
|------|------|
| `s[i] != s[i-1]` | 当前字符与前一个不同，围栏延长，更新最大值 |
| `s[i] == s[i-1]` | 围栏断开，重新开始计数 |

---

### 5. 输出

```cpp
    cout << maxLen << endl;

    return 0;
}
```

## 五、执行示例

### 样例输入

```
|||-||--|--|---|-||-|-|-|--||---||-||-||-|--||
```

### 手动分析

```
字符串：|||-||--|--|---|-||-|-|-|--||---||-||-||-|--||

遍历过程：
位置0: '|' → curLen=1, maxLen=1
位置1: '|' → s[1]==s[0] → curLen=1, maxLen=1
位置2: '|' → s[2]==s[1] → curLen=1, maxLen=1
位置3: '-' → s[3]!=s[2] → curLen=2, maxLen=2
位置4: '|' → s[4]!=s[3] → curLen=3, maxLen=3
位置5: '|' → s[5]==s[4] → curLen=1, maxLen=3
...
```

最终 `maxLen = 8` ✓

## 六、示例验证

### 示例1：`|-|-|-|`

```
位置0: '|' → curLen=1, maxLen=1
位置1: '-' → 不同 → curLen=2, maxLen=2
位置2: '|' → 不同 → curLen=3, maxLen=3
位置3: '-' → 不同 → curLen=4, maxLen=4
位置4: '|' → 不同 → curLen=5, maxLen=5
位置5: '-' → 不同 → curLen=6, maxLen=6
位置6: '|' → 不同 → curLen=7, maxLen=7

输出：7 ✓
```

### 示例2：`--`

```
位置0: '-' → curLen=1, maxLen=1
位置1: '-' → 相同 → curLen=1, maxLen=1

输出：1 ✓
```

### 示例3：`|-||-|`

```
位置0: '|' → curLen=1, maxLen=1
位置1: '-' → 不同 → curLen=2, maxLen=2
位置2: '|' → 不同 → curLen=3, maxLen=3
位置3: '|' → 相同 → curLen=1, maxLen=3
位置4: '-' → 不同 → curLen=2, maxLen=3
位置5: '|' → 不同 → curLen=3, maxLen=3

输出：3（围栏是 "|-|" 或 "-|-"） ✓
```

## 八、关键点总结

| 要点 | 说明 |
|------|------|
| **围栏定义** | 相邻字符都不相同 |
| **核心判断** | `s[i] != s[i-1]` |
| **断开条件** | `s[i] == s[i-1]` |
| **初始值** | 长度为 1 |

**核心思想：** 遍历字符串，统计连续相邻字符都不相同的最大子串长度。遇到相同字符时，围栏断开，重新计数。
