# 浮点数加法（大数）- 逐行详细解释

---

## 一、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

string add(string a, string b, int n) {
    // 分离整数和小数部分
    int da = a.find('.');
    int db = b.find('.');
    
    string ai = (da == -1) ? a : a.substr(0, da);
    string af = (da == -1) ? "" : a.substr(da + 1);
    string bi = (db == -1) ? b : b.substr(0, db);
    string bf = (db == -1) ? "" : b.substr(db + 1);
    
    if (ai.empty()) ai = "0";
    if (bi.empty()) bi = "0";
    
    // 小数部分补0到相同长度（多留一位用于四舍五入）
    int flen = max(max((int)af.size(), (int)bf.size()), n + 1);
    while ((int)af.size() < flen) af += "0";
    while ((int)bf.size() < flen) bf += "0";
    
    // 整数部分补0到相同长度（多一位防进位）
    int ilen = max((int)ai.size(), (int)bi.size()) + 1;
    while ((int)ai.size() < ilen) ai = "0" + ai;
    while ((int)bi.size() < ilen) bi = "0" + bi;
    
    // 小数部分相加
    string fsum = "";
    int carry = 0;
    for (int i = flen - 1; i >= 0; i--) {
        int s = (af[i] - '0') + (bf[i] - '0') + carry;
        fsum = char(s % 10 + '0') + fsum;
        carry = s / 10;
    }
    
    // 整数部分相加
    string isum = "";
    for (int i = ilen - 1; i >= 0; i--) {
        int s = (ai[i] - '0') + (bi[i] - '0') + carry;
        isum = char(s % 10 + '0') + isum;
        carry = s / 10;
    }
    if (carry) isum = "1" + isum;
    
    // 四舍五入
    if (n < (int)fsum.size() && fsum[n] >= '5') {
        string tmp = fsum.substr(0, n);
        int c = 1;
        for (int i = n - 1; i >= 0 && c; i--) {
            int v = (tmp[i] - '0') + c;
            tmp[i] = char(v % 10 + '0');
            c = v / 10;
        }
        if (c) {
            string tmp2 = isum;
            int c2 = 1;
            for (int i = (int)tmp2.size() - 1; i >= 0 && c2; i--) {
                int v = (tmp2[i] - '0') + c2;
                tmp2[i] = char(v % 10 + '0');
                c2 = v / 10;
            }
            if (c2) tmp2 = "1" + tmp2;
            isum = tmp2;
        }
        fsum = tmp;
    } else {
        fsum = fsum.substr(0, n);
    }
    while ((int)fsum.size() < n) fsum += "0";
    
    // 去掉整数前导零
    int p = 0;
    while (p + 1 < (int)isum.size() && isum[p] == '0') p++;
    isum = isum.substr(p);
    
    return isum + "." + fsum;
}

int main() {
    string a, b;
    int n;
    cin >> a >> b >> n;
    cout << add(a, b, n) << endl;
    return 0;
}
```

## 二、逐行详细解释

### 第一部分：函数定义

```cpp
string add(string a, string b, int n) {
```

- 函数名：`add`，计算两个浮点数的和
- 参数：`a` 和 `b` 是字符串形式的浮点数，`n` 是保留的小数位数
- 返回值：字符串形式的结果

---

### 第二部分：分离整数和小数部分

```cpp
    int da = a.find('.');
    int db = b.find('.');
```

- `a.find('.')`：查找小数点 `.` 的位置
- 如果找到，返回位置（如 `"123.45"` → 返回 `3`）
- 如果没找到，返回 `-1`（用 `int` 接收，替代 `size_t`）

```cpp
    string ai = (da == -1) ? a : a.substr(0, da);
    string af = (da == -1) ? "" : a.substr(da + 1);
    string bi = (db == -1) ? b : b.substr(0, db);
    string bf = (db == -1) ? "" : b.substr(db + 1);
```

| 变量 | 含义 | 示例（a="123.45"） |
|------|------|-------------------|
| `ai` | a 的整数部分 | `"123"` |
| `af` | a 的小数部分 | `"45"` |
| `bi` | b 的整数部分 | - |
| `bf` | b 的小数部分 | - |

**三目运算符解释：**
- `(da == -1) ? a : a.substr(0, da)`
  - 如果没有小数点 → 整个字符串是整数部分
  - 如果有小数点 → 取小数点前面的部分

```cpp
    if (ai.empty()) ai = "0";
    if (bi.empty()) bi = "0";
```

- 如果整数部分为空（如 `".123"`），设为 `"0"`

---

### 第三部分：小数部分补零

```cpp
    int flen = max(max((int)af.size(), (int)bf.size()), n + 1);
```

**计算小数部分需要保留的总位数：**
- `max(af.size(), bf.size())`：两个数中较长的小数位数
- `n + 1`：保留 n 位 + 额外 1 位用于四舍五入
- 取两者最大值

**示例：** `af = "333"`, `bf = "333333"`, `n = 5`

```
af.size() = 3
bf.size() = 6
n + 1 = 6
flen = max(6, 6) = 6
```

```cpp
    while ((int)af.size() < flen) af += "0";
    while ((int)bf.size() < flen) bf += "0";
```

- 在右侧补 0，使两个小数部分长度相同

**示例：** `af = "333"`, `bf = "333333"`, `flen = 6`

```
af: "333" → 补3个0 → "333000"
bf: "333333" → 不变 → "333333"
```

---

### 第四部分：整数部分补零

```cpp
    int ilen = max((int)ai.size(), (int)bi.size()) + 1;
```

- 整数部分长度 = 较长长度 + 1（多一位防止进位）

**示例：** `ai = "9999999999"`（10位）, `bi = "0"`（1位）

```
ilen = max(10, 1) + 1 = 11
```

```cpp
    while ((int)ai.size() < ilen) ai = "0" + ai;
    while ((int)bi.size() < ilen) bi = "0" + bi;
```

- 在左侧补 0，使两个整数部分长度相同

**示例：** `ai = "9999999999"`, `ilen = 11`

```
ai: "9999999999" → 补1个0 → "09999999999"
bi: "0" → 补10个0 → "00000000000"
```

---

### 第五部分：小数部分相加

```cpp
    string fsum = "";
    int carry = 0;
    for (int i = flen - 1; i >= 0; i--) {
        int s = (af[i] - '0') + (bf[i] - '0') + carry;
        fsum = char(s % 10 + '0') + fsum;
        carry = s / 10;
    }
```

**逐行解释：**

| 代码 | 含义 |
|------|------|
| `string fsum = ""` | 存储小数部分结果（从高位到低位） |
| `int carry = 0` | 进位，初始为0 |
| `for (int i = flen - 1; i >= 0; i--)` | 从**最低位**（最右边）开始遍历 |
| `af[i] - '0'` | 字符转数字（如 `'3'` → `3`） |
| `s = digitA + digitB + carry` | 当前位相加 |
| `s % 10` | 当前位的值（0-9） |
| `s / 10` | 进位（0或1） |
| `char(s % 10 + '0')` | 数字转字符 |
| `fsum = char(...) + fsum` | 插入到结果最前面（高位在前） |

**示例：** `af = "333000"`, `bf = "333333"`

```
从右往左：
i=5: 0+3=3 → fsum="3", carry=0
i=4: 0+3=3 → fsum="33", carry=0
i=3: 0+3=3 → fsum="333", carry=0
i=2: 3+3=6 → fsum="6333", carry=0
i=1: 3+3=6 → fsum="66333", carry=0
i=0: 3+3=6 → fsum="666333", carry=0
```

---

### 第六部分：整数部分相加

```cpp
    string isum = "";
    for (int i = ilen - 1; i >= 0; i--) {
        int s = (ai[i] - '0') + (bi[i] - '0') + carry;
        isum = char(s % 10 + '0') + isum;
        carry = s / 10;
    }
    if (carry) isum = "1" + isum;
```

- 逻辑与小数部分相同
- **注意：** `carry` 从小数部分传递过来
- 循环结束后，如果还有进位，在最高位补 `"1"`

---

### 第七部分：四舍五入

```cpp
    if (n < (int)fsum.size() && fsum[n] >= '5') {
        // ... 进位处理
    } else {
        fsum = fsum.substr(0, n);
    }
```

**条件：**
1. `n < (int)fsum.size()`：确保第 n 位存在
2. `fsum[n] >= '5'`：第 n 位（0-based）的数字 >= 5

**示例：** `fsum = "666333"`, `n = 3`

```
n = 3 → 检查 fsum[3] = '3'（0-based：0,1,2,3）
'3' < '5' → 不进位
fsum = fsum.substr(0, 3) = "666"
```

**示例：** `fsum = "6666333"`, `n = 3`

```
n = 3 → 检查 fsum[3] = '6'
'6' >= '5' → 需要进位
```

---

### 第七部分A：四舍五入 - 小数部分进位

```cpp
        string tmp = fsum.substr(0, n);
        int c = 1;
        for (int i = n - 1; i >= 0 && c; i--) {
            int v = (tmp[i] - '0') + c;
            tmp[i] = char(v % 10 + '0');
            c = v / 10;
        }
```

**作用：** 从第 n-1 位开始加 1（进位）

**示例：** `fsum = "6666333"`, `n = 3`

```
tmp = "666"（取前3位）
从 i=2 开始进位：
i=2: v=6+1=7 → tmp[2]='7', c=0
结果：tmp = "667"
```

**示例：** `fsum = "9999"`, `n = 3`

```
tmp = "999"
i=2: 9+1=10 → tmp[2]='0', c=1
i=1: 9+1=10 → tmp[1]='0', c=1
i=0: 9+1=10 → tmp[0]='0', c=1
循环结束，c=1 → 需要进位到整数部分
```

---

### 第七部分B：进位到整数部分

```cpp
        if (c) {
            string tmp2 = isum;
            int c2 = 1;
            for (int i = (int)tmp2.size() - 1; i >= 0 && c2; i--) {
                int v = (tmp2[i] - '0') + c2;
                tmp2[i] = char(v % 10 + '0');
                c2 = v / 10;
            }
            if (c2) tmp2 = "1" + tmp2;
            isum = tmp2;
        }
```

**示例：** `isum = "999"`, 小数进位溢出

```
tmp2 = "999"
从最低位加1：
i=2: 9+1=10 → tmp2[2]='0', c2=1
i=1: 9+1=10 → tmp2[1]='0', c2=1
i=0: 9+1=10 → tmp2[0]='0', c2=1
c2=1 → 补1 → tmp2 = "1000"
```

---

### 第八部分：补全小数位数

```cpp
    while ((int)fsum.size() < n) fsum += "0";
```

- 如果小数部分不足 n 位，在右侧补 0

**示例：** `fsum = "1"`, `n = 5`

```
fsum = "1" → 补4个0 → "10000"
```

---

### 第九部分：去掉整数前导零

```cpp
    int p = 0;
    while (p + 1 < (int)isum.size() && isum[p] == '0') p++;
    isum = isum.substr(p);
```

- 去掉整数部分前面的 0
- **保留至少一位**（即使全是0，也保留 `"0"`）

**示例：**
```
isum = "00000123" → p=5 → isum = "123"
isum = "00000000" → p=7 → isum = "0"（保留最后一位0）
```

---

### 第十部分：返回结果

```cpp
    return isum + "." + fsum;
}
```

- 合并整数部分和小数部分，用 `.` 连接

## 三、完整执行流程示例

### 输入：`9999999999.999999999999999999 .99999999999999999999 10`

```
步骤1：分离
a: ai="9999999999", af="999999999999999999"
b: bi="", bf="99999999999999999999"
bi = "0"

步骤2：补零
flen = max(18, 20, 11) = 20
af → "99999999999999999999"（补2个0）
bf → "99999999999999999999"（不变）

ilen = max(10, 1) + 1 = 11
ai → "09999999999"
bi → "00000000000"

步骤3：小数相加
af + bf → fsum = "199999999999999999998"（20位，最后有进位1）

步骤4：整数相加
ai + bi + carry(1) → isum = "10000000000"

步骤5：四舍五入（n=10）
fsum[10] = '9' >= '5' → 进位
小数部分：9999999999 → 进位 → 0000000000
整数部分：10000000000 + 1 = 10000000001

步骤6：输出
10000000001.0000000000 ✓
```

## 四、总结

| 步骤 | 说明 |
|------|------|
| 1 | 分离整数和小数部分 |
| 2 | 小数部分右侧补0，整数部分左侧补0 |
| 3 | 从最低位开始逐位相加 |
| 4 | 处理进位 |
| 5 | 四舍五入（检查第 n 位） |
| 6 | 去掉整数前导零，补全小数位数 |

**核心思想：用字符串模拟竖式加法，从小数点后最低位开始逐位相加，最后四舍五入。**
