## 二、数据结构：`set` 的特点

```cpp
set<int> A, B;
```

### set 的核心特性

| 特性 | 说明 |
|------|------|
| **自动去重** | 插入重复元素时，只会保留一个 |
| **自动排序** | 元素按升序存储（红黑树实现） |
| **查找高效** | `find()` 操作 O(log n) |
| **迭代有序** | 遍历时按升序输出 |

### 为什么用 set？

本题需要：
1. **去重**：集合中元素不重复
2. **排序输出**：题目要求升序
3. **查找**：判断元素是否属于另一个集合

`set` 完美满足这三个需求。

## 三、`printSet` 函数详解

```cpp
void printSet(const set<int>& s) {
    cout << "{";
    int count = 0;
    for (auto it = s.begin(); it != s.end(); ++it) {
        cout << *it;
        if (++count < s.size()) {
            cout << ",";
        }
    }
    cout << "}" << endl;
}
```

### 逐行解释

```cpp
void printSet(const set<int>& s) {
```
- `const set<int>&`：常量引用传递，避免拷贝，且函数内不修改集合
- 为什么要用引用？避免拷贝整个 set，提高效率

```cpp
    cout << "{";
```
- 输出左花括号

```cpp
    int count = 0;
```
- `count`：记录已经输出了多少个元素
- 用于判断是否需要在元素后面加逗号

```cpp
    for (auto it = s.begin(); it != s.end(); ++it) {
```
- `auto it = s.begin()`：迭代器指向第一个元素
- `it != s.end()`：遍历到最后一个元素之后
- `++it`：迭代器向前移动

**为什么用 `++it` 而不是 `it++`？**
- `++it` 效率更高（不创建临时对象）
- 这里是 C++ 最佳实践

```cpp
        cout << *it;
```
- `*it`：解引用迭代器，获取当前元素的值

```cpp
        if (++count < s.size()) {
            cout << ",";
        }
```
- `++count`：先自增，再比较（count 从 0 开始）
- 第一个元素：count 变成 1，1 < size() → 输出逗号（后面还有元素）
- 最后一个元素：count = size()，不满足条件，不输出逗号

**示例**：s = {1, 2, 3}
```
第1次：count=1, 1<3 → 输出 "1,"
第2次：count=2, 2<3 → 输出 "2,"
第3次：count=3, 3<3 → 输出 "3"
结果：{1,2,3}
```

```cpp
    cout << "}" << endl;
```
- 输出右花括号并换行

## 四、main 函数详解

### 4.1 读取输入

```cpp
int na, nb;
cin >> na >> nb;
```

- `na`：集合 A 的元素个数
- `nb`：集合 B 的元素个数

```cpp
set<int> A, B;

for (int i = 0; i < na; i++) {
    int num;
    cin >> num;
    A.insert(num);
}

for (int i = 0; i < nb; i++) {
    int num;
    cin >> num;
    B.insert(num);
}
```

- 用 `set` 存储，自动去重和排序
- `insert()` 时间复杂度 O(log n)

**示例输入**：
```
5 6
1 2 3 4 5
15 14 13 12 11 10
```

存入后：
```
A = {1, 2, 3, 4, 5}
B = {10, 11, 12, 13, 14, 15}
```

---

### 4.2 交集 C = A ∩ B

```cpp
set<int> C;
for (int x : A) {
    if (B.find(x) != B.end()) {
        C.insert(x);
    }
}
```

**逻辑**：遍历 A 的每个元素，检查是否也在 B 中

| 步骤 | 说明 |
|------|------|
| `for (int x : A)` | 范围 for 循环，遍历 A |
| `B.find(x)` | 在 B 中查找 x，返回迭代器 |
| `!= B.end()` | 如果找到了（没找到返回 end()） |
| `C.insert(x)` | 加入交集 C |

**时间复杂度**：
- 遍历 A：O(|A|)
- 每次查找：O(log |B|)
- 总：O(|A| · log |B|)

**示例**：
```
A = {1, 2, 3, 4, 5}
B = {10, 11, 12, 13, 14, 15}

1 在 B 中？否
2 在 B 中？否
...
C = {}（空集）
```

---

### 4.3 并集 D = A ∪ B

```cpp
set<int> D;
for (int x : A) D.insert(x);
for (int x : B) D.insert(x);
```

**逻辑**：把所有元素插入 D，set 自动去重

| 步骤 | 结果 |
|------|------|
| 插入 A 的所有元素 | D = {1, 2, 3, 4, 5} |
| 插入 B 的所有元素 | D = {1, 2, 3, 4, 5, 10, 11, 12, 13, 14, 15} |

**为什么 set 能自动去重？**
- 插入已存在的元素时，set 不会添加重复项

---

### 4.4 差集 E = A - B

```cpp
set<int> E;
for (int x : A) {
    if (B.find(x) == B.end()) {
        E.insert(x);
    }
}
```

**逻辑**：遍历 A 的每个元素，检查是否**不在** B 中

| 条件 | 说明 |
|------|------|
| `B.find(x) == B.end()` | 在 B 中没找到 x |
| `E.insert(x)` | 加入差集 E（属于 A 但不属于 B） |

---

### 4.5 输出

```cpp
printSet(C);
printSet(D);
printSet(E);
```

按顺序输出交集、并集、差集。

## 五、完整执行流程示例

### 输入
```
5 6
1 2 3 4 5
15 14 13 12 11 10
```

### 执行流程

```
1. 读取 na=5, nb=6
2. 读取 A: {1, 2, 3, 4, 5}
3. 读取 B: {10, 11, 12, 13, 14, 15}
4. 计算交集 C：
   遍历 A：1,2,3,4,5 都不在 B 中
   C = {}
5. 计算并集 D：
   插入 A：{1,2,3,4,5}
   插入 B：{1,2,3,4,5,10,11,12,13,14,15}
   D = {1,2,3,4,5,10,11,12,13,14,15}
6. 计算差集 E：
   遍历 A：1,2,3,4,5 都不在 B 中
   E = {1,2,3,4,5}
7. 输出：
   {}
   {1,2,3,4,5,10,11,12,13,14,15}
   {1,2,3,4,5}
```

## 六、代码改进建议

### 1. 使用 `auto` 简化迭代

```cpp
for (auto it = s.begin(); it != s.end(); ++it)
```

### 2. 使用范围 for 循环（C++11）

```cpp
for (int x : s) {
    cout << x;
}
```

### 3. 使用 STL 算法（更专业）

```cpp
#include <algorithm>
#include <iterator>

set<int> C, D, E;

set_intersection(A.begin(), A.end(), B.begin(), B.end(), 
                 inserter(C, C.begin()));
set_union(A.begin(), A.end(), B.begin(), B.end(), 
          inserter(D, D.begin()));
set_difference(A.begin(), A.end(), B.begin(), B.end(), 
               inserter(E, E.begin()));
```

## 七、关键点总结

| 知识点 | 代码 | 说明 |
|--------|------|------|
| set 自动排序 | `set<int> A;` | 元素升序存储 |
| set 自动去重 | `A.insert(num)` | 重复插入无效 |
| set 查找 | `B.find(x)` | O(log n) 查找 |
| 范围 for | `for (int x : A)` | 遍历所有元素 |
| 常量引用 | `const set<int>& s` | 避免拷贝 |
| 迭代器 | `auto it = s.begin()` | 遍历容器 |

## 八、一句话总结

**用 set 存储集合，利用其自动去重和排序特性，通过遍历和查找完成集合运算，最后用迭代器格式化输出。**
