# EOJ-1019 分析字符 - 完整解析

---

## 一、题目理解

### 问题描述

输入一行 ASCII 字符，统计其中：
1. 大写英文字母个数（A-Z）
2. 小写英文字母个数（a-z）
3. 空格个数（space）
4. 数字个数（0-9）
5. 其他字符个数（不属于以上四类的所有字符）

### 输入格式

- 第1行：整数 T，表示问题数
- 接下来 T 行：每行是要统计的一行字符（不超过100个字符）

**注意：** 一行字符可能包含空格，需要用 `getline` 读取。

### 输出格式

- 先输出 `case #编号:`
- 再输出：`大写个数 小写个数 空格个数 数字个数 其他个数`

## 三、解题思路

### 核心步骤

```
1. 读取整数 T
2. 用 cin.ignore() 跳过换行符
3. 对于每个问题：
   a. 用 getline 读取一行（包含空格）
   b. 遍历每个字符，分类统计
   c. 输出结果
```

### 字符分类

```cpp
if (ch >= 'A' && ch <= 'Z')       → 大写字母
else if (ch >= 'a' && ch <= 'z')  → 小写字母
else if (ch == ' ')               → 空格
else if (ch >= '0' && ch <= '9')  → 数字
else                              → 其他字符
```

## 四、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;
    cin.ignore();  // 跳过第一行末尾的换行符

    for (int caseNum = 0; caseNum < T; caseNum++) {
        string s;
        getline(cin, s);

        int upper = 0, lower = 0, space = 0, digit = 0, other = 0;

        for (char ch : s) {
            if (ch >= 'A' && ch <= 'Z') {
                upper++;
            } else if (ch >= 'a' && ch <= 'z') {
                lower++;
            } else if (ch == ' ') {
                space++;
            } else if (ch >= '0' && ch <= '9') {
                digit++;
            } else {
                other++;
            }
        }

        cout << "case #" << caseNum << ":" << endl;
        cout << upper << " " << lower << " " << space << " " 
             << digit << " " << other << endl;
    }

    return 0;
}
```

## 五、逐行详细解释

### 1. 头文件

```cpp
#include <bits/stdc++.h>
using namespace std;
```

万能头文件。

---

### 2. 主函数 - 读取 T

```cpp
int main() {
    int T;
    cin >> T;
    cin.ignore();
```

- `cin >> T`：读取问题数
- `cin.ignore()`：跳过第一行末尾的换行符

**为什么要用 `cin.ignore()`？**

```
输入：
4          ← cin >> T 读取 4，但换行符还在缓冲区
44444 ADDFFD  ← 如果不 ignore，getline 会读到空行！
```

`cin.ignore()` 会读取并丢弃一个字符（换行符），确保后面的 `getline` 能正确读取。

---

### 3. 循环处理每个问题

```cpp
    for (int caseNum = 0; caseNum < T; caseNum++) {
        string s;
        getline(cin, s);
```

- `caseNum` 从 0 开始编号
- `getline(cin, s)`：读取整行，**包含空格**

---

### 4. 初始化计数器

```cpp
        int upper = 0, lower = 0, space = 0, digit = 0, other = 0;
```

- `upper`：大写字母个数
- `lower`：小写字母个数
- `space`：空格个数
- `digit`：数字个数
- `other`：其他字符个数

---

### 5. 遍历统计

```cpp
        for (char ch : s) {
            if (ch >= 'A' && ch <= 'Z') {
                upper++;
            } else if (ch >= 'a' && ch <= 'z') {
                lower++;
            } else if (ch == ' ') {
                space++;
            } else if (ch >= '0' && ch <= '9') {
                digit++;
            } else {
                other++;
            }
        }
```

**分类逻辑：**

| 条件 | 分类 |
|------|------|
| `ch >= 'A' && ch <= 'Z'` | 大写字母 |
| `ch >= 'a' && ch <= 'z'` | 小写字母 |
| `ch == ' '` | 空格 |
| `ch >= '0' && ch <= '9'` | 数字 |
| 以上都不满足 | 其他字符 |

---

### 6. 输出

```cpp
        cout << "case #" << caseNum << ":" << endl;
        cout << upper << " " << lower << " " << space << " " 
             << digit << " " << other << endl;
    }

    return 0;
}
```

- 输出格式：先输出 `case #编号:`，换行
- 再输出五个数字，用空格分隔
