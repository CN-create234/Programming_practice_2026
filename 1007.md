# EOJ-1007 平衡二进制 - 完整解析

---

## 一、题目理解

### 什么是平衡二进制？

这道题中的"平衡二进制"实际上是一种特殊的**三进制**系统，数码是 **0、1、2**，其中 **2 表示 -1**。

| 数码字符 | 实际值 |
|----------|--------|
| `0` | 0 |
| `1` | 1 |
| `2` | -1 |

### 位权规则

```
从右往左（从低位到高位）：
第0位：3^0 = 1
第1位：3^1 = 3
第2位：3^2 = 9
第3位：3^3 = 27
...

小数部分：
第-1位：3^(-1) = 1/3
第-2位：3^(-2) = 1/9
```

### 示例验证

**1120.22（平衡二进制）转十进制：**

```
整数部分 1120：
1×27 + 1×9 + (-1)×3 + 0×1 = 27 + 9 - 3 + 0 = 33

小数部分 22：
(-1)×1/3 + (-1)×1/9 = -1/3 - 1/9 = -4/9

总值 = 33 - 4/9 = 32 + 5/9
输出：32 5 9
```

## 二、解题思路

### 核心步骤

```
1. 判断输入是否有负号
2. 分割整数部分和小数部分
3. 整数部分转十进制（霍纳法则）
4. 小数部分转分数（分子/分母）
5. 合并整数和小数部分
6. 约分
7. 按带分数格式输出
```

### 关键点

- 数码 `'2'` 表示 `-1`
- 小数部分化为分数：`0.d₁d₂...dₖ = (d₁×3^(k-1) + ... + dₖ) / 3^k`
- 结果以最简带分数输出

## 三、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

// 数码转数值：2 表示 -1
int digitValue(char c) {
    if (c == '0') return 0;
    if (c == '1') return 1;
    if (c == '2') return -1;
    return 0;
}

// 递归求最大公约数
long long gcdll(long long a, long long b) {
    return b == 0 ? abs(a) : gcdll(b, a % b);
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    cin >> s;

    // 1. 处理负号
    bool neg = false;
    if (s[0] == '-') {
        neg = true;
        s = s.substr(1);
    }

    // 2. 分割整数和小数部分
    size_t dotPos = s.find('.');
    string intPart, fracPart;
    if (dotPos == string::npos) {
        intPart = s;
        fracPart = "";
    } else {
        intPart = s.substr(0, dotPos);
        fracPart = s.substr(dotPos + 1);
    }

    if (intPart.empty()) intPart = "0";

    // 3. 整数部分转十进制（霍纳法则）
    long long intVal = 0;
    for (char c : intPart) {
        intVal = intVal * 3 + digitValue(c);
    }

    // 4. 小数部分转分数
    long long numF = 0;   // 分子
    long long den = 1;    // 分母
    for (char c : fracPart) {
        numF = numF * 3 + digitValue(c);
        den *= 3;
    }

    // 5. 合并：总值 = intVal + numF/den = (intVal * den + numF) / den
    long long totalNum = intVal * den + numF;

    // 6. 应用负号
    if (neg) totalNum = -totalNum;

    if (totalNum == 0) {
        cout << 0 << '\n';
        return 0;
    }

    // 7. 约分
    long long g = gcdll(abs(totalNum), den);
    totalNum /= g;
    den /= g;

    // 8. 分离整数部分和真分数部分
    int sign = (totalNum > 0 ? 1 : -1);
    long long absNum = abs(totalNum);
    long long whole = absNum / den;
    long long rem = absNum % den;

    // 9. 输出
    string signStr = (sign == 1 ? "" : "-");

    if (rem == 0) {
        cout << signStr << whole << '\n';
    } else if (whole == 0) {
        cout << signStr << rem << " " << den << '\n';
    } else {
        cout << signStr << whole << " " << rem << " " << den << '\n';
    }

    return 0;
}
```

### gcdll 函数

```cpp
long long gcdll(long long a, long long b) {
    return b == 0 ? abs(a) : gcdll(b, a % b);
}
```

递归求最大公约数，用于约分。

**示例：**
```
gcdll(4, 9) → gcdll(9, 4) → gcdll(4, 1) → gcdll(1, 0) → 1
gcdll(2, 6) → gcdll(6, 2) → gcdll(2, 0) → 2
```

### 5. 整数部分转换

```cpp
    long long intVal = 0;
    for (char c : intPart) {
        intVal = intVal * 3 + digitValue(c);
    }
```

**霍纳法则：** 从左往右，每次乘以 3 加上当前数码值。

**示例：** `intPart = "1120"`

```
intVal = 0
'1' → 0×3 + 1 = 1
'1' → 1×3 + 1 = 4
'2' → 4×3 + (-1) = 11
'0' → 11×3 + 0 = 33
```

---

### 6. 小数部分转换

```cpp
    long long numF = 0;
    long long den = 1;
    for (char c : fracPart) {
        numF = numF * 3 + digitValue(c);
        den *= 3;
    }
```

**公式：**
```
0.d₁d₂...dₖ = (d₁×3^(k-1) + d₂×3^(k-2) + ... + dₖ) / 3^k
```

**示例：** `fracPart = "22"`

```
numF = 0, den = 1
'2' → numF = 0×3 + (-1) = -1, den = 3
'2' → numF = -1×3 + (-1) = -4, den = 9
```

**结果：** `-4/9`

---

### 7. 合并

```cpp
    long long totalNum = intVal * den + numF;

    if (neg) totalNum = -totalNum;
```

**公式：**
```
总值 = intVal + numF/den = (intVal × den + numF) / den
```

**示例：** `intVal=33`, `numF=-4`, `den=9`

```
totalNum = 33×9 + (-4) = 293
```

---

### 8. 零值处理

```cpp
    if (totalNum == 0) {
        cout << 0 << '\n';
        return 0;
    }
```

如果结果为 0，直接输出。

---

### 9. 约分

```cpp
    long long g = gcdll(abs(totalNum), den);
    totalNum /= g;
    den /= g;
```

用最大公约数约分到最简。

**示例：** `totalNum=293`, `den=9`

```
g = gcdll(293, 9) = 1
不变
```

---

### 10. 分离整数和分数

```cpp
    int sign = (totalNum > 0 ? 1 : -1);
    long long absNum = abs(totalNum);
    long long whole = absNum / den;

```

| 变量 | 含义 |
|------|------|
| `sign` | 符号（1 或 -1） |
| `absNum` | 分子绝对值 |

**示例：** `totalNum=293`, `den=9`

```

```

## 六、关键点总结

| 步骤 | 说明 |
|------|------|
| **digitValue** | `'0'→0`, `'1'→1`, `'2'→-1` |
| **整数转换** | 霍纳法则：`intVal = intVal × 3 + digit` |
| **小数转换** | `numF = numF × 3 + digit`, `den = den × 3` |
| **合并** | `totalNum = intVal × den + numF` |
| **负号处理** | 输入 `-` 取反整个值 |
| **约分** | 用 `gcdll` 约分 |
| **输出格式** | 带分数 `[符号]整数 分子 分母` |

**核心思想：** 数码 `'2'` 代表 `-1`，用霍纳法则转换为十进制（分数），约分后按带分数格式输出。
