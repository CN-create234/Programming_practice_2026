# 1001. 不同进制转换
```cpp
int main() {
    int a, b;
    string n;
    cin >> a >> n >> b;
    
    // 步骤1：将 a 进制字符串 n 转换为十进制整数
    long long decimal_value = 0;
    for (char ch : n) {
        int digit;
        if (ch >= '0' && ch <= '9') {
            digit = ch - '0';
        } else {
            // 字母可能是大写或小写，统一转为大写处理
            digit = toupper(ch) - 'A' + 10;
        }
        decimal_value = decimal_value * a + digit;
    }
    
    // 步骤2：将十进制数转换为 b 进制
    if (decimal_value == 0) {
        cout << "0" << endl;
        return 0;
    }
    
    string result = "";
    while (decimal_value > 0) {
        int remainder = decimal_value % b;
        if (remainder < 10) {
            result += char('0' + remainder);
        } else {
            result += char('A' + remainder - 10);
        }
        decimal_value /= b;
    }
    
    // 反转字符串得到最终结果
    reverse(result.begin(), result.end());
    cout << result << endl;
    
    return 0;
}
```
## 一、问题核心理解

题目要求：将 **a 进制** 的数字字符串 `n`，转换为 **b 进制** 的数字字符串输出。

**关键点：**
- 进制范围：2 到 16
- 符号表示：0-9 表示数字，A-F（或 a-f）表示 10-15
- 输出必须是大写字母
- 数值在 32 位范围内（所以用 64 位整数存储完全足够）

---

## 二、进制转换的通用方法

任何进制转换，最通用的方法是：

```
a进制 → 十进制 → b进制
```

为什么用十进制作为中间桥梁？
- 因为我们人类和计算机都熟悉十进制运算
- 十进制是所有进制转换的"通用语言"

---

## 三、第一步：a进制 → 十进制

### 原理

对于 a 进制数 `n`，假设它的各位数字从高位到低位是：`d₁ d₂ d₃ ... dₖ`

那么它对应的十进制值为：

```
十进制值 = d₁ × a^(k-1) + d₂ × a^(k-2) + ... + dₖ × a^0
```

### 举例说明

**例1：** 二进制 `1011`（a=2）转十进制

```
从高位到低位：1, 0, 1, 1
= 1×2³ + 0×2² + 1×2¹ + 1×2⁰
= 8 + 0 + 2 + 1
= 11
```

**例2：** 十六进制 `A3`（a=16）转十进制

```
A = 10, 3 = 3
= 10×16¹ + 3×16⁰
= 160 + 3
= 163
```

**例3：** 题目样例 `Aab3`（15进制）

```
A = 10, a = 10, b = 11, 3 = 3
= 10×15³ + 10×15² + 11×15¹ + 3×15⁰
= 10×3375 + 10×225 + 11×15 + 3
= 33750 + 2250 + 165 + 3
= 36168
```

### 代码实现技巧：霍纳法则（Horner's Method）

实际代码中，我们**不会**真的去计算幂次，而是用**累乘累加**的方法：

```cpp
decimal_value = 0;
for (每个字符 ch) {
    digit = 字符转数字;
    decimal_value = decimal_value * a + digit;
}
```

**为什么这样可以？**

以二进制 `1011` 为例：
```
初始：value = 0
读入 '1'：value = 0×2 + 1 = 1
读入 '0'：value = 1×2 + 0 = 2
读入 '1'：value = 2×2 + 1 = 5
读入 '1'：value = 5×2 + 1 = 11
```

这正是 `1×2³ + 0×2² + 1×2¹ + 1×2⁰` 的计算过程，只是换了一种等价的计算顺序。

这就是著名的**霍纳法则**，效率更高，不需要计算幂次。

## 四、第二步：十进制 → b进制

### 原理

采用**除基取余法**：

```
while (十进制数 > 0) {
    余数 = 十进制数 % b;   // 得到最低位
    将余数转为字符保存;
    十进制数 = 十进制数 / b; // 去掉最低位
}
最后将得到的字符序列反转
```

### 举例说明

**例：** 十进制 11 转二进制

```
11 ÷ 2 = 5 余 1  → 最低位是 1
5  ÷ 2 = 2 余 1  → 下一位是 1
2  ÷ 2 = 1 余 0  → 下一位是 0
1  ÷ 2 = 0 余 1  → 最高位是 1

从下往上读：1011 ✓
```

**例：** 十进制 36168 转 7 进制（题目样例）

```
36168 ÷ 7 = 5166 余 6
5166  ÷ 7 = 738  余 0
738   ÷ 7 = 105  余 3
105   ÷ 7 = 15   余 0
15    ÷ 7 = 2    余 1
2     ÷ 7 = 0    余 2

从下往上读：210306 ✓
```

### 字符映射

当进制大于 10 时，余数可能是 10-15，需要用字母表示：

```
余数 0-9  → 字符 '0'-'9'
余数 10   → 字符 'A'
余数 11   → 字符 'B'
...
余数 15   → 字符 'F'
```

代码中的映射：
```cpp
if (remainder < 10) {
    result += char('0' + remainder);      // 0→'0', 1→'1', ...
} else {
    result += char('A' + remainder - 10); // 10→'A', 11→'B', ...
}
```

## 五、代码逐行详解

```cpp
#include <string>
#include <algorithm>

    string n;

```
- 输入格式是：`15 Aab3 7`
- `cin >> a` 读取 15
- `cin >> n` 读取 "Aab3"
- `cin >> b` 读取 7

---

```cpp
    long long decimal_value = 0;
    for (char ch : n) {
        int digit;

```
- `long long` 保证足够存储 32 位范围内的数值
- `toupper(ch)` 将小写转大写，统一处理
- 每次迭代应用霍纳法则累乘累加

---

```cpp
    if (decimal_value == 0) {
        cout << "0" << endl;
        return 0;
    }
```
- 特殊情况：如果输入是 "0"，直接输出 "0"
- 如果不处理，下面的循环会跳过，导致输出空字符串

---

```cpp
    string result = "";
    while (decimal_value > 0) {
        int remainder = decimal_value % b;
        if (remainder < 10) {
            result += char('0' + remainder);
        } else {
            result += char('A' + remainder - 10);
        }
        decimal_value /= b;
    }
```
- 不断取余、除基
- 余数转为字符追加到 `result` 末尾
- 注意：这里得到的顺序是反的（最低位在前）

---

```cpp
    reverse(result.begin(), result.end());
    cout << result << endl;
```
- 反转字符串得到正确的从高位到低位的顺序
- 输出结果

## 六、完整流程示例

以输入 `15 Aab3 7` 为例：

```
步骤1：Aab3 转十进制
'A' → 10 → value = 0×15+10 = 10
'a' → 10 → value = 10×15+10 = 160
'b' → 11 → value = 160×15+11 = 2411
'3' → 3  → value = 2411×15+3 = 36168

步骤2：36168 转 7 进制
36168 % 7 = 6 → result = "6"
36168 / 7 = 5166
5166 % 7 = 0  → result = "60"
5166 / 7 = 738
738 % 7 = 3   → result = "603"
738 / 7 = 105
105 % 7 = 0   → result = "6030"
105 / 7 = 15
15 % 7 = 1    → result = "60301"
15 / 7 = 2
2 % 7 = 2     → result = "603012"
2 / 7 = 0

步骤3：反转
"603012" → "210306"

输出：210306 ✓
```

## 七、时间复杂度分析

- 转十进制：遍历每个字符，O(k)，k 是字符串长度
- 转目标进制：循环次数是对数级别，O(log₂N)，N 是数值大小
- 总体时间复杂度：O(k + log N)，非常高效

---

## 八、需要注意的细节

1. **大小写处理**：输入可能包含小写字母，输出必须是大写，所以转十进制时用 `toupper()`，输出时直接拼接大写字母。

2. **零值处理**：如果输入是 "0"，无论什么进制，输出都应该是 "0"。

3. **数据类型**：虽然题目说在 32 位范围内，但为了安全使用 `long long`。

4. **字符转换**：
   - 数字字符：`ch - '0'` 得到数值
   - 字母字符：`toupper(ch) - 'A' + 10` 得到 10-15
