# 最长无重复子串 - 完整解析

---

## 一、题目理解

### 问题描述

给定一个字符串 s，找出其中最长的**无重复字符子串**（连续的一段），如果有多个长度相同的最长子串，返回位置最靠前的那个。

### 示例

| 输入 | 输出 | 说明 |
|------|------|------|
| `"b"` | `"b"` | 单个字符 |
| `"abcdabc"` | `"abcd"` | `"abcd"` 和 `"dabc"` 长度都是4，返回第一个 `"abcd"` |
| `"aabcb"` | `"abc"` | 最长无重复子串是 `"abc"` |

---

## 二、解题思路

### 核心思想：滑动窗口（双指针）

维护一个窗口 `[left, right]`，保证窗口内的字符都不重复。

1. **右指针向右移动**，扩大窗口
2. **如果当前字符已经出现过**，移动左指针到重复字符的下一个位置
3. **更新最长子串**的位置

### 示例：`"abcda"`

```
left=0, right=0: 窗口 "a" → 最长 "a"
left=0, right=1: 窗口 "ab" → 最长 "ab"
left=0, right=2: 窗口 "abc" → 最长 "abc"
left=0, right=3: 窗口 "abcd" → 最长 "abcd"
left=1, right=4: 窗口 "bcda" → 长度4，但 "abcd" 更靠前
最终结果 "abcd"
```

## 三、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;

    int n = s.size();
    vector<int> lastPos(256, -1);  // 记录每个字符最后出现的位置

    int maxLen = 0;
    int start = 0;  // 最长子串的起始位置

    int left = 0;   // 窗口左边界

    for (int right = 0; right < n; right++) {
        char ch = s[right];

        // 如果字符已经出现过，且在窗口内，更新左边界
        if (lastPos[ch] != -1 && lastPos[ch] >= left) {
            left = lastPos[ch] + 1;
        }

        // 更新当前字符的最新位置
        lastPos[ch] = right;

        // 更新最长子串
        int curLen = right - left + 1;
        if (curLen > maxLen) {
            maxLen = curLen;
            start = left;
        }
    }

    cout << s.substr(start, maxLen) << endl;

    return 0;
}
```

## 四、逐行详细解释

### 1. 头文件

```cpp
#include <bits/stdc++.h>
using namespace std;
```

万能头文件。

---

### 2. 读取输入

```cpp
int main() {
    string s;
    cin >> s;
```

读取字符串。

---

### 3. 初始化数据结构

```cpp
    int n = s.size();
    vector<int> lastPos(256, -1);  // 记录每个字符最后出现的位置
```

- `lastPos` 数组大小 256，覆盖所有 ASCII 字符（英文字母和数字）
- 初始值 `-1` 表示字符尚未出现

---

### 4. 定义变量

```cpp
    int maxLen = 0;
    int start = 0;   // 最长子串的起始位置
    int left = 0;    // 当前窗口左边界
```

---

### 5. 滑动窗口遍历

```cpp
    for (int right = 0; right < n; right++) {
        char ch = s[right];
```

- `right` 是窗口右边界
- 每次循环，右指针向右移动一位

---

### 6. 处理重复字符

```cpp
        if (lastPos[ch] != -1 && lastPos[ch] >= left) {
            left = lastPos[ch] + 1;
        }
```

**逻辑：**
- 如果当前字符 `ch` 已经出现过
- 且出现位置在窗口内（`>= left`）
- 则移动左边界到重复位置的下一个位置，跳过重复字符

**示例：** `"abca"`

```
处理到 right=3（字符 'a'）：
lastPos['a'] = 0，left = 0
lastPos['a'] >= left → left = 0 + 1 = 1
窗口变为 "bca"
```

---

### 7. 更新字符位置

```cpp
        lastPos[ch] = right;
```

记录当前字符的最新位置。

---

### 8. 更新最长子串

```cpp
        int curLen = right - left + 1;
        if (curLen > maxLen) {
            maxLen = curLen;
            start = left;
        }
```

- 计算当前窗口长度
- 如果比之前的最长长度大，更新 `maxLen` 和 `start`

**注意：** 使用 `>` 而不是 `>=`，确保相同长度时保留最靠前的位置（因为左指针只向右移动，第一次达到最大长度时一定是最靠前的）。

---

### 9. 输出结果

```cpp
    cout << s.substr(start, maxLen) << endl;

    return 0;
}
```

从 `start` 开始截取 `maxLen` 个字符。

## 五、执行示例

### 示例1：`"abcdabc"`

| right | ch | lastPos[ch] | left | 窗口 | maxLen | start |
|-------|----|-------------|------|------|--------|-------|
| 0 | a | -1 | 0 | a | 1 | 0 |
| 1 | b | -1 | 0 | ab | 2 | 0 |
| 2 | c | -1 | 0 | abc | 3 | 0 |
| 3 | d | -1 | 0 | abcd | 4 | 0 |
| 4 | a | 0→0+1=1 | 1 | bcda | 4 | 0（不更新） |
| 5 | b | 1→1+1=2 | 2 | cdab | 4 | 0（不更新） |
| 6 | c | 2→2+1=3 | 3 | dabc | 4 | 0（不更新） |

最终：`start=0, maxLen=4` → 输出 `"abcd"` ✓

---

### 示例2：`"aabcb"`

| right | ch | lastPos[ch] | left | 窗口 | maxLen | start |
|-------|----|-------------|------|------|--------|-------|
| 0 | a | -1 | 0 | a | 1 | 0 |
| 1 | a | 0→0+1=1 | 1 | a | 1 | 0 |
| 2 | b | -1 | 1 | ab | 2 | 1 |
| 3 | c | -1 | 1 | abc | 3 | 1 |
| 4 | b | 2→2+1=3 | 3 | cb | 3 | 1（不更新） |

最终：`start=1, maxLen=3` → 输出 `"abc"` ✓

---

### 示例3：`"b"`

| right | ch | lastPos[ch] | left | 窗口 | maxLen | start |
|-------|----|-------------|------|------|--------|-------|
| 0 | b | -1 | 0 | b | 1 | 0 |

最终：`start=0, maxLen=1` → 输出 `"b"` ✓

## 六、算法流程总结

```
1. 初始化 left = 0，maxLen = 0，start = 0
2. 遍历字符串，right 从 0 到 n-1：
   a. 如果 s[right] 在窗口内出现过：
      left = 上次出现位置 + 1
   b. 更新 s[right] 的最新位置 = right
   c. 计算当前窗口长度 = right - left + 1
   d. 如果当前长度 > maxLen：
      更新 maxLen 和 start
3. 输出 s.substr(start, maxLen)
```

|------|-----------|-----------|

## 八、关键点总结

| 要点 | 说明 |
|------|------|
| **核心算法** | 滑动窗口（双指针） |
| **左指针移动条件** | 当前字符在窗口内重复时 |
| **左指针移动方式** | `left = lastPos[ch] + 1` |
| **更新最长子串条件** | `curLen > maxLen`（确保最靠前） |
| **字符记录** | `lastPos` 数组记录每个字符的最新位置 |
| **时间复杂度** | O(n) |

---

## 九、常见错误

| 错误 | 正确做法 |
|------|----------|
| 使用 `curLen >= maxLen` 更新 | 使用 `>`，否则可能返回后面的子串 |
| 只判断 `lastPos[ch] != -1` | 还要判断 `lastPos[ch] >= left`（在窗口内） |
| 不更新 `lastPos[ch]` | 每次都要更新为当前位置 |
| 用 `substr` 截取错误长度 | 使用 `maxLen` 作为长度 |

---

## 十、总结

**核心思想：** 使用滑动窗口（双指针）维护一个无重复字符的窗口，右指针每次移动时，如果当前字符在窗口内重复，则移动左指针跳过重复字符，窗口始终保持无重复字符，同时记录窗口的最大长度和起始位置。
