# EOJ-1024 找出最小字符串 - 逐行详细解释

---

## 一、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;

    string result = "";
    int n = s.size();

    for (int i = 0; i < n; i++) {
        // 最后一个字符不扩展
        if (i == n - 1) {
            result += s[i];
            continue;
        }

        // 找 i 之后第一个不等于 s[i] 的字符
        int j = i + 1;
        while (j < n && s[j] == s[i]) {
            j++;
        }

        bool expand = false;
        if (j == n) {
            // 后面全是相同字符 → 不扩展
            expand = false;
        } else if (s[i] < s[j]) {
            // 后面的不同字符更大 → 扩展
            expand = true;
        } else {
            // 后面的不同字符更小 → 不扩展
            expand = false;
        }

        if (expand) {
            result += s[i];
            result += s[i];
        } else {
            result += s[i];
        }
    }

    cout << result << endl;

    return 0;
}
```

### 第四部分：处理最后一个字符

```cpp
        // 最后一个字符不扩展
        if (i == n - 1) {
            result += s[i];
            continue;
        }
```

**为什么要特殊处理最后一个字符？**

| 原因 | 说明 |
|------|------|
| **扩展无意义** | 最后一个字符扩展后，字符串变长，但字典序不会变小 |
| **字典序规则** | 短的字符串在作为前缀时更小（如 `"A"` < `"AA"`） |
| **贪心最优** | 最后一个字符不扩展总是最优 |

**示例：** `"C"`

```
i=0: i == n-1 → result = "C"
输出：C ✓
```

### 第五部分：找第一个不同字符（核心）

```cpp
        // 找 i 之后第一个不等于 s[i] 的字符
        int j = i + 1;
        while (j < n && s[j] == s[i]) {
            j++;
        }
```

**逐行解释：**

| 行 | 代码 | 含义 |
|----|------|------|
| 1 | `int j = i + 1` | 从下一个位置开始查找 |
| 2 | `while (j < n && s[j] == s[i])` | 只要还没到末尾，且当前字符等于 `s[i]`，就继续 |
| 3 | `j++` | 指针向后移动 |

**作用：** 跳过所有与 `s[i]` 相同的连续字符，找到第一个不同的字符。

**示例：** `s = "AAABC"`

```
i=0, s[0]='A'
j=1: s[1]=='A' → j=2
j=2: s[2]=='A' → j=3
j=3: s[3]=='B' != 'A' → 停止
j=3, s[j]='B'
```

**示例：** `s = "AAA"`

```
i=0, s[0]='A'
j=1: s[1]=='A' → j=2
j=2: s[2]=='A' → j=3
j=3: j == n → 停止
j=3, j == n（后面没有不同字符）
```

### 第六部分：判断是否扩展

```cpp
        bool expand = false;
        if (j == n) {
            // 后面全是相同字符 → 不扩展
            expand = false;
        } else if (s[i] < s[j]) {
            // 后面的不同字符更大 → 扩展
            expand = true;
        } else {
            // 后面的不同字符更小 → 不扩展
            expand = false;
        }
```

**逐行解释：**

| 行 | 代码 | 含义 |
|----|------|------|
| 1 | `bool expand = false` | 默认不扩展 |
| 2 | `if (j == n)` | 如果后面没有不同字符（全是相同字符） |
| 3 | `expand = false` | 不扩展 |
| 4 | `else if (s[i] < s[j])` | 如果当前字符 < 后面第一个不同字符 |
| 5 | `expand = true` | 扩展 |
| 6 | `else` | 如果当前字符 > 后面第一个不同字符 |
| 7 | `expand = false` | 不扩展 |

**三种情况总结：**

| 条件 | expand | 原因 |
|------|--------|------|
| `j == n`（后面全是相同字符） | false | 扩展只会让字符串变长，不会变小 |
| `s[i] < s[j]`（当前字符更小） | true | 扩展可以让更小的字符多出现一次 |
| `s[i] > s[j]`（当前字符更大） | false | 扩展会让更大的字符多出现一次 |

**示例：** `s = "ABC"`

| i | s[i] | j | s[j] | 比较 | expand |
|---|------|---|------|------|--------|
| 0 | 'A' | 1 | 'B' | 'A' < 'B' | true |
| 1 | 'B' | 2 | 'C' | 'B' < 'C' | true |
| 2 | 'C' | - | - | 最后一个 | false |

**结果：** A 扩展，B 扩展，C 不扩展 → `AABBC`

---

### 第七部分：执行扩展或不扩展

```cpp
        if (expand) {
            result += s[i];
            result += s[i];
        } else {
            result += s[i];
        }
```

| 条件 | 操作 | 说明 |
|------|------|------|
| `expand == true` | `result += s[i]` 两次 | 扩展成2个 |
| `expand == false` | `result += s[i]` 一次 | 保持1个 |

}

## 四、决策逻辑图解

```
对于位置 i：

找到第一个不等于 s[i] 的字符位置 j

        ┌─────────────────────────────────────┐
        │          j 存在吗？                 │
        └─────────────┬───────────────────────┘
                      │
        ┌─────────────┴─────────────┐
        │                           │
        ▼                           ▼
   不存在（j==n）              存在（j<n）
        │                           │
        ▼                           ▼
   不扩展              ┌─────────────────────────┐
                     │ s[i] 和 s[j] 比较       │
                     └───────────┬─────────────┘
                                 │
                 ┌───────────────┴───────────────┐
                 │                               │
                 ▼                               ▼
           s[i] < s[j]                     s[i] > s[j]
                 │                               │
                 ▼                               ▼
             扩展                           不扩展
```

## 五、总结

| 要点 | 说明 |
|------|------|
| **核心** | 找第一个不同字符，比较大小决定是否扩展 |
| **扩展条件** | 当前字符 < 后面第一个不同字符 |
| **不扩展条件** | 当前字符 >= 后面第一个不同字符，或后面全是相同字符 |
| **最后字符** | 永远不扩展 |
| **时间复杂度** | O(n²)（最坏情况全是相同字符） |

**核心思想：** 跳过连续相同字符，看第一个不同的字符。如果当前字符比它小，扩展可以让小字符更早出现，使字典序更小。
