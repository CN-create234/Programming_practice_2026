# EOJ-1020 括号匹配 - 完整解析（中文版）

# 1020. 平衡括号

单点时限: 2.0 sec  
内存限制: 256 MB

括号被认为是以下字符之一：`(`，`)`，`{`，`}`，`[`，`]`。

如果左括号 `(`，`[` 或 `{` 出现在右括号 `)`，`]` 或 `}` 的左边，并且是相同类型，则这两个括号被视为匹配的一对。匹配的括号对共有三种类型：`()`，`[]` 和 `{}`。

如果一对匹配括号所包含的括号集合不匹配，则该匹配括号对是不平衡的。例如，`{[(])}` 是不平衡的，因为 `[` 和 `]` 之间包含的内容不平衡。具体来说，圆括号对 `()` 包含了一个不平衡的右方括号 `]`，而方括号对 `[]` 包含了一个不平衡的左圆括号 `(`。

根据这个逻辑，如果满足以下条件，我们称一个括号序列是平衡的：

1. 不包含未匹配的括号。
2. 匹配括号对内部所包含的括号子集本身也是一组匹配的括号对。

给定若干括号字符串，判断每个括号序列是否平衡。如果字符串是平衡的，输出 `YES`；否则输出 `NO`。

---

## 输入格式

第一行包含一个整数 `T`，表示字符串的数量。

接下来 `T` 行，每行包含一个字符串 `s`，表示一个括号序列。

保证 `1 ≤ T ≤ 100`，`1 ≤ |s| ≤ 1000`，其中 `|s|` 是序列的长度。

序列中的每个字符都是括号字符（即 `(`，`)`，`{`，`}`，`[`，`]` 之一）。

---

## 输出格式

对于每个字符串，在新的一行中输出该括号序列是否平衡。如果平衡，输出 `YES`；否则输出 `NO`。

---

## 样例

```
输入：
3
{[()]}
{[(])}
{{[[(())]]}}

输出：
YES
NO
YES
```

---

## 提示

- 字符串 `{[()]}` 满足平衡的两个条件，因此输出 `YES`。
- 字符串 `{[(])}` 不平衡，因为匹配括号对 `[ ]` 和 `( )` 内部包含的括号不匹配。
- 字符串 `{{[[(())]]}}` 满足平衡条件，因此输出 `YES`。

## 一、题目理解

### 什么是括号匹配？

一个括号序列是**平衡的**，需要满足：
1. 每个左括号都有对应的右括号，且类型相同
2. 括号正确**嵌套**（就像数学表达式一样）

### 括号类型

| 左括号 | 右括号 |
|--------|--------|
| `(` | `)` |
| `[` | `]` |
| `{` | `}` |

### 示例

| 字符串 | 是否平衡 | 原因 |
|--------|----------|------|
| `{[()]}` | ✅ 是 | 正确嵌套 |
| `{[(])}` | ❌ 否 | 嵌套错误：`]` 先于 `)` 关闭 |
| `{{[[(())]]}}` | ✅ 是 | 正确嵌套 |

---

## 二、解题思路

### 核心思想：栈（Stack）

使用**栈**来追踪左括号：

1. **遇到左括号** `(`、`[`、`{` → 压入栈
2. **遇到右括号** `)`、`]`、`}` → 检查是否匹配栈顶
   - 如果栈为空 → **不平衡**（没有匹配的左括号）
   - 如果栈顶不匹配 → **不平衡**（类型错误）
   - 如果匹配 → 弹出栈顶
3. **处理完所有字符后**：
   - 栈为空 → 平衡 ✅
   - 栈不为空 → 不平衡 ❌

## 三、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;

    while (T--) {
        string s;
        cin >> s;

        stack<char> st;
        bool balanced = true;

        for (char ch : s) {
            if (ch == '(' || ch == '[' || ch == '{') {
                st.push(ch);
            } else {
                // 遇到右括号
                if (st.empty()) {
                    balanced = false;
                    break;
                }

                char top = st.top();
                st.pop();

                if ((ch == ')' && top != '(') ||
                    (ch == ']' && top != '[') ||
                    (ch == '}' && top != '{')) {
                    balanced = false;
                    break;
                }
            }
        }

        if (balanced && st.empty()) {
            cout << "YES" << endl;
        } else {
            cout << "NO" << endl;
        }
    }

    return 0;
}
```

## 五、执行示例

### 示例1：`{[()]}`

| 步骤 | 字符 | 栈 | 操作 |
|------|------|----|------|
| 1 | `{` | `{` | 压入 `{` |
| 2 | `[` | `{ [` | 压入 `[` |
| 3 | `(` | `{ [ (` | 压入 `(` |
| 4 | `)` | `{ [` | `)` 匹配 `(` → 弹出 |
| 5 | `]` | `{` | `]` 匹配 `[` → 弹出 |
| 6 | `}` | 空 | `}` 匹配 `{` → 弹出 |

最终：栈为空 → **YES** ✅

---

### 示例2：`{[(])}`

| 步骤 | 字符 | 栈 | 操作 |
|------|------|----|------|
| 1 | `{` | `{` | 压入 `{` |
| 2 | `[` | `{ [` | 压入 `[` |
| 3 | `(` | `{ [ (` | 压入 `(` |
| 4 | `]` | `{ [ (` | `]` 不匹配 `(` → **NO** ❌ |

---

### 示例3：`{{[[(())]]}}`

| 步骤 | 字符 | 栈 | 操作 |
|------|------|----|------|
| 1-8 | 全部压入 | `{ { [ [ ( ( ) ) ] ] } }` | 全部匹配 |
| 最终 | 空 | **YES** ✅ |

## 七、关键点总结

| 要点 | 说明 |
|------|------|
| **数据结构** | 栈（后进先出） |
| **左括号** | 压入栈 |
| **右括号** | 弹出栈顶并检查匹配 |
| **不平衡情况1** | 右括号出现时栈为空 |
| **不平衡情况2** | 右括号不匹配栈顶 |
| **不平衡情况3** | 处理完后栈不为空 |
| **平衡** | 所有括号匹配且栈为空 |

---

## 八、常见错误

| 错误 | 正确做法 |
|------|----------|
| 忘记检查栈是否为空 | 访问 `st.top()` 前检查 `st.empty()` |
| 没有检查最终栈是否为空 | 循环结束后检查 `st.empty()` |
| 匹配对错误 | 使用正确配对：`()`、`[]`、`{}` |
| 没有提前跳出循环 | 发现不平衡时用 `break` 跳出 |

---

## 九、总结

**核心思想：** 用栈追踪左括号。遇到右括号时，最近的左括号必须与它匹配。处理完所有字符后，栈必须为空，序列才是平衡的。
