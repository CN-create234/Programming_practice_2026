# 1043.矩形个数

---

## 宏观理解

### 问题本质
给定一个 r×c 的 0/1 矩阵，统计所有**子矩形**中，1 的个数 ≥ k 的矩形数量。

### 关键点
- 单个元素也算一个矩形
- 矩形边界平行于矩阵边界
- 矩阵最大 10×10，总格子数最多 100
- 可以暴力枚举所有矩形

---

## 总体思路

### 方法一：暴力枚举所有矩形

1. 构建 r×c 的 0/1 矩阵
2. 枚举所有可能的矩形（四个边界）
3. 统计每个矩形中 1 的个数
4. 如果 ≥ k，答案 +1

### 矩形表示
一个矩形由四条边界决定：
- 上边界：`top` (0 到 r-1)
- 下边界：`bottom` (top 到 r-1)
- 左边界：`left` (0 到 c-1)
- 右边界：`right` (left 到 c-1)

### 总矩形数
```
矩形总数 = r*(r+1)/2 * c*(c+1)/2
r=10, c=10 时：55 * 55 = 3025 个
```
暴力枚举完全可行。

---

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;
    
    for (int caseNum = 0; caseNum < T; caseNum++) {
        int r, c, n, k;
        cin >> r >> c >> n >> k;
        
        // 构建矩阵，初始全 0
        vector<vector<int>> grid(r, vector<int>(c, 0));
        
        // 读入 1 的位置
        for (int i = 0; i < n; i++) {
            int x, y;
            cin >> x >> y;
            grid[x - 1][y - 1] = 1;  // 转为 0 索引
        }
        
        // 计算前缀和，方便快速查询矩形内 1 的个数
        vector<vector<int>> prefix(r + 1, vector<int>(c + 1, 0));
        for (int i = 1; i <= r; i++) {
            for (int j = 1; j <= c; j++) {
                prefix[i][j] = grid[i - 1][j - 1] 
                             + prefix[i - 1][j] 
                             + prefix[i][j - 1] 
                             - prefix[i - 1][j - 1];
            }
        }
        
        // 查询矩形内 1 的个数
        auto getSum = [&](int top, int bottom, int left, int right) {
            return prefix[bottom + 1][right + 1] 
                 - prefix[top][right + 1] 
                 - prefix[bottom + 1][left] 
                 + prefix[top][left];
        };
        
        int ans = 0;
        
        // 枚举所有矩形
        for (int top = 0; top < r; top++) {
            for (int bottom = top; bottom < r; bottom++) {
                for (int left = 0; left < c; left++) {
                    for (int right = left; right < c; right++) {
                        int cnt = getSum(top, bottom, left, right);
                        if (cnt >= k) {
                            ans++;
                        }
                    }
                }
            }
        }
        
        cout << "case #" << caseNum << ":" << endl;
        cout << ans << endl;
    }
    
    return 0;
}
```

## 逐行详细解释

### 1. 读取测试用例数
```cpp
int T;
cin >> T;
```
- 读取 T，表示有多少个问题

---

### 2. 外层循环：处理每个问题
```cpp
for (int caseNum = 0; caseNum < T; caseNum++) {
```
- `caseNum` 从 0 开始，用于输出 `case #0:`

---

### 3. 读取矩阵信息
```cpp
int r, c, n, k;
cin >> r >> c >> n >> k;
```
- `r`：行数
- `c`：列数
- `n`：1 的个数
- `k`：阈值

---

### 4. 构建矩阵
```cpp
vector<vector<int>> grid(r, vector<int>(c, 0));

for (int i = 0; i < n; i++) {
    int x, y;
    cin >> x >> y;
    grid[x - 1][y - 1] = 1;
}
```
- 创建 r×c 的矩阵，初始全 0
- 读入每个 1 的位置，在矩阵中标记为 1
- `x - 1` 和 `y - 1` 将 1-based 索引转为 0-based

---

### 5. 计算二维前缀和
```cpp
vector<vector<int>> prefix(r + 1, vector<int>(c + 1, 0));
for (int i = 1; i <= r; i++) {
    for (int j = 1; j <= c; j++) {
        prefix[i][j] = grid[i - 1][j - 1] 
                     + prefix[i - 1][j] 
                     + prefix[i][j - 1] 
                     - prefix[i - 1][j - 1];
    }
}
```

### 前缀和原理
```
prefix[i][j] = 从 (0,0) 到 (i-1,j-1) 的矩形内 1 的个数

公式：
prefix[i][j] = grid[i-1][j-1] + prefix[i-1][j] + prefix[i][j-1] - prefix[i-1][j-1]

图解：
┌─────────┬─────────┐
│         │         │
│   A     │   B     │  ← prefix[i-1][j]
│         │         │
├─────────┼─────────┤
│         │ grid    │
│   C     │ [i-1]   │  ← grid[i-1][j-1]
│         │ [j-1]   │
└─────────┴─────────┘
  ↑
prefix[i][j-1]

prefix[i][j] = grid[i-1][j-1] + A + B + C
              = grid + prefix[i-1][j] + prefix[i][j-1] - prefix[i-1][j-1]
```

---

### 6. 查询矩形内 1 的个数
```cpp
auto getSum = [&](int top, int bottom, int left, int right) {
    return prefix[bottom + 1][right + 1] 
         - prefix[top][right + 1] 
         - prefix[bottom + 1][left] 
         + prefix[top][left];
};
```

### 查询原理
```
矩形范围：
- 行：[top, bottom]
- 列：[left, right]

使用前缀和查询：
sum = prefix[bottom+1][right+1] 
    - prefix[top][right+1] 
    - prefix[bottom+1][left] 
    + prefix[top][left]

图解：
┌─────────┬─────────┐
│         │         │
│    A    │    B    │
│         │         │
├─────────┼─────────┤
│         │  目标   │
│    C    │  矩形   │
│         │         │
└─────────┴─────────┘

目标矩形 = 全部 - A - B - C
         = prefix[bottom+1][right+1]
         - prefix[top][right+1]     (A+B)
         - prefix[bottom+1][left]   (A+C)
         + prefix[top][left]        (A，被减了两次，加回来)
```

---

### 7. 枚举所有矩形
```cpp
int ans = 0;

for (int top = 0; top < r; top++) {
    for (int bottom = top; bottom < r; bottom++) {
        for (int left = 0; left < c; left++) {
            for (int right = left; right < c; right++) {
                int cnt = getSum(top, bottom, left, right);
                if (cnt >= k) {
                    ans++;
                }
            }
        }
    }
}
```

### 枚举范围
- `top`：0 到 r-1
- `bottom`：top 到 r-1（确保非空矩形）
- `left`：0 到 c-1
- `right`：left 到 c-1（确保非空矩形）

### 每个矩形
```
top=0, bottom=2, left=1, right=3 表示：
行 0-2，列 1-3 的子矩阵
```

---

### 8. 输出
```cpp
cout << "case #" << caseNum << ":" << endl;
cout << ans << endl;
```
- 先输出 `case #0:`，再输出答案
- 格式与样例一致

## 关键点总结

| 知识点 | 说明 |
|--------|------|
| 二维前缀和 | O(1) 时间查询任意矩形内 1 的个数 |
| 枚举矩形 | 四重循环枚举所有可能的矩形 |
| 矩形总数 | r×(r+1)/2 × c×(c+1)/2 |
| 复杂度 | O(r²×c²) ≈ 55×55 = 3025 |

---

## 一句话总结

**用二维前缀和预处理，然后暴力枚举所有矩形，统计 1 的个数 ≥ k 的矩形数量。**
