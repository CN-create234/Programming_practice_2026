# EOJ-1010 邮件地址排序 - 完整解析

## 一、题目理解

### 排序规则

邮件地址格式：`用户名@主机域名`

排序规则：
1. **优先按主机域名**升序排列（字典序）
2. **如果主机域名相同**，按用户名降序排列（字典序）

### 样例分析

```
8
23485@qq.com
rieruer@163.com
39489384@qq.com
eruie@ecnu.edu.cn
rtff@163.com
84934804@qq.com
fdll@ecnu.edu.cn
598695@qq.com
```

**分组：**

| 主机域名 | 用户名列表 |
|---------|-----------|
| 163.com | rieruer, rtff |
| ecnu.edu.cn | eruie, fdll |
| qq.com | 23485, 39489384, 84934804, 598695 |

**排序：**
1. 主机域名升序：`163.com < ecnu.edu.cn < qq.com`
2. 同一主机域名，用户名降序：
   - 163.com：`rtff > rieruer`（r t > r i → t > i）
   - ecnu.edu.cn：`fdll > eruie`（f > e）
   - qq.com：`84934804 > 598695 > 39489384 > 23485`

**输出：**
```
rtff@163.com
rieruer@163.com
fdll@ecnu.edu.cn
eruie@ecnu.edu.cn
84934804@qq.com
598695@qq.com
39489384@qq.com
23485@qq.com
```

## 二、解题思路

### 核心步骤

```
1. 读取 n 和所有邮件地址
2. 对每个邮件地址，分割出用户名和主机域名
3. 自定义排序：
   a. 先比较主机域名（升序）
   b. 主机域名相同，比较用户名（降序）
4. 输出排序后的邮件地址
```

### 分割方法

```
邮件地址：用户名@主机域名
例如：23485@qq.com
用户名：23485
主机域名：qq.com

分割方法：
- 找到 '@' 的位置
- 左边是用户名
- 右边是主机域名
```

## 三、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Email {
    string full;    // 完整邮件地址
    string user;    // 用户名
    string domain;  // 主机域名
};

bool compareEmail(const Email& a, const Email& b) {
    // 规则1：先按主机域名升序
    if (a.domain != b.domain) {
        return a.domain < b.domain;
    }
    // 规则2：主机域名相同，按用户名降序
    return a.user > b.user;
}

int main() {
    int n;
    cin >> n;

    vector<Email> emails(n);

    for (int i = 0; i < n; i++) {
        cin >> emails[i].full;

        // 找到 @ 的位置
        size_t atPos = emails[i].full.find('@');

        // 分割用户名和主机域名
        emails[i].user = emails[i].full.substr(0, atPos);
        emails[i].domain = emails[i].full.substr(atPos + 1);
    }

    // 排序
    sort(emails.begin(), emails.end(), compareEmail);

    // 输出
    for (int i = 0; i < n; i++) {
        cout << emails[i].full << endl;
    }

    return 0;
}
```

## 四、逐行详细解释

### 1. 结构体定义

```cpp
struct Email {
    string full;    // 完整邮件地址
    string user;    // 用户名
    string domain;  // 主机域名
};
```

**作用：** 将邮件地址的三个信息打包在一起，方便排序时访问。

---

### 2. 自定义比较函数

```cpp
bool compareEmail(const Email& a, const Email& b) {
    // 规则1：先按主机域名升序
    if (a.domain != b.domain) {
        return a.domain < b.domain;
    }
    // 规则2：主机域名相同，按用户名降序
    return a.user > b.user;
}
```

**解释：**

| 条件 | 返回值 | 含义 |
|------|--------|------|
| `a.domain != b.domain` | `a.domain < b.domain` | 主机域名不同，升序 |
| `a.domain == b.domain` | `a.user > b.user` | 主机域名相同，用户名降序 |

**注意：** `>` 是降序（大的排前面），`<` 是升序（小的排前面）。

```cpp

```

- 读取 `n`
- 创建 `vector<Email>` 大小为 `n`
- 读取每个邮件地址存入 `full`

---

### 4. 分割用户名和主机域名

```cpp
        size_t atPos = emails[i].full.find('@');

        emails[i].user = emails[i].full.substr(0, atPos);
        emails[i].domain = emails[i].full.substr(atPos + 1);
```

**`find('@')`：** 返回 `@` 的位置

**`substr(0, atPos)`：** 从 0 开始，取 `atPos` 个字符 → 用户名

**`substr(atPos + 1)`：** 从 `@` 后面到末尾 → 主机域名

**示例：** `"23485@qq.com"`

```

atPos = 5
user = full.substr(0, 5) = "23485"
domain = full.substr(6) = "qq.com"
```

## 六、完整代码（含注释）

```cpp
#include <bits/stdc++.h>
using namespace std;

// 邮件地址结构体
struct Email {
    string full;    // 完整邮件地址
    string user;    // 用户名（@左边）
    string domain;  // 主机域名（@右边）
};

// 自定义比较函数
bool compareEmail(const Email& a, const Email& b) {
    // 1. 主机域名升序
    if (a.domain != b.domain) {
        return a.domain < b.domain;
    }
    // 2. 主机域名相同，用户名降序
    return a.user > b.user;
}

int main() {
    int n;
    cin >> n;

    vector<Email> emails(n);

    for (int i = 0; i < n; i++) {
        cin >> emails[i].full;

        // 查找 '@' 的位置
        size_t atPos = emails[i].full.find('@');

        // 分割用户名和主机域名
        emails[i].user = emails[i].full.substr(0, atPos);
        emails[i].domain = emails[i].full.substr(atPos + 1);
    }

    // 排序
    sort(emails.begin(), emails.end(), compareEmail);

    // 输出
    for (int i = 0; i < n; i++) {
        cout << emails[i].full << endl;
    }

    return 0;
}
```

## 七、关键知识点

### 1. `find()` 函数

```cpp
size_t pos = str.find('@');
```
- 在字符串中查找指定字符
- 返回第一次出现的位置
- 如果找不到，返回 `string::npos`

### 2. `substr()` 函数

```cpp
str.substr(pos, count);  // 从 pos 开始，取 count 个字符
str.substr(pos);         // 从 pos 开始到末尾
```

### 3. `sort` 自定义比较

```cpp
sort(vec.begin(), vec.end(), compareFunction);
```
- 比较函数返回 `true` 表示 a 排在 b 前面
- `return a.domain < b.domain` → 升序
- `return a.user > b.user` → 降序

### 4. 字典序比较

```cpp
string a = "abc";
string b = "abd";
a < b  // true（a 在 b 前面）
```

## 八、复杂度分析

| 操作 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 读取 | O(n) | O(1) |
| 分割 | O(n × L) | O(1) |
| 排序 | O(n log n) | O(1) |
| **总体** | **O(n log n)** | **O(n)** |

其中 L 是邮件地址平均长度。

## 九、总结

| 要点 | 说明 |
|------|------|
| **排序规则** | 主机域名升序 → 用户名降序 |
| **分割方式** | 用 `find('@')` 和 `substr()` |
| **数据结构** | 用结构体存储完整地址、用户名、主机域名 |
| **自定义比较** | 先比较域名，再比较用户名 |
| **输出** | 输出完整邮件地址 |

**核心思想：分割后按指定规则排序，排序时携带完整地址信息。**
