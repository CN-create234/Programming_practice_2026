# EOJ-1027 单词表 - 完整解析

---

## 一、题目理解

### 问题描述

从英文文本中提取所有单词（去重），按字典序排序后输出。

### 分隔符

单词之间用以下字符分隔：
- 空格 ` `
- 逗号 `,`
- 句号 `.`
- 感叹号 `!`
- 问号 `?`

### 单词定义

- 只包含 26 个小写英文字母
- 文本中只有小写字母和上述分隔符

### 输入格式

- 第1行：整数 T（问题数）
- 接下来 T 行：每行一段文本（长度 ≤ 500）

---

## 二、解题思路

### 核心步骤

```
1. 读取 T
2. 用 cin.ignore() 跳过换行符
3. 对于每段文本：
   a. 用 getline 读取整行（可能包含空格）
   b. 遍历每个字符，按分隔符分割单词
   c. 将单词存入 set（自动去重 + 排序）
   d. 输出 set 中的单词
```

### 为什么用 set？

- `set` 自动去重
- `set` 自动按字典序排序
- 遍历 `set` 即可得到排序后的单词表

## 三、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

// 判断是否为分隔符
bool isDelimiter(char ch) {
    return ch == ' ' || ch == ',' || ch == '.' ||
           ch == '!' || ch == '?';
}

int main() {
    int T;
    cin >> T;
    cin.ignore();  // 跳过第一行末尾的换行符

    for (int caseNum = 0; caseNum < T; caseNum++) {
        string line;
        getline(cin, line);

        set<string> words;
        string word = "";

        for (char ch : line) {
            if (isDelimiter(ch)) {
                // 遇到分隔符，处理前一个单词
                if (!word.empty()) {
                    words.insert(word);
                    word = "";
                }
            } else {
                // 是字母，追加到当前单词
                word += ch;
            }
        }

        // 处理最后一个单词（如果行末没有分隔符）
        if (!word.empty()) {
            words.insert(word);
        }

        cout << "case #" << caseNum << ":" << endl;

        bool first = true;
        for (const string& w : words) {
            if (!first) cout << " ";
            cout << w;
            first = false;
        }
        cout << endl;
    }

    return 0;
}
```

## 四、逐行详细解释

### 1. 头文件

```cpp
#include <bits/stdc++.h>
using namespace std;
```

万能头文件。

---

### 2. 判断分隔符函数

```cpp
bool isDelimiter(char ch) {
    return ch == ' ' || ch == ',' || ch == '.' ||
           ch == '!' || ch == '?';
}
```

**作用：** 判断字符是否为分隔符。

| 字符 | 类型 |
|------|------|
| 空格 ` ` | 分隔符 |
| 逗号 `,` | 分隔符 |
| 句号 `.` | 分隔符 |
| 感叹号 `!` | 分隔符 |
| 问号 `?` | 分隔符 |
| 其他（a-z） | 单词字符 |

---

### 3. 主函数 - 读取 T

```cpp
int main() {
    int T;
    cin >> T;
    cin.ignore();
```

- `cin >> T`：读取问题数
- `cin.ignore()`：跳过第一行末尾的换行符

**为什么需要 `cin.ignore()`？**

```
输入：
3          ← cin >> T 读取 3，但换行符还在缓冲区
ask not... ← 如果不 ignore，getline 会读到空行！
```

---

### 4. 循环处理每个问题

```cpp
    for (int caseNum = 0; caseNum < T; caseNum++) {
        string line;
        getline(cin, line);
```

- `caseNum` 从 0 开始（输出 `case #0:`）
- `getline(cin, line)`：读取整行，**包含空格**

---

### 5. 初始化 set 和当前单词

```cpp
        set<string> words;
        string word = "";
```

- `set<string> words`：存储所有单词（自动去重 + 排序）
- `word`：当前正在累积的单词

---

### 6. 遍历字符串，分割单词

```cpp
        for (char ch : line) {
            if (isDelimiter(ch)) {
                if (!word.empty()) {
                    words.insert(word);
                    word = "";
                }
            } else {
                word += ch;
            }
        }
```

**逻辑：**

| 条件 | 操作 |
|------|------|
| 当前字符是分隔符 | 处理之前累积的单词，存入 set，清空 word |
| 当前字符不是分隔符 | 将字符追加到 word |

---

### 7. 处理最后一个单词

```cpp
        if (!word.empty()) {
            words.insert(word);
        }
```

**为什么需要这一步？**

```
输入："ask not"
遍历过程：
'a','s','k' → word="ask"
' ' → 处理 "ask"，word=""
'n','o','t' → word="not"
遍历结束 → 需要单独处理 "not"
```

---

### 8. 输出

```cpp
        cout << "case #" << caseNum << ":" << endl;

        bool first = true;
        for (const string& w : words) {
            if (!first) cout << " ";
            cout << w;
            first = false;
        }
        cout << endl;
    }

    return 0;
}
```

- 先输出 `case #编号:`
- 遍历 `set`（已自动排序），输出每个单词
- 用 `first` 控制空格：第一个单词前不加空格，后续单词前加空格

## 五、执行示例

### 样例1

```
输入：ask not what your country can do for you,ask what you can do for your country.

存入 set（去重）：

ask can country do for not what you your ✓
```

## 六、用 stringstream 的替代写法

```cpp
#include <bits/stdc++.h>
using namespace std;

bool isDelimiter(char ch) {
    return ch == ' ' || ch == ',' || ch == '.' ||
           ch == '!' || ch == '?';
}

int main() {
    int T;
    cin >> T;
    cin.ignore();

    for (int caseNum = 0; caseNum < T; caseNum++) {
        string line;
        getline(cin, line);

        // 将分隔符替换为空格
        for (char& ch : line) {
            if (isDelimiter(ch)) {
                ch = ' ';
            }
        }

        set<string> words;
        stringstream ss(line);
        string word;
        while (ss >> word) {
            words.insert(word);
        }

        cout << "case #" << caseNum << ":" << endl;
        bool first = true;
        for (const string& w : words) {
            if (!first) cout << " ";
            cout << w;
            first = false;
        }
        cout << endl;
    }

    return 0;
}
```

**优点：** 用 `stringstream` 自动按空格分割，代码更简洁。

## 七、复杂度分析

| 操作 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 遍历字符串 | O(n) | O(1) |
| set 插入 | O(log m) | O(m) |
| **总体** | **O(n log m)** | **O(m)** |

其中 n 是文本长度，m 是不同单词的个数。

## 八、关键点总结

| 要点 | 说明 |
|------|------|
| **数据结构** | `set<string>` 自动去重 + 排序 |
| **分隔符** | 空格、逗号、句号、感叹号、问号 |
| **读取方式** | `getline` 读取包含空格的整行 |
| **单词分割** | 遍历字符，遇分隔符处理单词 |
| **最后一个单词** | 循环结束后单独处理 |

**核心思想：** 遍历文本，按分隔符分割单词，存入 set 自动去重排序。
