# EOJ-1008 点对 - 完整解析

---

## 一、题目理解

### 问题描述

直线上有 `n` 个点（n 为偶数），需要将这 `n` 个点两两配对，分成 `n/2` 组点对 `(a_i, b_i)`，使得所有点对的距离之和最小。

**目标：** 最小化 `Σ|a_i - b_i|`

### 样例验证

**样例1：**
```
n = 4
点：1 2 3 4

配对方案： (1,2) + (3,4) = 1 + 1 = 2
方案： (1,3) + (2,4) = 2 + 2 = 4
方案： (1,4) + (2,3) = 3 + 1 = 4

最小值为 2 ✓
```

**样例2：**
```
n = 4
点：0 0 0 0

配对方案：任意配对，距离都为 0
最小值为 0 ✓
```

---

## 二、核心规律

### 关键结论

**将排序后的点按相邻两两配对（第1和第2，第3和第4，...），距离之和最小。**

### 为什么？

对于一维直线上的一组点，要最小化两两配对的距离之和，**最优策略是将排序后的点按相邻顺序配对**。

**证明思路：**

1. 假设有 4 个点：`a ≤ b ≤ c ≤ d`
2. 可能的配对方案：
   - `(a,b) + (c,d)` → 距离：`(b-a) + (d-c)`
   - `(a,c) + (b,d)` → 距离：`(c-a) + (d-b)`
   - `(a,d) + (b,c)` → 距离：`(d-a) + (c-b)`

3. 比较：
```
(b-a) + (d-c) = b - a + d - c
(c-a) + (d-b) = c - a + d - b
(d-a) + (c-b) = d - a + c - b

第一种 - 第二种：
(b-a+d-c) - (c-a+d-b) = b - a + d - c - c + a - d + b = 2b - 2c ≤ 0
所以第一种 ≤ 第二种

第一种 - 第三种：
(b-a+d-c) - (d-a+c-b) = b - a + d - c - d + a - c + b = 2b - 2c ≤ 0
所以第一种 ≤ 第三种
```

因此，相邻配对是最优的。

---

## 三、解题步骤

```
1. 读入 n 和 n 个点的坐标
2. 将数组排序（从小到大）
3. 相邻两两配对：sum += arr[i+1] - arr[i]（i = 0, 2, 4, ...）
4. 输出结果
```

---

## 四、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;

    vector<long long> points(n);
    for (int i = 0; i < n; i++) {
        cin >> points[i];
    }

    sort(points.begin(), points.end());

    long long sum = 0;
    for (int i = 0; i < n; i += 2) {
        sum += points[i + 1] - points[i];
    }

    cout << sum << endl;

    return 0;
}
```

### 第三部分：排序


将点按坐标从小到大排序。

### 第四部分：相邻配对求和

```cpp
    long long sum = 0;
    for (int i = 0; i < n; i += 2) {
        sum += points[i + 1] - points[i];
    }
```

- 从索引 0 开始，每次跳 2 步
- `points[i+1] - points[i]` 是相邻两点的距离
- 累加到 `sum`

**示例：** `points = [1, 2, 3, 4]`

```
i=0: sum += 2 - 1 = 1
i=2: sum += 4 - 3 = 1
sum = 2
```

```cpp

```

输出最小距离和。

---

## 六、为什么不用处理负数？

排序后，`points[i+1]` 总是大于等于 `points[i]`，所以 `points[i+1] - points[i]` 总是非负的。

**示例：** `[-3, -1, 2, 5]`

```
排序后：[-3, -1, 2, 5]
配对：(-3,-1) + (2,5) = 2 + 3 = 5
绝对值：|-3 - (-1)| + |2 - 5| = 2 + 3 = 5 ✓
```

## 八、完整代码（含注释）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;

    vector<long long> points(n);
    for (int i = 0; i < n; i++) {
        cin >> points[i];
    }

    // 排序后相邻配对是最优的
    sort(points.begin(), points.end());

    long long sum = 0;
    for (int i = 0; i < n; i += 2) {
        sum += points[i + 1] - points[i];
    }

    cout << sum << endl;

    return 0;
}
```

## 九、总结

| 要点 | 说明 |
|------|------|
| **核心结论** | 排序后相邻两两配对，距离之和最小 |
| **排序** | 使用 `sort` 从小到大排序 |
| **配对** | `(arr[0], arr[1])`, `(arr[2], arr[3])`, ... |
| **数据类型** | 用 `long long` 防止溢出 |
| **时间复杂度** | O(n log n) |

**核心思想：一维直线上，最优配对方案是排序后的相邻配对。**
