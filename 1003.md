# EOJ-1003 神秘信息 - 完整解析
```txt
神秘信息
单点时限: 2.0 sec
内存限制: 256 MB

未来的某一天，人类登陆火星。在火星上发现一组由各种符号表示的神秘信息。研究发现：每种符号表示一个数码，神秘信息是一个整数。但是不知道每个符号表示的是哪个数码，也不知道它是几进制数 (只知道不可能是一进制)，但知道相同的符号表示相同的数码，不同的符号表示不同的数码。

符号总计有 62 种，为方便起见，每种符号分别用大小写英文字母和数字 0-9 表示。

现在我们想要知道神秘信息表示的最小整数值。

例如 :

11001001 最小整数值和人类的二进制数一样，为 201；

cats 最小整数值为 75(设是 4 进制数，c 表示 1，a 表示 0，t 表示 2，s 表示 3)；

pig 最小整数值为 11(设是 3 进制数，p 表示 1，i 表示 0，g 表示 2)。

注意：神秘信息表示的数必须是有意义的，第一位不能为 0。

输入格式
第 1 行：整数 T (1 <= T <= 10) 为问题数。

第 2 行：第一个问题的数据。一个字符串表示的神秘信息，长度范围为 [1,60]。

第 3 至 T+1 行：后面问题的数据，格式与第一个问题相同。

输出格式
对于每个问题，输出一行问题的编号（0 开始编号，格式：case #0: 等），然后在一行中输出神秘信息表示的最小整数值（输入数据保证输出的整数不超过 10^18）。

样例
Input
3
11001001
cats
zig

Output
case #0:
201
case #1:
75
case #2:
11
```
## 一、题目理解

### 核心问题
给定一个由未知符号组成的字符串，每个符号代表一个数码（0-9或更大），要找到**这个字符串能表示的最小十进制整数**。

### 关键规则
1. **相同符号 = 相同数码**，不同符号 = 不同数码
2. **进制数 = 不同符号的个数**（但不能是1进制）
3. **第一位不能为0**（最高位有意义）
4. 符号共62种：`0-9, a-z, A-Z`

### 示例分析

**示例1：** `11001001`
```
不同符号：'1', '0' → 2种 → 二进制
'1' 不能为0（第一位），所以 '1'=1, '0'=0
11001001₂ = 201₁₀ ✓
```

**示例2：** `cats`
```
不同符号：c, a, t, s → 4种 → 四进制
c 不能为0（第一位）→ c=1
a=0, t=2, s=3（剩余数码从小到大分配）
cats₄ = 1023₄ = 1×4³ + 0×4² + 2×4 + 3 = 64+0+8+3 = 75 ✓
```

**示例3：** `pig`
```
不同符号：p, i, g → 3种 → 三进制
p 不能为0 → p=1
i=0, g=2
pig₃ = 102₃ = 1×9 + 0×3 + 2 = 11 ✓
```

---

## 二、解题思路

### 核心策略：让数值最小

要让整个数最小，需要：
1. **进制尽量小** → 不同符号数量尽量少
2. **高位数码尽量小** → 首位分配最小非零数码（1）
3. **剩余数码从小到大分配**

### 贪心分配规则

| 优先级 | 数码值 | 分配对象 |
|--------|--------|----------|
| 1 | 1 | 第一个字符（最高位，不能为0） |
| 2 | 0 | 第二个出现的不同字符 |
| 3 | 2 | 第三个出现的不同字符 |
| 4 | 3 | 第四个出现的不同字符 |
| ... | ... | ... |

**为什么第二个给0而不是2？**
- 为了让数最小，高位要尽可能小
- 0 是最小的数码，应该给**次高位**（第二位出现的字符）

### 特殊情况
- 如果只有1种不同字符 → 进制为2（不能是1进制）
- 例如：`111` → 只有'1'一种 → 二进制 → 最小值为 111₂ = 7

---

## 三、算法步骤

```
1. 遍历字符串，记录每个字符首次出现的顺序
2. 统计不同字符个数 R（即为进制）
3. 如果 R == 1，R = 2（不能是1进制）
4. 分配数码：
   - 第一个字符（最高位）→ 1
   - 第二个出现的字符 → 0
   - 第三个出现的字符 → 2
   - 第四个出现的字符 → 3
   - ...
5. 按分配规则将字符串转为十进制
```

## 四、代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;

    for (int caseNum = 0; caseNum < T; caseNum++) {
        string s;
        cin >> s;

        // 1. 记录字符出现的顺序（去重）
        vector<int> order(256, -1);  // -1表示未出现
        vector<char> chars;          // 按出现顺序存储字符

        for (char ch : s) {
            if (order[ch] == -1) {
                order[ch] = chars.size();
                chars.push_back(ch);
            }
        }

        int R = chars.size();  // 不同字符个数 = 进制
        if (R == 1) R = 2;     // 不能是1进制

        // 2. 分配数码
        vector<int> value(256, -1);
        
        // 第一个字符 → 1（最高位不能为0）
        value[chars[0]] = 1;

        // 分配剩余数码
        int nextDigit = 0;
        for (int i = 1; i < (int)chars.size(); i++) {
            // 第一个已经用了1，所以先分配0
            if (i == 1) {
                value[chars[i]] = 0;
            } else {
                // i=2 → 2, i=3 → 3, ...
                value[chars[i]] = i;
            }
        }

        // 3. 转换为十进制
        long long result = 0;
        for (char ch : s) {
            int digit = value[ch];
            result = result * R + digit;
        }

        cout << "case #" << caseNum << ":" << endl;
        cout << result << endl;
    }

    return 0;
}
```

### 3. 记录字符出现顺序

```cpp
vector<int> order(256, -1);  // ASCII共256个字符
vector<char> chars;          // 按出现顺序存储

for (char ch : s) {
    if (order[(unsigned char)ch] == -1) {
        order[(unsigned char)ch] = chars.size();
        chars.push_back(ch);
    }
}
```

**示例：** `s = "cats"`

| 遍历 | ch | order[ch] | chars |
|------|----|-----------|-------|
| 1 | 'c' | -1 → 0 | ['c'] |
| 2 | 'a' | -1 → 1 | ['c','a'] |
| 3 | 't' | -1 → 2 | ['c','a','t'] |
| 4 | 's' | -1 → 3 | ['c','a','t','s'] |

---

### 4. 确定进制

```cpp
int R = chars.size();  // 不同字符个数
if (R == 1) R = 2;     // 不能是1进制
```

**为什么不能是1进制？**
- 1进制只能表示0，无法表示正整数
- 最小有效进制是2进制

---

### 5. 分配数码值

```cpp
vector<int> value(256, -1);

// 第一个字符 → 1
value[(unsigned char)chars[0]] = 1;

// 剩余字符分配
for (int i = 1; i < (int)chars.size(); i++) {
    if (i == 1) {
        value[(unsigned char)chars[i]] = 0;
    } else {
        value[(unsigned char)chars[i]] = i;
    }
}
```

**分配逻辑：**

| 出现顺序 i | 分配数码 | 说明 |
|-----------|---------|------|
| 0 | 1 | 第一位，不能为0 |
| 1 | 0 | 最小的数码 |
| 2 | 2 | 下一个最小 |
| 3 | 3 | 以此类推 |

**示例：** `chars = ['c','a','t','s']`

```
c → 1
a → 0
t → 2
s → 3
```

---

### 6. 转换为十进制

```cpp
long long result = 0;
for (char ch : s) {
    int digit = value[(unsigned char)ch];
    result = result * R + digit;
}
```

**示例：** `s = "cats"`, `R = 4`

```
初始 result=0
'c' → digit=1 → result=0×4+1=1
'a' → digit=0 → result=1×4+0=4
't' → digit=2 → result=4×4+2=18
's' → digit=3 → result=18×4+3=75 ✓
```

---

### 7. 输出


**注意格式：** "case #0:" 后面换行，再输出结果

## 十、关键要点总结

| 要点 | 说明 |
|------|------|
| **进制 = 不同字符数** | 最小进制，使数值最小 |
| **第一位 → 1** | 最高位不能为0 |
| **第二位 → 0** | 尽可能小 |
| **后续 → 2,3,4...** | 从小到大分配 |
| **特殊：R=1 → R=2** | 不能是1进制 |

---

**核心思想：贪心分配数码，让高位尽可能小，进制尽可能小，从而得到最小值。**
