# EOJ-1011 Maya历日期的排序 - 完整解析

---

## 一、题目理解

### Maya 历法规则

Maya 历每年 19 个月：

| 月份 | 名称 | 天数 | 日范围 |
|------|------|------|--------|
| 1 | pop | 20 | 0-19 |
| 2 | no | 20 | 0-19 |
| 3 | zip | 20 | 0-19 |
| 4 | zotz | 20 | 0-19 |
| 5 | tzec | 20 | 0-19 |
| 6 | xul | 20 | 0-19 |
| 7 | yoxkin | 20 | 0-19 |
| 8 | mol | 20 | 0-19 |
| 9 | chen | 20 | 0-19 |
| 10 | yax | 20 | 0-19 |
| 11 | zac | 20 | 0-19 |
| 12 | ceh | 20 | 0-19 |
| 13 | mac | 20 | 0-19 |
| 14 | kankin | 20 | 0-19 |
| 15 | muan | 20 | 0-19 |
| 16 | pax | 20 | 0-19 |
| 17 | koyab | 20 | 0-19 |
| 18 | cumhu | 20 | 0-19 |
| 19 | uayet | 4-5 | 0-4（或0-5闰年） |

### 日期格式

```
日. 月名 年份
例如：8. yoxkin 2012
```

**注意：**
- 日 和 月名 之间有 `.` 和空格
- 月名 和 年份 之间有空格

---

## 二、解题思路

### 核心步骤

```
1. 定义月份名称到索引的映射
2. 对于每个日期，计算从"0年0月0日"到该日期的总天数
3. 按总天数排序
4. 输出原始日期
```

### 计算总天数

```
总天数 = 年份 × 365 + 月份索引 × 20 + 日

例如：8. yoxkin 2012
- yoxkin 是第6个月（索引从0开始）
- 总天数 = 2012 × 365 + 6 × 20 + 8 = 734380 + 120 + 8 = 734508
```

**注意：** 最后一个月份 uayet 的天数不影响总天数计算，因为它的日数不影响年份内的总天数（1年永远是365天）。

## 四、正确的读取方式

### 输入格式

```
8. yoxkin 2012
```

分为三部分：
1. `8.`（日 + 点）
2. `yoxkin`（月名）
3. `2012`（年份）

### 正确读取方法

```cpp
string dot;  // 会读成 "8."
string month;
int year;

cin >> dot >> month >> year;

// 从 "8." 中提取日
```

## 五、完整修正代码

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<string> months = {
    "pop", "no", "zip", "zotz", "tzec", "xul",
    "yoxkin", "mol", "chen", "yax", "zac", "ceh",
    "mac", "kankin", "muan", "pax", "koyab", "cumhu",
    "uayet"
};

map<string, int> monthIndex;

struct Date {
    string original;
    int day;
    string month;
    int year;
    int totalDays;
};

int calculateTotalDays(int day, const string& month, int year) {
    int monthIdx = monthIndex[month];
    return year * 365 + monthIdx * 20 + day;
}

bool compareDate(const Date& a, const Date& b) {
    return a.totalDays < b.totalDays;
}

int main() {
    for (int i = 0; i < (int)months.size(); i++) {
        monthIndex[months[i]] = i;
    }

    int T;
    cin >> T;

    for (int caseNum = 0; caseNum < T; caseNum++) {
        int n;
        cin >> n;

        vector<Date> dates(n);

        for (int i = 0; i < n; i++) {
            string dot;
            string month;
            int year;

            cin >> dot >> month >> year;

            // 从 "8." 中提取日
            int day = stoi(dot.substr(0, dot.find('.')));

            // 构造完整日期字符串
            string original = dot + " " + month + " " + to_string(year);

            dates[i].original = original;
            dates[i].day = day;
            dates[i].month = month;
            dates[i].year = year;
            dates[i].totalDays = calculateTotalDays(day, month, year);
        }

        sort(dates.begin(), dates.end(), compareDate);

        cout << "case #" << caseNum << ":" << endl;
        for (int i = 0; i < n; i++) {
            cout << dates[i].original << endl;
        }
    }

    return 0;
}
```

## 六、样例验证

### 输入

```
2
2
8. yoxkin 2012
0. pop 2000
3

9. yoxkin 2010
0. uayet 2012
```

### 处理第1个问题

| 日期 | 日 | 月名 | 年份 | 月份索引 | 总天数 |
|------|----|------|------|---------|--------|
| 8. yoxkin 2012 | 8 | yoxkin | 2012 | 6 | 2012×365 + 6×20 + 8 = 734508 |
| 0. pop 2000 | 0 | pop | 2000 | 0 | 2000×365 + 0 + 0 = 730000 |

**排序：** 730000 < 734508

**输出：**
```
0. pop 2000
8. yoxkin 2012
```

### 处理第2个问题

| 日期 | 总天数 |
|------|--------|
| 8. yoxkin 2012 | 734508 |
| 9. yoxkin 2010 | 2010×365 + 6×20 + 9 = 733650 + 120 + 9 = 733779 |
| 0. uayet 2012 | 2012×365 + 18×20 + 0 = 734380 + 360 = 734740 |

**排序：** 733779 < 734508 < 734740

**输出：**
```
9. yoxkin 2010
8. yoxkin 2012
0. uayet 2012
```

## 七、逐行详细解释

### 1. 月份定义和映射

```cpp
vector<string> months = {
    "pop", "no", "zip", "zotz", "tzec", "xul",
    "yoxkin", "mol", "chen", "yax", "zac", "ceh",
    "mac", "kankin", "muan", "pax", "koyab", "cumhu",
    "uayet"
};

map<string, int> monthIndex;

for (int i = 0; i < (int)months.size(); i++) {
    monthIndex[months[i]] = i;
}
```

**作用：** 将月份名称映射到索引（0-18）

| 月份名 | 索引 |
|--------|------|
| pop | 0 |
| no | 1 |
| zip | 2 |
| ... | ... |
| uayet | 18 |

---

### 2. 结构体定义

```cpp
struct Date {
    string original;  // 原始日期字符串
    int day;          // 日
    string month;     // 月名
    int year;         // 年份
    int totalDays;    // 从0年开始的总天数
};
```

**作用：** 存储日期的完整信息，方便排序。

---

### 3. 计算总天数

```cpp
int calculateTotalDays(int day, const string& month, int year) {
    int monthIdx = monthIndex[month];
    return year * 365 + monthIdx * 20 + day;
}
```

**公式：**
```
总天数 = 年份 × 365 + 月份索引 × 20 + 日
```

**示例：** `8. yoxkin 2012`
```
year = 2012
monthIdx = 6（yoxkin）
day = 8
总天数 = 2012 × 365 + 6 × 20 + 8 = 734508
```

---

### 4. 读取输入

```cpp
string dot;
string month;
int year;

cin >> dot >> month >> year;

int day = stoi(dot.substr(0, dot.find('.')));
```

**示例：** `8. yoxkin 2012`

```
dot = "8."
month = "yoxkin"
year = 2012
dot.find('.') = 1
dot.substr(0, 1) = "8"
stoi("8") = 8
```

---

### 5. 构造完整日期字符串

```cpp
string original = dot + " " + month + " " + to_string(year);
```

**示例：**
```
dot = "8."
month = "yoxkin"
year = 2012
original = "8. yoxkin 2012"
```

---

### 6. 排序

```cpp
sort(dates.begin(), dates.end(), compareDate);

bool compareDate(const Date& a, const Date& b) {
    return a.totalDays < b.totalDays;
}
```

按总天数升序排列，就是时间先后顺序。

## 八、关键注意事项

| 注意点 | 说明 |
|--------|------|
| **读取方式** | 用 `cin >> dot >> month >> year` 读取三部分 |
| **日提取** | 从 `"8."` 中提取数字 |
| **总天数计算** | `year * 365 + monthIdx * 20 + day` |
| **uayet 月份** | 天数为4-5天，但不影响总天数计算 |
| **排序** | 按总天数升序 |

|------|-----------|-----------|

## 十、总结

| 要点 | 说明 |
|------|------|
| **核心** | 将日期转为总天数，按总天数排序 |
| **月份映射** | `map<string, int>` 将月名转为索引 |
| **总天数公式** | `year × 365 + monthIdx × 20 + day` |
| **读取方式** | `cin >> dot >> month >> year` |
| **日提取** | `stoi(dot.substr(0, dot.find('.')))` |

**核心思想：将日期映射为整数（总天数），按整数排序即可得到时间先后顺序。**
