# 1042. 魔方阵
---

## 宏观理解

### 什么是魔方阵（幻方）？
一个 n×n 的方阵（n 为奇数），填入 1 到 n² 的数字，使得：
- 每一行之和相等
- 每一列之和相等
- 两条对角线之和相等

### 题目特点
- n 为奇数（3, 5, 7, ...）
- **1 放在最后一行的中间位置**（这是特殊的起始规则）
- 填充规则：按 45° 向左下方向行走

### 整体思路
按照提示的规则，依次把 1 到 n² 填入方阵中。

---

## 填充规则详解

### 规则1：起始位置
```
1 放在最后一行的中间位置
```
- 行：`n - 1`（最后一行，0 索引）
- 列：`n / 2`（中间列）

### 规则2：移动方向
```
按 45° 向左下方向行走
```
- 行：`row = (row + 1) % n`（向下移动一行）
- 列：`col = (col - 1 + n) % n`（向左移动一列）

### 规则3：回绕
```
如果行列范围超出矩阵范围，则回绕
```
- 最后一行绕到第 1 行
- 第 1 列绕到最后一列

### 规则4：特殊冲突处理（两个条件）
```
如果按上述规则确定的位置上已放了一个数，或上一个数是最后一行第 1 列时，
则把下一个数放在上一个数的上方
```

**条件A**：目标位置已被占用 → 放到上一个数的上方
**条件B**：上一个数在**最后一行第 1 列** → 放到上一个数的上方

**"放到上一个数的上方"**：
- 行：`row = (row - 1 + n) % n`（向上移动一行）
- 列：保持不变

## 填充规则流程图

```
┌─────────────────────────────────────────────────────────┐
│                    填充流程                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. 放置数字 1                                          │
│     位置：(n-1, n/2)                                   │
│                                                         │
│  2. 对于 num = 2 到 n*n：                              │
│                                                         │
│     计算下一个位置：                                     │
│     nextRow = (row + 1) % n      // 向下                │
│     nextCol = (col - 1 + n) % n  // 向左                │
│                                                         │
│     if (下一个位置已被占用 或 当前位置是 (n-1, 0)):    │
│         // 放在上方                                     │
│         nextRow = (row - 1 + n) % n                     │
│         nextCol = col                                   │
│                                                         │
│     把 num 放在 (nextRow, nextCol)                      │
│     更新 row = nextRow, col = nextCol                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    while (cin >> n) {
        // 创建 n×n 的方阵，初始化为 0
        vector<vector<int>> magic(n, vector<int>(n, 0));
        
        // 1 的位置：最后一行，中间列
        int row = n - 1;
        int col = n / 2;
        magic[row][col] = 1;
        
        // 从 2 开始填充到 n*n
        for (int num = 2; num <= n * n; num++) {
            // 按规则计算下一个位置：向左下移动
            int nextRow = (row + 1) % n;
            int nextCol = (col - 1 + n) % n;
            
            // 如果下一个位置已被占用，或上一个位置是最后一行第1列
            if (magic[nextRow][nextCol] != 0 || (row == n - 1 && col == 0)) {
                // 放在上一个数的上方
                nextRow = (row - 1 + n) % n;
                nextCol = col;
            }
            
            // 放置数字
            magic[nextRow][nextCol] = num;
            
            // 更新当前位置
            row = nextRow;
            col = nextCol;
        }
        
        // 输出魔方阵
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (j > 0) cout << " ";
                cout << magic[i][j];
            }
            cout << endl;
        }
    }
    return 0;
}
```

### 2. 创建方阵

- 创建 n×n 的二维向量，所有元素初始化为 0

```cpp

int col = n / 2;
magic[row][col] = 1;
```
- `row = n - 1`：最后一行
- `col = n / 2`：中间列（整数除法，例如 5/2=2）
- 在 `(n-1, n/2)` 位置放入 1

---

### 4. 填充 2 到 n²
```cpp

```
- 从 2 开始，依次填充到 n²

---

### 5. 计算下一个位置（向左下移动）
```cpp
int nextRow = (row + 1) % n;
int nextCol = (col - 1 + n) % n;
```
- `(row + 1) % n`：向下移动一行，超出则回绕到第 0 行
- `(col - 1 + n) % n`：向左移动一列，超出则回绕到第 n-1 列
- `+ n` 是为了防止负数取模的问题

---

### 6. 判断是否需要放在上方
```cpp
if (magic[nextRow][nextCol] != 0 || (row == n - 1 && col == 0)) {
    nextRow = (row - 1 + n) % n;
    nextCol = col;
}
```
- **条件1**：`magic[nextRow][nextCol] != 0` → 目标位置已被占用
- **条件2**：`(row == n - 1 && col == 0)` → 上一个数在最后一行第 1 列
- 满足任一条件，放在上一个数的上方
- `(row - 1 + n) % n`：向上移动一行，回绕

---

### 7. 放置数字并更新位置
```cpp
magic[nextRow][nextCol] = num;
row = nextRow;
col = nextCol;
```
- 将数字放入计算出的位置
- 更新当前位置为下一位置

## 关键点总结

| 知识点 | 说明 |
|--------|------|
| 起始位置 | 1 在最后一行中间 `(n-1, n/2)` |
| 移动方向 | 向左下：行+1，列-1 |
| 回绕 | 超出边界则回绕到对侧 |
| 冲突处理1 | 目标位置已被占用 → 放上方 |
| 冲突处理2 | 当前位置是最后一行第1列 → 放上方 |
| 放上方 | 行-1，列不变 |

---

## 为什么魔方阵成立？

这个算法是**奇数阶幻方的标准构造法（Siamese method）**，其数学原理保证了：
1. 每行、每列、对角线之和都等于 `n(n²+1)/2`
2. 所有数字 1 到 n² 恰好出现一次
