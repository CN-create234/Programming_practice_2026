## 一、什么是组合数？

### 定义
从 n 个不同元素中取出 r 个元素的组合数：
```
C(n, r) = n! / (r! · (n-r)!)
```

### 物理意义
在二项式展开中，`C(k, n)` 表示从 k 个因子中选 n 个 `(a·x)` 的方式数。

**例如**：`(x + y)³ = x³ + 3x²y + 3xy² + y³`
- `C(3, 1) = 3`（3x²y 的系数）
- `C(3, 2) = 3`（3xy² 的系数）

---

## 二、组合数的递推公式（帕斯卡三角）

### 核心公式
```
C(n, r) = C(n-1, r-1) + C(n-1, r)
```

### 帕斯卡三角

```
n=0:         1
n=1:        1 1
n=2:       1 2 1
n=3:      1 3 3 1
n=4:     1 4 6 4 1
n=5:   1 5 10 10 5 1
n=6:  1 6 15 20 15 6 1
```

**每个数 = 左上方的数 + 右上方的数**

### 示例验证
```
C(5, 2) = C(4, 1) + C(4, 2) = 4 + 6 = 10
C(6, 3) = C(5, 2) + C(5, 3) = 10 + 10 = 20
```

## 三、预处理代码详解

```cpp
const int MAXK = 1000;
vector<vector<int>> C(MAXK + 1, vector<int>(MAXK + 1, 0));

for (int i = 0; i <= MAXK; i++) {
    C[i][0] = C[i][i] = 1;          // 边界条件
    for (int j = 1; j < i; j++) {
        C[i][j] = (C[i - 1][j - 1] + C[i - 1][j]) % MOD;
    }
}
```

### 步骤1：创建二维数组

```cpp
vector<vector<int>> C(MAXK + 1, vector<int>(MAXK + 1, 0));
```

**含义**：创建一个 `(MAXK+1) × (MAXK+1)` 的二维数组，所有元素初始化为 0。

```
C[0][0] C[0][1] C[0][2] ... C[0][1000]
C[1][0] C[1][1] C[1][2] ... C[1][1000]
C[2][0] C[2][1] C[2][2] ... C[2][1000]
...
C[1000][0] ...              C[1000][1000]
```

**为什么大小是 MAXK+1？**
- 需要存储 C(0,0) 到 C(1000,1000)
- 索引从 0 开始，所以需要 1001 个位置

---

### 步骤2：外层循环遍历 n

```cpp
for (int i = 0; i <= MAXK; i++) {
```

**含义**：从 n=0 到 n=1000，逐行计算组合数

---

### 步骤3：设置边界条件

```cpp
C[i][0] = C[i][i] = 1;
```

**含义**：
- `C(i, 0) = 1`：从 i 个元素中取 0 个，只有 1 种方式（什么都不取）
- `C(i, i) = 1`：从 i 个元素中取 i 个，只有 1 种方式（全部取）

**帕斯卡三角边界**：
```
每一行的第一个和最后一个都是 1
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
```

---

### 步骤4：内层循环计算中间值

```cpp
for (int j = 1; j < i; j++) {
    C[i][j] = (C[i - 1][j - 1] + C[i - 1][j]) % MOD;
}
```

**含义**：
- `j` 从 1 到 i-1（不包含边界）
- 每个值等于上一行相邻两个值之和

**取模**：每步取模，防止溢出

## 五、为什么取模 10007？

### 防止溢出

组合数增长非常快：
```
C(1000, 500) ≈ 2.7 × 10^299
```
远超过 64 位整数范围（约 9.22 × 10^18）

### 取模性质

```
(A + B) % MOD = (A % MOD + B % MOD) % MOD
```

所以每一步取模，结果不变。

### 为什么 MOD = 10007？

- 10007 是质数（重要！）
- 较小的质数，适合取模运算
- 题目指定了模数

## 六、完整代码（带注释）

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MOD = 10007;
const int MAXK = 1000;

int main() {
    // 1. 预处理组合数
    vector<vector<int>> C(MAXK + 1, vector<int>(MAXK + 1, 0));
    
    for (int i = 0; i <= MAXK; i++) {
        C[i][0] = 1;      // C(i, 0) = 1
        C[i][i] = 1;      // C(i, i) = 1
        
        for (int j = 1; j < i; j++) {
            // 递推公式：C(i, j) = C(i-1, j-1) + C(i-1, j)
            C[i][j] = (C[i - 1][j - 1] + C[i - 1][j]) % MOD;
        }
    }
    
    // 2. 快速幂（计算 a^n 和 b^m）
    auto modPow = [&](long long base, int exp) {
        long long result = 1;
        base %= MOD;
        while (exp > 0) {
            if (exp & 1) result = (result * base) % MOD;
            base = (base * base) % MOD;
            exp >>= 1;
        }
        return result;
    };
    
    // 3. 处理测试用例
    int T;
    cin >> T;
    
    for (int caseNum = 0; caseNum < T; caseNum++) {
        long long a, b;
        int k, n, m;
        cin >> a >> b >> k >> n >> m;
        
        // 系数 = C(k, n) * a^n * b^m
        long long ans = C[k][n];           // 组合数
        ans = (ans * modPow(a, n)) % MOD;  // a^n
        ans = (ans * modPow(b, m)) % MOD;  // b^m
        
        cout << "case #" << caseNum << ":" << endl;
        cout << ans << endl;
    }
    
    return 0;
}
```

## 十、为什么不用阶乘 + 逆元？

### 阶乘 + 逆元方法

```cpp
C(k, n) = fact[k] * inv(fact[n]) * inv(fact[m]) % MOD
```

**需要求逆元**，需要理解费马小定理。

### 递推方法

```cpp
C[i][j] = C[i-1][j-1] + C[i-1][j]
```

**只需要加法**，更简单直观。

### 对比

| 方法 | 优点 | 缺点 |
|------|------|------|
| 递推 | 简单直观，无需逆元 | 需要 O(n²) 空间 |
| 阶乘+逆元 | 空间 O(n) | 需要理解逆元 |

## 总结

**预处理组合数 = 构建帕斯卡三角**：
1. 创建一个二维数组
2. 每行第一个和最后一个为 1
3. 中间每个数 = 左上 + 右上
4. 每步取模 10007
5. 查询时 O(1) 获取 C(k, n)
