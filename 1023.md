# EOJ-1023 统计字符串个数 - DP版本逐行详细解释

---

## 一、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    const int MAXN = 100;
    vector<long long> dp00(MAXN + 1), dp01(MAXN + 1), dp10(MAXN + 1), dp11(MAXN + 1);
    vector<long long> ans(MAXN + 1);

    // 长度为2的字符串
    dp00[2] = 1;
    dp01[2] = 1;
    dp10[2] = 1;
    dp11[2] = 1;
    ans[2] = 4;

    // 长度为1的字符串
    ans[1] = 2;

    for (int i = 3; i <= MAXN; i++) {
        dp00[i] = dp00[i - 1] + dp10[i - 1];
        dp01[i] = dp00[i - 1];
        dp10[i] = dp01[i - 1] + dp11[i - 1];
        dp11[i] = dp01[i - 1] + dp11[i - 1];
        ans[i] = dp00[i] + dp01[i] + dp10[i] + dp11[i];
    }

    int n;
    while (cin >> n && n != -1) {
        if (n == 1) cout << 2 << endl;
        else cout << ans[n] << endl;
    }

    return 0;
}
```

## 二、逐行详细解释

### 第一部分：头文件

### 第二部分：主函数和变量定义

```cpp
int main() {
    const int MAXN = 100;
    vector<long long> dp00(MAXN + 1), dp01(MAXN + 1), dp10(MAXN + 1), dp11(MAXN + 1);
    vector<long long> ans(MAXN + 1);
```

| 代码 | 含义 |
|------|------|
| `const int MAXN = 100` | 定义最大长度 100 |
| `vector<long long> dp00(MAXN + 1)` | 以 `"00"` 结尾的合法字符串个数 |
| `vector<long long> dp01(MAXN + 1)` | 以 `"01"` 结尾的合法字符串个数 |
| `vector<long long> dp10(MAXN + 1)` | 以 `"10"` 结尾的合法字符串个数 |
| `vector<long long> dp11(MAXN + 1)` | 以 `"11"` 结尾的合法字符串个数 |
| `vector<long long> ans(MAXN + 1)` | 长度为 i 的合法字符串总数 |

**为什么用 `long long`？**

`n=20` 时结果约 97229，`n=100` 时结果更大，`long long` 可以存储。

**四个状态的含义：**

| 状态 | 含义 | 示例 |
|------|------|------|
| `dp00[i]` | 长度为 i，以 `"00"` 结尾，不包含 `"101"` | `...00` |
| `dp01[i]` | 长度为 i，以 `"01"` 结尾，不包含 `"101"` | `...01` |
| `dp10[i]` | 长度为 i，以 `"10"` 结尾，不包含 `"101"` | `...10` |
| `dp11[i]` | 长度为 i，以 `"11"` 结尾，不包含 `"101"` | `...11` |

### 第三部分：初始化

```cpp
    // 长度为2的字符串
    dp00[2] = 1;
    dp01[2] = 1;
    dp10[2] = 1;
    dp11[2] = 1;
    ans[2] = 4;

    // 长度为1的字符串
    ans[1] = 2;
```

**长度为 2 的所有字符串：**

| 字符串 | 结尾状态 | 是否合法 |
|--------|----------|----------|
| `00` | `dp00` | ✅ |
| `01` | `dp01` | ✅ |
| `10` | `dp10` | ✅ |
| `11` | `dp11` | ✅ |

每种状态各 1 个，总共 4 个。

**长度为 1 的字符串：**

| 字符串 | 是否合法 |
|--------|----------|
| `0` | ✅ |
| `1` | ✅ |

总共 2 个。

**为什么 `dp00[2]=1` 而不是从 `dp00[1]` 推过来？**

因为长度为 1 的字符串没有两位，无法确定最后两位是 `"00"` 还是 `"01"` 等，所以从长度为 2 开始初始化。

### 第四部分：DP 递推

```cpp
    for (int i = 3; i <= MAXN; i++) {
        dp00[i] = dp00[i - 1] + dp10[i - 1];
        dp01[i] = dp00[i - 1];
        dp10[i] = dp01[i - 1] + dp11[i - 1];
        dp11[i] = dp01[i - 1] + dp11[i - 1];
        ans[i] = dp00[i] + dp01[i] + dp10[i] + dp11[i];
    }
```

---

### 第1行：`dp00[i] = dp00[i - 1] + dp10[i - 1]`

**含义：** 以 `"00"` 结尾的字符串 = 前一个状态加 0 得到的。

**哪些状态加 0 会变成 `"00"` 结尾？**

| 原字符串结尾 | 加0后 | 新结尾 |
|-------------|-------|--------|
| `"00"` | `"000"` | `"00"` ✅ |
| `"01"` | `"010"` | `"10"` ❌ |
| `"10"` | `"100"` | `"00"` ✅ |
| `"11"` | `"110"` | `"10"` ❌ |

所以只有 `dp00` 和 `dp10` 能转移到 `dp00`。

**转移公式：**
```
dp00[i] = dp00[i-1] + dp10[i-1]
```

---

### 第2行：`dp01[i] = dp00[i - 1]`

**含义：** 以 `"01"` 结尾的字符串 = 前一个状态加 1 得到的。

**哪些状态加 1 会变成 `"01"` 结尾？**

| 原字符串结尾 | 加1后 | 新结尾 | 是否合法 |
|-------------|-------|--------|----------|
| `"00"` | `"001"` | `"01"` ✅ | ✅ |
| `"01"` | `"011"` | `"11"` ❌ | - |
| `"10"` | `"101"` | `"01"` ✅ | ❌ **包含"101"！** |
| `"11"` | `"111"` | `"11"` ❌ | - |

`"10"` 加 1 会形成 `"101"`，非法！

所以只有 `dp00` 能转移到 `dp01`。

**转移公式：**
```
dp01[i] = dp00[i-1]
```

---

### 第3行：`dp10[i] = dp01[i - 1] + dp11[i - 1]`

**含义：** 以 `"10"` 结尾的字符串 = 前一个状态加 0 得到的。

**哪些状态加 0 会变成 `"10"` 结尾？**

| 原字符串结尾 | 加0后 | 新结尾 |
|-------------|-------|--------|
| `"00"` | `"000"` | `"00"` ❌ |
| `"01"` | `"010"` | `"10"` ✅ |
| `"10"` | `"100"` | `"00"` ❌ |
| `"11"` | `"110"` | `"10"` ✅ |

所以 `dp01` 和 `dp11` 能转移到 `dp10`。

**转移公式：**
```
dp10[i] = dp01[i-1] + dp11[i-1]
```

---

### 第4行：`dp11[i] = dp01[i - 1] + dp11[i - 1]`

**含义：** 以 `"11"` 结尾的字符串 = 前一个状态加 1 得到的。

**哪些状态加 1 会变成 `"11"` 结尾？**

| 原字符串结尾 | 加1后 | 新结尾 | 是否合法 |
|-------------|-------|--------|----------|
| `"00"` | `"001"` | `"01"` ❌ | - |
| `"01"` | `"011"` | `"11"` ✅ | ✅ |
| `"10"` | `"101"` | `"01"` ❌ | ❌ **包含"101"！** |
| `"11"` | `"111"` | `"11"` ✅ | ✅ |

所以 `dp01` 和 `dp11` 能转移到 `dp11`。

**转移公式：**
```
dp11[i] = dp01[i-1] + dp11[i-1]
```

---

### 第5行：`ans[i] = dp00[i] + dp01[i] + dp10[i] + dp11[i]`

**含义：** 长度为 i 的合法字符串总数 = 四种状态之和。

## 四、验证示例

### n = 3

```
初始化：dp00[2]=1, dp01[2]=1, dp10[2]=1, dp11[2]=1

i=3：
dp00[3] = dp00[2] + dp10[2] = 1+1 = 2   // "000","100"
dp01[3] = dp00[2] = 1                    // "001"
dp10[3] = dp01[2] + dp11[2] = 1+1 = 2   // "010","110"
dp11[3] = dp01[2] + dp11[2] = 1+1 = 2   // "011","111"

ans[3] = 2+1+2+2 = 7 ✓
```

### n = 4

```
i=4：
dp00[4] = dp00[3] + dp10[3] = 2+2 = 4
dp01[4] = dp00[3] = 2
dp10[4] = dp01[3] + dp11[3] = 1+2 = 3
dp11[4] = dp01[3] + dp11[3] = 1+2 = 3

ans[4] = 4+2+3+3 = 12 ✓
```

## 五、总结

| 状态 | 含义 | 转移来源 |
|------|------|----------|
| `dp00[i]` | 以 `"00"` 结尾 | `dp00[i-1]` + `dp10[i-1]`（加0） |
| `dp01[i]` | 以 `"01"` 结尾 | `dp00[i-1]`（加1） |
| `dp10[i]` | 以 `"10"` 结尾 | `dp01[i-1]` + `dp11[i-1]`（加0） |
| `dp11[i]` | 以 `"11"` 结尾 | `dp01[i-1]` + `dp11[i-1]`（加1） |

**核心思想：** 四个状态追踪字符串的最后两位，从 `"10"` 加 1 会形成 `"101"`，所以这个转移被禁止。其他所有转移都是合法的。
