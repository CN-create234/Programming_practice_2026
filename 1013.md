# EOJ-1013 二进制倒置 - 完整解析

---

## 一、题目理解

### 问题描述

给定一个整数 n，将其二进制表示（去掉前导零）前后倒置，然后输出倒置后二进制数对应的十进制值。

### 关键点

1. **去掉前导零**：二进制表示中开头可能有的 0 要去掉
2. **前后倒置**：反转二进制字符串
3. **输出十进制**：倒置后的二进制数对应的十进制值

### 示例

```
n = 10
二进制表示：1010
倒置后：0101
对应十进制：5

输出：5
```

### 特殊示例

```
n = 0
二进制表示：0
倒置后：0
对应十进制：0

输出：0
```

---

## 二、解题思路

### 核心步骤

```
1. 将 n 转为二进制字符串（不含前导零）
2. 反转这个字符串
3. 将反转后的二进制字符串转为十进制数
```

### 为什么大数需要特殊处理？

题目中的 n 可能非常大（如样例3），达到 100 多位，远超 `long long` 范围。

**所以需要用字符串处理！**

---

## 三、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

// 将十进制字符串转为二进制字符串（不含前导零）
string decToBin(string dec) {
    if (dec == "0") return "0";
    
    string binary = "";
    string num = dec;
    
    while (num != "0") {
        // 除以 2
        string quotient = "";
        int remainder = 0;
        int len = num.size();
        
        for (int i = 0; i < len; i++) {
            int digit = (num[i] - '0') + remainder * 10;
            int q = digit / 2;
            if (q != 0 || !quotient.empty()) {
                quotient += (q + '0');
            }
            remainder = digit % 2;
        }
        
        binary = (remainder ? "1" : "0") + binary;
        num = quotient.empty() ? "0" : quotient;
    }
    
    return binary;
}

// 将二进制字符串转为十进制字符串
string binToDec(string binary) {
    string result = "0";
    
    for (char ch : binary) {
        // result = result * 2 + digit
        int digit = ch - '0';
        string temp = "";
        int carry = 0;
        
        // result * 2
        for (int i = (int)result.size() - 1; i >= 0; i--) {
            int val = (result[i] - '0') * 2 + carry;
            temp = char(val % 10 + '0') + temp;
            carry = val / 10;
        }
        if (carry > 0) {
            temp = char(carry + '0') + temp;
        }
        
        // + digit
        if (digit == 0) {
            result = temp;
        } else {
            string add = temp;
            int carry2 = 1;
            for (int i = (int)add.size() - 1; i >= 0; i--) {
                int val = (add[i] - '0') + carry2;
                add[i] = char(val % 10 + '0');
                carry2 = val / 10;
            }
            if (carry2 > 0) {
                add = char(carry2 + '0') + add;
            }
            result = add;
        }
    }
    
    return result;
}

int main() {
    int T;
    cin >> T;
    
    for (int caseNum = 0; caseNum < T; caseNum++) {
        string n;
        cin >> n;
        
        // 转为二进制
        string bin = decToBin(n);
        
        // 反转二进制
        reverse(bin.begin(), bin.end());
        
        // 转为十进制
        string result = binToDec(bin);
        
        cout << "case #" << caseNum << ":" << endl;
        cout << result << endl;
    }
    
    return 0;
}
```

## 四、逐行详细解释

### 1. decToBin 函数

```cpp
string decToBin(string dec) {
    if (dec == "0") return "0";
    
    string binary = "";
    string num = dec;
    
    while (num != "0") {
        // 除以 2
        string quotient = "";
        int remainder = 0;
        int len = num.size();
        
        for (int i = 0; i < len; i++) {
            int digit = (num[i] - '0') + remainder * 10;
            int q = digit / 2;
            if (q != 0 || !quotient.empty()) {
                quotient += (q + '0');
            }
            remainder = digit % 2;
        }
        
        binary = (remainder ? "1" : "0") + binary;
        num = quotient.empty() ? "0" : quotient;
    }
    
    return binary;
}
```

**作用：** 将十进制字符串转为二进制字符串。

**算法：** 不断除以 2，记录余数。

**示例：** `decToBin("10")`

```
num = "10"

第1次循环：
  digit = 1 → q=0, rem=1
  digit = 0+10=10 → q=5, rem=0
  quotient = "5"
  binary = "0" + "" = "0"
  num = "5"

第2次循环：
  digit = 5 → q=2, rem=1
  quotient = "2"
  binary = "1" + "0" = "10"
  num = "2"

第3次循环：
  digit = 2 → q=1, rem=0
  quotient = "1"
  binary = "0" + "10" = "010"
  num = "1"

第4次循环：
  digit = 1 → q=0, rem=1
  quotient = ""
  binary = "1" + "010" = "1010"
  num = "0"

返回 "1010"
```

### 2. binToDec 函数

```cpp
string binToDec(string binary) {
    string result = "0";
    
    for (char ch : binary) {
        int digit = ch - '0';
        string temp = "";
        int carry = 0;
        
        // result = result * 2
        for (int i = (int)result.size() - 1; i >= 0; i--) {
            int val = (result[i] - '0') * 2 + carry;
            temp = char(val % 10 + '0') + temp;
            carry = val / 10;
        }
        if (carry > 0) {
            temp = char(carry + '0') + temp;
        }
        
        // result = result + digit
        if (digit == 0) {
            result = temp;
        } else {
            string add = temp;
            int carry2 = 1;
            for (int i = (int)add.size() - 1; i >= 0; i--) {
                int val = (add[i] - '0') + carry2;
                add[i] = char(val % 10 + '0');
                carry2 = val / 10;
            }
            if (carry2 > 0) {
                add = char(carry2 + '0') + add;
            }
            result = add;
        }
    }
    
    return result;
}
```

**作用：** 将二进制字符串转为十进制字符串。

**算法：** 霍纳法则：`result = result * 2 + digit`

**示例：** `binToDec("1010")`

```
result = "0"

ch='1' (digit=1):
  result*2 = "0"
  +1 = "1"
  result = "1"

ch='0' (digit=0):
  result*2 = "2"
  +0 = "2"
  result = "2"

ch='1' (digit=1):
  result*2 = "4"
  +1 = "5"
  result = "5"

ch='0' (digit=0):
  result*2 = "10"
  +0 = "10"
  result = "10"

返回 "10"
```

## 六、为什么不用位运算？

对于普通整数，可以用位运算：

```cpp
unsigned int reverseBits(unsigned int n) {
    unsigned int result = 0;
    while (n > 0) {
        result = (result << 1) | (n & 1);
        n >>= 1;
    }
    return result;
}
```

但是题目中的数可能非常大（100+ 位），远超 `unsigned long long` 范围，所以必须用字符串处理。

## 七、复杂度分析

| 操作 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| decToBin | O(n²) | O(n) |
| reverse | O(n) | O(1) |
| binToDec | O(n²) | O(n) |
| **总体** | **O(n²)** | **O(n)** |

其中 n 是数字的位数（最多 100 位）。

## 八、总结

| 要点 | 说明 |
|------|------|
| **核心** | 十进制 → 二进制 → 反转 → 十进制 |
| **大数处理** | 用字符串模拟十进制和二进制转换 |
| **decToBin** | 不断除以 2，记录余数 |
| **binToDec** | 霍纳法则：result = result × 2 + digit |
| **反转** | `reverse(bin.begin(), bin.end())` |

**核心思想：用字符串实现大数的进制转换和反转。**

# 大整数除基取余法与普通整数除法的区别

---

## 一、问题：为什么大整数不能直接用 `%` 和 `/`？

普通整数：
```cpp
int n = 58;
while (n > 0) {
    int remainder = n % 2;  // 直接用 %
    n /= 2;                 // 直接用 /
}
```

大整数（字符串表示）：
```cpp
string num = "12345678901234567890";
while (num != "0") {
    int remainder = ?;     // 无法直接用 %
    num = ?;               // 无法直接用 /
}
```

**原因：** 大整数位数可能超过 64 位（如 100 位），无法存入 `long long`，必须用**字符串模拟除法**。

---

## 二、普通整数除法的原理

### 二进制除法（以 58 ÷ 2 为例）

```
十进制 58
二进制 111010

58 ÷ 2 = 29 余 0
```

计算机内部直接进行二进制除法，效率很高。

### 普通整数代码

```cpp
int n = 58;
int remainder = n % 2;  // 0
int quotient = n / 2;   // 29
```

计算机直接计算，不需要手动处理每一位。

---

## 三、大整数除法的原理

### 竖式除法（以 58 ÷ 2 为例）

```
被除数：58
除数：2

步骤：
1. 取第一位 5：5 ÷ 2 = 2 余 1
2. 取第二位 8：8 + 1×10 = 18，18 ÷ 2 = 9 余 0

商：29
余数：0
```

### 竖式除法（以 123 ÷ 2 为例）

```
被除数：123
除数：2

步骤：
1. 取第一位 1：1 ÷ 2 = 0 余 1  → 跳过前导零
2. 取第二位 2：2 + 1×10 = 12，12 ÷ 2 = 6 余 0
3. 取第三位 3：3 + 0×10 = 3，3 ÷ 2 = 1 余 1

商：61
余数：1
```

---

## 四、代码对比

### 普通整数除法

```cpp
int n = 58;
int remainder = n % 2;  // 直接取余
int quotient = n / 2;   // 直接除
```

**特点：** 一行代码完成整个数的除法。

---

### 大整数除法（字符串模拟）

```cpp
string num = "58";
string quotient = "";
int remainder = 0;

for (int i = 0; i < num.size(); i++) {
    int digit = (num[i] - '0') + remainder * 10;
    int q = digit / 2;
    if (q != 0 || !quotient.empty()) {
        quotient += (q + '0');
    }
    remainder = digit % 2;
}
```

**特点：** 需要逐位处理，模拟竖式除法。

---

## 五、为什么大整数不能直接用 `%` 和 `/`？

### 原因1：数据范围限制

| 类型 | 最大范围 | 最大位数 |
|------|---------|---------|
| int | 2×10⁹ | 10 位 |
| long long | 9×10¹⁸ | 19 位 |
| 题目要求 | - | 1000 位 |

大整数超出基本类型范围，无法直接存储和运算。

---

### 原因2：运算符不支持字符串

```cpp
string num = "12345678901234567890";
int remainder = num % 2;   // ❌ 编译错误！
int quotient = num / 2;    // ❌ 编译错误！
```

`%` 和 `/` 只支持数值类型，不支持字符串。

---

### 原因3：逐位模拟的必要性

大整数除法必须模拟**竖式除法**：

```
  123456 ÷ 2
= 61728 余 0
```

竖式除法需要逐位处理，这是字符串模拟的核心。

---

## 六、逐位除法的详细过程

### 示例：`"123456" ÷ 2`

| 步骤 | 当前位 | 上一位余数 | digit | q | 余数 | quotient |
|------|--------|-----------|-------|---|------|----------|
| 1 | 1 | 0 | 1 | 0 | 1 | "" |
| 2 | 2 | 1 | 12 | 6 | 0 | "6" |
| 3 | 3 | 0 | 3 | 1 | 1 | "61" |
| 4 | 4 | 1 | 14 | 7 | 0 | "617" |
| 5 | 5 | 0 | 5 | 2 | 1 | "6172" |
| 6 | 6 | 1 | 16 | 8 | 0 | "61728" |

**结果：** 商 `"61728"`，余数 `0`

## 八、总结

| 方面 | 普通整数 | 大整数（字符串） |
|------|---------|-----------------|
| **存储** | 二进制存储 | 字符串存储 |
| **除法** | `n / 2` | 逐位模拟 |
| **取余** | `n % 2` | 逐位模拟 |
| **效率** | O(1) | O(n) |
| **范围** | 有限（约20位） | 无限（任意位数） |

## 九、一句话总结

**大整数不能直接用 `%` 和 `/`，因为字符串不支持这些运算符，且大整数可能超出基本类型的范围，必须用逐位模拟竖式除法来实现。**
