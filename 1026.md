# EOJ-1026 最小的01串权值 - 完整解析

---

## 一、题目理解

### 问题描述

给定一个包含 `0`、`1` 和 `?` 的字符串，将每个 `?` 替换成 `0` 或 `1`，使得最终字符串的**权值**最小。

### 权值定义

- 子串 `01` 的权值为 `a`
- 子串 `10` 的权值为 `b`
- 一个01串的权值 = 所有子串 `01` 的权值之和 + 所有子串 `10` 的权值之和

**换句话说：** 权值 = `(01出现的次数) × a + (10出现的次数) × b`

### 示例

```
字符串：010111
子串01：位置(1,2) 和 (3,4) → 2个
子串10：位置(2,3) → 1个
权值 = 2×a + 1×b
```

---

## 二、核心观察

### 权值只与相邻字符对有关

子串 `01` 和 `10` 都只涉及**相邻的两个字符**。

所以权值 = 所有相邻字符对 `(s[i], s[i+1])` 的贡献之和。

| 相邻字符对 | 贡献 |
|-----------|------|
| `00` | 0 |
| `01` | a |
| `10` | b |
| `11` | 0 |

### 转化为图论问题

把每个位置看作节点，相邻位置之间的边有贡献：
- 如果边是 `01`，贡献 `a`
- 如果边是 `10`，贡献 `b`
- 如果边是 `00` 或 `11`，贡献 `0`

**目标：** 给每个 `?` 赋值 0 或 1，最小化所有相邻边的贡献之和。

---

## 三、动态规划解法

### 状态定义

```
dp[i][c] = 处理到位置 i，且 s[i] = c (0或1) 时，前 i 个字符的最小权值
```

### 状态转移

对于位置 `i`，如果 `s[i]` 是确定的（0或1），只能取该值；如果是 `?`，可以取 0 或 1。

```
dp[i][c] = min(dp[i-1][prev] + cost(prev, c))
```

其中 `cost(prev, c)` 是相邻字符对 `(prev, c)` 的权值：
- `prev=0, c=1` → `a`
- `prev=1, c=0` → `b`
- 其他情况 → `0`

### 初始化

```
dp[0][0] = 0   // 第0位取0
dp[0][1] = 0   // 第0位取1
```

但需要根据 `s[0]` 限制：
- 如果 `s[0] == '0'`：只能取0
- 如果 `s[0] == '1'`：只能取1
- 如果 `s[0] == '?'`：可以取0或1

## 四、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long long a, b;
    string s;

    for (int t = 0; t < 4; t++) {
        cin >> a >> b >> s;
        int n = s.size();

        // dp[0][0] = 以0结尾的最小权值，dp[0][1] = 以1结尾的最小权值
        vector<long long> dp0(n + 1, 1e18), dp1(n + 1, 1e18);

        // 初始化第0位
        if (s[0] == '0' || s[0] == '?') dp0[0] = 0;
        if (s[0] == '1' || s[0] == '?') dp1[0] = 0;

        for (int i = 1; i < n; i++) {
            // 当前位取0
            if (s[i] == '0' || s[i] == '?') {
                // 前一位取0 → 00 → 权值0
                dp0[i] = min(dp0[i], dp0[i - 1] + 0);
                // 前一位取1 → 10 → 权值 b
                dp0[i] = min(dp0[i], dp1[i - 1] + b);
            }

            // 当前位取1
            if (s[i] == '1' || s[i] == '?') {
                // 前一位取0 → 01 → 权值 a
                dp1[i] = min(dp1[i], dp0[i - 1] + a);
                // 前一位取1 → 11 → 权值0
                dp1[i] = min(dp1[i], dp1[i - 1] + 0);
            }
        }

        cout << min(dp0[n - 1], dp1[n - 1]) << endl;
    }

    return 0;
}
```

- `dp0[i]`：处理到位置 `i`，且 `s[i] = 0` 时的最小权值
- `dp1[i]`：处理到位置 `i`，且 `s[i] = 1` 时的最小权值


|------|--------|--------|

### 5. DP 状态转移

```cpp
        for (int i = 1; i < n; i++) {
            // 当前位取0
            if (s[i] == '0' || s[i] == '?') {
                dp0[i] = min(dp0[i], dp0[i - 1] + 0);
                dp0[i] = min(dp0[i], dp1[i - 1] + b);
            }

            // 当前位取1
            if (s[i] == '1' || s[i] == '?') {
                dp1[i] = min(dp1[i], dp0[i - 1] + a);
                dp1[i] = min(dp1[i], dp1[i - 1] + 0);
            }
        }
```

**转移规则：**

| 前一位 | 当前位 | 相邻字符对 | 贡献 |
|--------|--------|-----------|------|
| 0 | 0 | 00 | 0 |
| 1 | 0 | 10 | b |
| 0 | 1 | 01 | a |
| 1 | 1 | 11 | 0 |

**当前位取0：**
- 前一位取0 → `00` → 权值 `0`
- 前一位取1 → `10` → 权值 `b`

**当前位取1：**
- 前一位取0 → `01` → 权值 `a`
- 前一位取1 → `11` → 权值 `0`

## 六、执行示例

### 示例1：`a=2, b=3, s="01?00?"`

```
n=6

初始化：
dp0[0] = 0（s[0]='0'）
dp1[0] = ∞

i=1, s[1]='1'：
  当前位取1：dp1[1] = dp0[0] + a = 0+2=2
  dp1[1]=2

i=2, s[2]='?'：
  当前位取0：dp0[2] = min(dp0[1]+0, dp1[1]+b) = min(∞, 2+3)=5
  当前位取1：dp1[2] = min(dp0[1]+a, dp1[1]+0) = min(∞, 2+0)=2

i=3, s[3]='0'：
  当前位取0：dp0[3] = min(dp0[2]+0, dp1[2]+b) = min(5+0, 2+3)=5

i=4, s[4]='0'：
  当前位取0：dp0[4] = min(dp0[3]+0, dp1[3]+b) = min(5+0, ∞)=5

i=5, s[5]='?'：
  当前位取0：dp0[5] = min(dp0[4]+0, dp1[4]+b) = min(5, ∞)=5
  当前位取1：dp1[5] = min(dp0[4]+a, dp1[4]+0) = min(5+2, ∞)=7

结果：min(dp0[5], dp1[5]) = min(5,7) = 5 ✓
```

## 八、总结

| 要点 | 说明 |
|------|------|
| **核心** | 动态规划，状态为当前位置的字符值（0或1） |
| **状态定义** | `dp0[i]`：位置 i 取0的最小权值；`dp1[i]`：位置 i 取1的最小权值 |
| **转移** | 根据相邻字符对的权值转移 |
| **权值** | `00` 和 `11` 贡献0，`01` 贡献 a，`10` 贡献 b |

**核心思想：** 用 DP 遍历每个位置，记录当前位置取0或取1时的最小权值，根据相邻字符对的权值进行转移。
