### 核心算法

**霍纳法则：**
```
R进制字符串 → 十进制整数
value = 0
for each char in s:
    value = value * R + digitValue(char)
```

**除基取余法：**
```
十进制 → R进制
while n > 0:
    rem = n % R
    result = digitToChar(rem) + result
    n = n / R
```

**负二进制调整：**
```
rem = n % (-2)  // 可能是 -1, 0, 1
if rem < 0:
    rem += 2      // 调整为 1
    n += 1        // 商加1抵消
```

```
反转字符串（个位在索引0）

反转回来
```

**大整数减法（借位）：**
```
反转字符串

处理借位

```

**大整数除以2（竖式除法）：**
```
从高位到低位逐位处理
当前位 = (上一位余数×10 + 当前位数字) / 2
余数 = (上一位余数×10 + 当前位数字) % 2
```

```
        maxLen = right - left + 1
        start = left
```

## 四、动态规划类

| 题号 | 题目名称 | 核心考察点 |
|------|----------|-----------|
| 1023 | 统计字符串个数 | 状态DP（四个状态：00,01,10,11结尾），不包含"101"子串，状态转移 |
| 1026 | 最小的01串权值 | 状态DP（dp0/dp1），最小化相邻字符对权值，字符替换（?） |
| 1030 | 0-1背包问题 | 0-1背包DP，一维数组优化，倒序遍历容量，状态转移 `dp[j] = max(dp[j], dp[j-w]+v)` |
| 1032 | 台阶走法数 | 简单DP（斐波那契变种），每步走1-4级，递推公式 `f[n]=f[n-1]+f[n-2]+f[n-3]+f[n-4]` |

### 核心算法

**0-1背包（一维DP）：**
```
dp[0..C] = 0
for each item (w, v):
    for j = C down to w:
        dp[j] = max(dp[j], dp[j-w] + v)
```

**状态DP（四状态，不包含"101"）：**
```
dp00[i] = dp00[i-1] + dp10[i-1]  // 加0
dp01[i] = dp00[i-1]              // 加1
dp10[i] = dp01[i-1] + dp11[i-1]  // 加0
dp11[i] = dp01[i-1] + dp11[i-1]  // 加1
// dp10 -> 加1 会形成 "101"，禁止！
```

**状态DP（01串权值最小化）：**
```
if s[i] == '0' or '?':
    dp0[i] = min(dp0[i-1] + 0, dp1[i-1] + b)  // 00贡献0，10贡献b
if s[i] == '1' or '?':
    dp1[i] = min(dp0[i-1] + a, dp1[i-1] + 0)  // 01贡献a，11贡献0
```

## 十一、核心模板函数

**GCD：**
```cpp
long long gcd(long long a, long long b) {
    return b ? gcd(b, a % b) : abs(a);
}
```

**字符转数码：**
```cpp
int digitValue(char c) {
    if (c >= '0' && c <= '9') return c - '0';
    if (c >= 'A' && c <= 'Z') return c - 'A' + 10;
    if (c >= 'a' && c <= 'z') return c - 'a' + 10;
    return -1;
}
```

**大整数除以2（竖式除法）：**
```cpp
pair<string, int> div2(const string& s) {
    string q;
    int r = 0;
    for (char c : s) {
        int cur = (c - '0') + r * 10;
        int d = cur / 2;
        if (d != 0 || !q.empty()) q += char(d + '0');
        r = cur % 2;
    }
    if (q.empty()) q = "0";
    return {q, r};
}
```
