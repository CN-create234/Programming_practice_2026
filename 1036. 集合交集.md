# 1036. 集合交集
## 完整代码（先贴出来方便对照）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;

    vector<vector<int>> sets(n);

    // 读入所有集合
    for (int i = 0; i < n; i++) {
        int m;
        cin >> m;
        sets[i].resize(m);
        for (int j = 0; j < m; j++) {
            cin >> sets[i][j];
        }
    }

    // 用第一个集合作为初始交集
    vector<int> result = sets[0];

    // 逐个与后面的集合取交集
    for (int i = 1; i < n; i++) {
        vector<int> temp;
        // 遍历 result 中的每个元素，检查是否在当前集合中
        for (int x : result) {
            // 判断 x 是否在 sets[i] 中
            bool found = false;
            for (int y : sets[i]) {
                if (x == y) {
                    found = true;
                    break;
                }
            }
            if (found) {
                temp.push_back(x);
            }
        }
        result = temp;
    }

    // 排序输出
    sort(result.begin(), result.end());

    for (int i = 0; i < result.size(); i++) {
        if (i > 0) cout << " ";
        cout << result[i];
    }
    cout << endl;

    return 0;
}
```

- 创建一个二维 vector，外层大小为 n

### 13. 遍历 result 中的每个元素
```cpp

```
- 这是 C++11 的**范围 for 循环**
- `int x : result` 表示依次取出 `result` 中的每个元素，赋值给 `x`
- 等价于：
```cpp
for (int idx = 0; idx < result.size(); idx++) {
    int x = result[idx];

```

### 14. 判断 x 是否在当前集合中
```cpp
            bool found = false;
            for (int y : sets[i]) {
                if (x == y) {
                    found = true;
                    break;
                }
            }
```
- `found` 初始为 `false`，表示还没找到
- 遍历当前集合 `sets[i]` 中的每个元素 `y`
- 如果 `x == y`，说明 `x` 在当前集合中
  - 将 `found` 设为 `true`
  - `break` 跳出内层循环，不再继续查找（因为已经找到了）

---

### 15. 如果找到了，加入临时结果
```cpp
            if (found) {
                temp.push_back(x);
            }
```
- 如果 `x` 在当前集合中（`found == true`），把 `x` 加入 `temp`
- `push_back` 是在 vector 末尾添加元素

---

### 16. 更新 result 为当前交集
```cpp
        result = temp;
```
- 将 `result` 更新为 `temp`
- 此时 `result` 就是新的交集

---

### 17. 外层循环结束
```cpp
    }
```
- 所有集合处理完毕
- 最终 `result` 包含所有集合的交集

---

### 18. 排序
```cpp
    sort(result.begin(), result.end());
```
- 将 `result` 中的元素从小到大排序
- 题目要求输出按从小到大的顺序

---

### 19. 输出结果
```cpp

```
- 遍历 `result` 中的每个元素
- `if (i > 0) cout << " ";`：除了第一个元素，其他元素前面都加一个空格
- 这样输出格式就是 `1 3 5`，而不是 `1 3 5 `（末尾没有多余空格）
- 最后输出换行

## 样例运行过程

### 输入：
```
2
5 -1 3 1 -4 5
6 5 3 1 7 9 11
```

### 详细步骤：

**第 1 步：读取 n**
```
n = 2
```

**第 2 步：读取第 0 个集合**
```
m = 5
读取 5 个元素：-1, 3, 1, -4, 5
sets[0] = {-1, 3, 1, -4, 5}
```

**第 3 步：读取第 1 个集合**
```
m = 6
读取 6 个元素：5, 3, 1, 7, 9, 11
sets[1] = {5, 3, 1, 7, 9, 11}
```

**第 4 步：初始化 result**
```
result = {-1, 3, 1, -4, 5}
```

**第 5 步：取交集（i = 1）**
```
遍历 result 中的每个元素：

x = -1：
  检查 -1 是否在 {5, 3, 1, 7, 9, 11} 中
  → 不在，跳过

x = 3：
  检查 3 是否在 {5, 3, 1, 7, 9, 11} 中
  → 在！加入 temp
  temp = {3}

x = 1：
  检查 1 是否在 {5, 3, 1, 7, 9, 11} 中
  → 在！加入 temp
  temp = {3, 1}

x = -4：
  检查 -4 是否在 {5, 3, 1, 7, 9, 11} 中
  → 不在，跳过

x = 5：
  检查 5 是否在 {5, 3, 1, 7, 9, 11} 中
  → 在！加入 temp
  temp = {3, 1, 5}

result = {3, 1, 5}
```

**第 6 步：排序**
```
sort(result.begin(), result.end())
result = {1, 3, 5}
```

**第 7 步：输出**
```
1 3 5
```

## 关键知识点总结

| 知识点 | 说明 | 代码示例 |
