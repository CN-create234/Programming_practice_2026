# EOJ-1032 台阶走法数 - 完整解析

---

## 一、题目理解

### 问题描述

有 n 级台阶，每一步可以走 1、2、3 或 4 级，求总共有多少种不同的走法。

### 示例验证

**n = 1：**
```
只能走1步：1种
输出：1 ✓
```

**n = 4：**
```
枚举所有走法：
1+1+1+1
1+1+2
1+2+1
2+1+1
2+2
1+3
3+1
4
共 8 种
输出：8 ✓
```

**n = 50：**
```
输出：100808458960497（约 1e14） ✓
```

---

## 二、解题思路

### 递推关系（动态规划）

设 `f[n]` 为走到第 n 级台阶的走法数。

**最后一步可以走 1、2、3、4 级：**

```
f[n] = f[n-1] + f[n-2] + f[n-3] + f[n-4]
```

### 初始条件

```
f[0] = 1  // 站在地面，不走也是一种方式
f[1] = 1  // 1
f[2] = 2  // 1+1, 2
f[3] = 4  // 1+1+1, 1+2, 2+1, 3
f[4] = 8  // 上述 8 种
```

### 验证递推

```
f[4] = f[3] + f[2] + f[1] + f[0] = 4 + 2 + 1 + 1 = 8 ✓
f[5] = f[4] + f[3] + f[2] + f[1] = 8 + 4 + 2 + 1 = 15
f[6] = 15 + 8 + 4 + 2 = 29
...
```

## 三、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;

    // 预计算到 50
    const int MAXN = 50;
    vector<long long> dp(MAXN + 1);
    dp[0] = 1;
    dp[1] = 1;
    dp[2] = 2;
    dp[3] = 4;

    for (int i = 4; i <= MAXN; i++) {
        dp[i] = dp[i - 1] + dp[i - 2] + dp[i - 3] + dp[i - 4];
    }

    for (int caseNum = 0; caseNum < T; caseNum++) {
        int n;
        cin >> n;

        cout << "case #" << caseNum << ":" << endl;
        cout << dp[n] << endl;
    }

    return 0;
}
```

## 八、为什么用 long long？

n=50 时，结果约 1e14，在 `long long` 范围内（最大 9.22e18）。如果 n 更大，需要用大整数，但题目最大 50，`long long` 足够。

---

## 九、总结

| 要点 | 说明 |
|------|------|
| **核心算法** | 动态规划 |
| **递推公式** | `f[i] = f[i-1] + f[i-2] + f[i-3] + f[i-4]` |
| **初始条件** | `f[0]=1, f[1]=1, f[2]=2, f[3]=4` |
| **数据类型** | `long long` |
| **预计算** | 先算好所有值，查询 O(1) |

**核心思想：** 到达第 i 级台阶的最后一步可以是 1、2、3 或 4 级，所以 `f[i]` 等于前四个状态之和。
