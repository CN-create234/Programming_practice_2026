# 1047. 最大 SP 值

好的，这道题是把一些素数分成两组，第一组求和得到 S，第二组求积得到 P，要求 S = P 时最大化 SP 值（即 S）。素数总个数不超过 10，可以暴力枚举所有分组方式。

---

## 宏观理解

### 问题本质

给定若干个素数（可能有重复），把它们分成两组：
- **第一组**：求**和** → S
- **第二组**：求**积** → P

要求 `S = P`，最大化 `SP = S`。

### 关键点
- 每个素数必须属于**恰好一组**
- 素数总个数 ≤ 10，可以暴力枚举（2^10 = 1024 种）
- 第二组的乘积可能很大（10 个 499 相乘 ≈ 499^10 ≈ 10^27），需要用 64 位整数

---

## 核心思路

### 枚举所有分组

用二进制表示分组状态：
- 0 表示放入第一组（求和）
- 1 表示放入第二组（求积）

### 算法步骤

1. 读取所有素数，存入数组 `primes`
2. 总个数 `cnt`
3. 枚举 `mask` 从 0 到 `(1 << cnt) - 1`：
   - 计算第一组和 `S`
   - 计算第二组积 `P`
   - 如果 `S == P`，更新最大 SP 值
4. 输出 `cnt` 和最大 SP 值

### 复杂度
- 最多 10 个数
- 枚举 2^10 = 1024 种
- 每次 O(10)
- 总 O(1024 × 10) ≈ 10^4，非常快

---

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    vector<int> primes;  // 存储所有素数
    int p, n;
    
    // 读取输入，直到 EOF
    while (cin >> p >> n) {
        for (int i = 0; i < n; i++) {
            primes.push_back(p);
        }
    }
    
    int cnt = primes.size();
    long long maxSP = 0;
    
    // 枚举所有分组方式
    // 0 表示放入第一组（求和），1 表示放入第二组（求积）
    for (int mask = 0; mask < (1 << cnt); mask++) {
        long long S = 0;   // 第一组和
        long long P = 1;   // 第二组积（初始为 1）
        
        for (int i = 0; i < cnt; i++) {
            if (mask & (1 << i)) {
                // 第 i 个素数放入第二组（求积）
                P *= primes[i];
            } else {
                // 第 i 个素数放入第一组（求和）
                S += primes[i];
            }
        }
        
        if (S == P) {
            maxSP = max(maxSP, S);
        }
    }
    
    cout << cnt << " " << maxSP << endl;
    
    return 0;
}
```

### 2. 枚举所有分组

```cpp
int cnt = primes.size();
long long maxSP = 0;

for (int mask = 0; mask < (1 << cnt); mask++) {
```

**二进制表示**：
- `mask` 从 0 到 2^cnt - 1
- 第 i 位为 0 → 第 i 个素数放入第一组（求和）
- 第 i 位为 1 → 第 i 个素数放入第二组（求积）

**示例**：cnt=3
```
mask=0  (000)：全部放入第一组
mask=1  (001)：第0个放入第二组，其他第一组
mask=2  (010)：第1个放入第二组，其他第一组
mask=3  (011)：第0、1个放入第二组
...
mask=7  (111)：全部放入第二组
```

---

### 3. 计算 S 和 P

```cpp
long long S = 0;
long long P = 1;

for (int i = 0; i < cnt; i++) {
    if (mask & (1 << i)) {
        P *= primes[i];
    } else {
        S += primes[i];
    }
}
```

**含义**：
- 检查 `mask` 的第 i 位
- 为 1：放入第二组，乘到 P
- 为 0：放入第一组，加到 S

**示例**：`primes = [2, 3, 5]`，`mask = 3 (011)`
```
i=0: mask & 1 = 1 → P = 1 * 2 = 2
i=1: mask & 2 = 2 → P = 2 * 3 = 6
i=2: mask & 4 = 0 → S = 0 + 5 = 5

结果：S=5, P=6
```

---

### 4. 更新最大 SP 值

```cpp
if (S == P) {
    maxSP = max(maxSP, S);
}
```

**含义**：如果 S == P，更新最大 SP 值

---

```cpp
cout << cnt << " " << maxSP << endl;
```

**样例1**：
```
输出：7 25
```

## 特殊情况处理

### 1. 第二组为空
```
所有素数都在第一组：P = 1（空集乘积为 1）
```
所以如果 S = 1 且所有数都在第一组，可能匹配，但 N 是素数，S ≥ 2，所以不会匹配。

### 2. 第一组为空
```
所有素数都在第二组：S = 0
```
P > 1，所以 0 ≠ P，不会匹配。

## 完整代码（带注释）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    vector<int> primes;
    int p, n;
    
    // 读取所有素数
    while (cin >> p >> n) {
        for (int i = 0; i < n; i++) {
            primes.push_back(p);
        }
    }
    
    int cnt = primes.size();
    long long maxSP = 0;
    
    // 枚举所有分组方式
    // 二进制位：0=第一组(求和)，1=第二组(求积)
    for (int mask = 0; mask < (1 << cnt); mask++) {
        long long S = 0;   // 第一组和
        long long P = 1;   // 第二组积
        
        for (int i = 0; i < cnt; i++) {
            if (mask & (1 << i)) {
                P *= primes[i];      // 放入第二组
            } else {
                S += primes[i];      // 放入第一组
            }
        }
        
        if (S == P) {
            maxSP = max(maxSP, S);
        }
    }
    
    cout << cnt << " " << maxSP << endl;
    
    return 0;
}
```
