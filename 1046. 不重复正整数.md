# 1046. 不重复正整数

好的，这道题是**不重复正整数拆分**问题，即用 1 到 M 中的不同数字（每个数字最多用一次）凑成 N，求方案数。

---

## 宏观理解

### 问题本质

用 1 到 M 中的若干个**不同**数字，组成和为 N，求方案数。

### 与完全背包的区别

| 类型 | 每个数字可用次数 | 例子 |
|------|-----------------|------|
| 完全背包 | 无限次 | 1+1+1+3 允许 |
| 0/1 背包 | 最多一次 | 1+3 可以，1+1+3 不行 |

**本题是 0/1 背包**：每个数字最多用一次。

### 示例验证

**样例1**：`N=6, M=4`

可用数字：1, 2, 3, 4

拆分方案（数字不重复）：
```
6 = 2 + 4
6 = 1 + 2 + 3
```
共 2 种 ✅

**注意**：`6 = 1 + 1 + 4` 不行（1 重复）
`6 = 1 + 2 + 3` 和 `6 = 3 + 2 + 1` 算同一种（不区分顺序）

---

## 核心思路

### 动态规划

**状态定义**：
```
dp[j] = 用若干不重复数字凑成和 j 的方案数
```

**状态转移**（0/1 背包，逆序）：
```
对于每个数字 i (1 到 M)：
  从 j = N 到 i：
    dp[j] += dp[j - i]
```

**为什么逆序？**
- 保证每个数字只用一次
- 正序会重复使用同一个数字

---

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;
    
    for (int caseNum = 0; caseNum < T; caseNum++) {
        int N, M;
        cin >> N >> M;
        
        // dp[j] 表示凑成和 j 的方案数
        vector<long long> dp(N + 1, 0);
        dp[0] = 1;  // 凑成 0 有 1 种方案（什么都不选）
        
        // 遍历每个数字 1 到 M
        for (int i = 1; i <= M; i++) {
            // 逆序遍历，保证每个数字只用一次
            for (int j = N; j >= i; j--) {
                dp[j] += dp[j - i];
            }
        }
        
        cout << "case #" << caseNum << ":" << endl;
        cout << dp[N] << endl;
    }
    
    return 0;
}
```

## 逐行详细解释

### 1. 读取测试用例数
```cpp
int T;
cin >> T;
```
- 读取 T，表示问题个数

---

### 2. 外层循环：处理每个问题
```cpp
for (int caseNum = 0; caseNum < T; caseNum++) {
    int N, M;
    cin >> N >> M;
```
- `caseNum` 从 0 开始，用于输出 `case #0:`
- `N`：目标和
- `M`：最大可用数字

---

### 3. 初始化 DP 数组
```cpp
vector<long long> dp(N + 1, 0);
dp[0] = 1;
```

**含义**：
- `dp[j]`：用不重复数字凑成和 j 的方案数
- `dp[0] = 1`：凑成 0 有 1 种方案（不选任何数字）

**为什么用 long long？**
- 方案数可能很大（样例中 N=50, M=20 时是 1969）
- N=50 时最大方案数约 10^6，int 够用，但 long long 更安全

---

### 4. 遍历每个数字
```cpp
for (int i = 1; i <= M; i++) {
```

**含义**：依次考虑数字 1, 2, 3, ..., M

**每个数字的状态**：选 或 不选

---

### 5. 逆序更新 DP
```cpp
for (int j = N; j >= i; j--) {
    dp[j] += dp[j - i];
}
```

**含义**：
- `dp[j - i]`：凑成 `j-i` 的方案数
- 加上数字 i，就凑成 j
- 所以 `dp[j] += dp[j - i]`

**为什么逆序（从 N 到 i）？**
- 正序会重复使用同一个数字
- 逆序保证每个数字只用一次

### 逆序演示

```
N=6, M=3

初始：dp[0]=1, dp[1..6]=0

处理数字 1（逆序 j=6 到 1）：
j=6: dp[6] += dp[5]=0
j=5: dp[5] += dp[4]=0
j=4: dp[4] += dp[3]=0
j=3: dp[3] += dp[2]=0
j=2: dp[2] += dp[1]=0
j=1: dp[1] += dp[0]=1 → dp[1]=1

结果：dp[1]=1 (1), dp[2..6]=0

处理数字 2（逆序 j=6 到 2）：
j=6: dp[6] += dp[4]=0
j=5: dp[5] += dp[3]=0
j=4: dp[4] += dp[2]=0
j=3: dp[3] += dp[1]=1 → dp[3]=1 (1+2)
j=2: dp[2] += dp[0]=1 → dp[2]=1 (2)

结果：dp[1]=1, dp[2]=1, dp[3]=1

处理数字 3（逆序 j=6 到 3）：
j=6: dp[6] += dp[3]=1 → dp[6]=1 (1+2+3)
j=5: dp[5] += dp[2]=1 → dp[5]=1 (2+3)
j=4: dp[4] += dp[1]=1 → dp[4]=1 (1+3)
j=3: dp[3] += dp[0]=1 → dp[3]=2 (3 和 1+2)

最终 dp[6]=1，方案：1+2+3 ✅
```

## 边界情况

### 1. `N < M`
```
N=3, M=5
可用数字：1,2,3,4,5
实际能用的只有 1,2,3（4,5 太大）
结果：3 = 3 = 1+2 → 2 种
```

### 2. `M=1`
```
N=5, M=1
只能使用数字 1
但 1 只能用一次，最大和是 1
所以 N=5 无法凑成，dp[5]=0
```

### 3. `N=0`
题目没说 N=0，但如果有：
```
拆分方案：1 种（什么都不选）
dp[0]=1
```

### 4. 最大和 < N
```
N=10, M=3
最大和 = 1+2+3 = 6 < 10
无法凑成，dp[10]=0
```

## 优化：空间压缩

```cpp
// 一维 DP
vector<long long> dp(N + 1, 0);
dp[0] = 1;

for (int i = 1; i <= M; i++) {
    for (int j = N; j >= i; j--) {
        dp[j] += dp[j - i];
    }
}
```

**为什么可以压缩？**
- 每个数字只用一次，逆序更新
- 当前行依赖上一行的值
- 一维数组足够

## 完整代码（带注释）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int T;
    cin >> T;
    
    for (int caseNum = 0; caseNum < T; caseNum++) {
        int N, M;
        cin >> N >> M;
        
        // dp[j] = 用不重复数字凑成 j 的方案数
        vector<long long> dp(N + 1, 0);
        dp[0] = 1;  // 空集是 1 种方案
        
        // 遍历每个可用数字
        for (int i = 1; i <= M; i++) {
            // 逆序更新，保证每个数字只用一次
            for (int j = N; j >= i; j--) {
                dp[j] += dp[j - i];
            }
        }
        
        cout << "case #" << caseNum << ":" << endl;
        cout << dp[N] << endl;
    }
    
    return 0;
}
```

## 与完全背包对比

| 特点 | 完全背包（无限次） | 0/1 背包（一次） |
|------|-------------------|------------------|
| 更新方向 | 正序 | 逆序 |
| 数字可重复 | ✅ 是 | ❌ 否 |
| 本题 | ❌ | ✅ |

**正序 vs 逆序**：
```cpp
// 完全背包（可重复）
for (int j = i; j <= N; j++) {
    dp[j] += dp[j - i];
}

// 0/1 背包（不重复）← 本题
for (int j = N; j >= i; j--) {
    dp[j] += dp[j - i];
}
```

## 关键点总结

| 知识点 | 说明 |
|--------|------|
| 问题类型 | 0/1 背包（每个数字最多一次） |
| 状态定义 | dp[j] = 凑成 j 的方案数 |
| 初始化 | dp[0] = 1 |
| 更新方向 | 逆序（从 N 到 i） |
| 时间复杂度 | O(M × N) ≤ 20 × 50 = 1000 |
| 空间复杂度 | O(N) |

---

## 一句话总结

**每个数字最多用一次，按 0/1 背包逆序更新，dp[j] += dp[j-i]。**
