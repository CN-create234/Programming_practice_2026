# 1038. 一元多项式乘法
---

## 完整代码（先贴出来方便对照）

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<int> parse(string s) {
    vector<int> p(100);
    int i = 0, n = s.size();
    while (i < n) {
        int sign = 1, coef = 0, exp = 0;
        if (s[i] == '+') i++;
        else if (s[i] == '-') sign = -1, i++;
        
        bool hasNum = false;
        while (i < n && isdigit(s[i])) coef = coef*10 + s[i++]-'0', hasNum = true;
        if (!hasNum) coef = 1;
        coef *= sign;
        
        if (i < n && s[i] == 'x') {
            i++;
            if (i < n && s[i] == '^') {
                i++;
                while (i < n && isdigit(s[i])) exp = exp*10 + s[i++]-'0';
            } else exp = 1;
        }
        p[exp] += coef;
    }
    return p;
}

int main() {
    string line;
    while (getline(cin, line)) {
        if (line.empty()) continue;
        int pos = line.find(' ');
        vector<int> a = parse(line.substr(0, pos));
        vector<int> b = parse(line.substr(pos + 1));
        vector<int> ans(200);
        
        for (int i = 0; i < 100; i++)
            for (int j = 0; j < 100; j++)
                ans[i+j] += a[i] * b[j];
        
        bool first = true;
        for (int i = 199; i >= 0; i--)
            if (ans[i]) {
                if (!first) cout << " ";
                cout << ans[i];
                first = false;
            }
        cout << endl;
    }
}
```

### 2.2 主循环 - 遍历每个项

```cpp
    while (i < n) {
```
- 循环解析每个项（如 `+3x^2`、`-x`、`+1`）

---

### 2.3 读取符号

```cpp
        int sign = 1, coef = 0, exp = 0;
        if (s[i] == '+') i++;
        else if (s[i] == '-') sign = -1, i++;
```
- `sign`：当前项的符号，默认为 1（正）
- `coef`：系数，默认为 0
- `exp`：次数，默认为 0
- 如果遇到 `+`，跳过，符号保持正
- 如果遇到 `-`，符号变为 -1，跳过

**示例**：
```
"+3x^2" → sign=1, i指向 '3'
"-x"    → sign=-1, i指向 'x'
"x+1"   → 第一个项没有符号，sign=1
```

---

### 2.4 读取系数

```cpp
        bool hasNum = false;
        while (i < n && isdigit(s[i])) {
            coef = coef * 10 + (s[i] - '0');
            hasNum = true;
            i++;
        }
```
- `isdigit(s[i])`：判断字符是否是数字（'0'~'9'）
- 循环读取连续的数字字符，组成系数
- `hasNum`：标记是否读取到了数字

**示例**：
```
"123x"  → coef = 123, hasNum = true
"x"     → 没有数字，coef = 0, hasNum = false
```

---

### 2.5 处理系数省略

```cpp
        if (!hasNum) coef = 1;
        coef *= sign;
```
- 如果没有读取到数字（如 `x` 或 `-x`），系数默认为 1
- 乘以符号，得到最终系数

**示例**：
```
"x"   → coef = 1
"-x"  → coef = -1
"+x"  → coef = 1
"2x"  → coef = 2
```

---

### 2.6 读取指数

```cpp
        if (i < n && s[i] == 'x') {
            i++;
            if (i < n && s[i] == '^') {
                i++;
                while (i < n && isdigit(s[i])) {
                    exp = exp * 10 + (s[i] - '0');
                    i++;
                }
            } else {
                exp = 1;
            }
        }
```
- 如果当前字符是 `x`，说明这一项包含变量
- 跳过 `x`
- 如果后面是 `^`，跳过 `^`，读取指数数字
- 如果没有 `^`，指数默认为 1（`x` 表示 `x^1`）
- 如果不是 `x`（常数项），exp 保持为 0

**示例**：
```
"x^2"   → i指向x，跳过x，遇到^，读取2 → exp=2
"x"     → i指向x，跳过x，没有^ → exp=1
"1"     → 没有x → exp=0
```

---

### 2.7 存储系数

```cpp
        p[exp] += coef;
```
- 将系数累加到对应次数
- 例如 `3x^2` 会执行 `p[2] += 3`
- 合并同类项（如果同一次数有多个项）

---

### 2.8 返回结果

```cpp
    }
    return p;
}
```
- 循环结束，返回解析结果

|------|---------|------|------|
| 第1项 | 'x' | sign=1, coef=1, 有x无^ → exp=1 | p[1]=1 |
| 第2项 | '+' | 跳过 | |

| 第3项 | '+' | 跳过 | |
| 第3项 | '1' | coef=1, 无x → exp=0 | p[0]+=1 |

返回：`p[2]=1, p[1]=-3, p[0]=1` 表示 `x^2 - 3x + 1`

### 示例4：`"2"`

| 步骤 | 当前字符 | 操作 | 结果 |

### 4.2 跳过空行

- 如果行是空的，跳过（防止空行干扰）

### 4.3 分割两个多项式

```cpp
        int pos = line.find(' ');
        vector<int> a = parse(line.substr(0, pos));
        vector<int> b = parse(line.substr(pos + 1));
```
- `line.find(' ')`：找到空格的位置
- `line.substr(0, pos)`：取空格左边的字符串（第一个多项式）
- `line.substr(pos + 1)`：取空格右边的字符串（第二个多项式）
- 分别解析两个多项式

---

### 4.4 初始化结果数组

```cpp
        vector<int> ans(200);
```
- 创建大小为 200 的结果数组
- 最高次数：100 + 100 - 2 = 198，200 足够

---

### 4.5 多项式乘法

```cpp
        for (int i = 0; i < 100; i++)
            for (int j = 0; j < 100; j++)
                ans[i + j] += a[i] * b[j];
```
- 遍历第一个多项式的所有次数 i
- 遍历第二个多项式的所有次数 j
- `a[i] * b[j]`：系数相乘
- `i + j`：次数相加
- 累加到 `ans[i+j]`

**示例**：`(2x)(3x^2) = 6x^3`
- `a[1] = 2, b[2] = 3`
- `ans[1+2] += 2*3 = 6`

- `first`：标记是否是第一个输出的系数（用于控制空格）
- 从高次到低次遍历（199 到 0）

- 第一个系数前面不加空格，后面的加空格

## 5. 完整执行流程图

```
开始
  ↓
读取一行 line
  ↓
line 为空？ → 是 → 继续读取下一行
  ↓ 否

  ↓

  ↓

  ↓

  ↓
ans 初始化为全 0
  ↓

  ↓
从高次到低次输出非零系数
  ↓
换行
  ↓
继续读取下一行（直到 EOF）
```

## 6. 样例执行过程

### 输入：`x+1 x-1`

**解析 a = "x+1"**：
```
a[1] = 1  (x)
a[0] = 1  (1)
```

**解析 b = "x-1"**：
```
b[1] = 1   (x)
b[0] = -1  (-1)
```

**乘法**：
```
i=1, j=1: ans[2] += 1*1 = 1
i=1, j=0: ans[1] += 1*(-1) = -1
i=0, j=1: ans[1] += 1*1 = 0
i=0, j=0: ans[0] += 1*(-1) = -1
```

**结果**：
```
ans[2] = 1
ans[1] = 0
ans[0] = -1
```

**输出**：`1 -1`

|--------|------|------|
