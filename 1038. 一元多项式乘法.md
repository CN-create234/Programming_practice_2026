## 一、整体思路：把多项式当作"数组"来处理

### 核心思想
**一个多项式 = 一个数组**
- 数组的**下标** = 次数
- 数组的**值** = 系数

比如：
```
多项式：3x² + 2x + 1
用数组表示：p[2]=3, p[1]=2, p[0]=1
```

这样做的好处：
- 多项式加法：直接对应位置相加
- 多项式乘法：a[i] × b[j] 贡献到 ans[i+j]

## 三、为什么用数组表示多项式？

### 传统方式 vs 数组方式

| 方式 | 存储 | 乘法复杂度 |
|------|------|-----------|
| 传统：链表/向量存储项 | `[(3,2), (2,1), (1,0)]` | 需要匹配次数，麻烦 |
| 数组方式 | `p[2]=3, p[1]=2, p[0]=1` | 直接按次数索引，简单 |

**数组方式的优势**：
- 乘法：直接双重循环，i+j 就是结果次数
- 不需要排序、不需要合并同类项（自动累加）
- 代码极其简洁

## 四、parse 函数：把字符串变成数组

### 输入 → 输出
```
"x²-3x+1"  →  [0, 1, -3, 1]
               ↑  ↑   ↑   ↑
             p[0] p[1] p[2] p[3]
```

### 解析的四个动作
```
字符串：  "x² - 3x + 1"
            ↓   ↓    ↓    ↓
动作1：   读符号  →  + 或 -
动作2：   读系数  →  数字 或 默认1
动作3：   读x    →  有x表示有变量
动作4：   读指数  →  ^数字 或 默认1
```

**本质**：从字符串中提取每个项的 `(系数, 次数)`，填入数组。

## 五、乘法部分：核心就两行

```cpp
for (int i = 0; i < 100; i++)
    for (int j = 0; j < 100; j++)
        ans[i+j] += a[i] * b[j];
```

### 数学原理
两个多项式相乘：
```
(a₀ + a₁x + a₂x² + ...) × (b₀ + b₁x + b₂x² + ...)
= Σ a_i·b_j · x^(i+j)
```

**代码直接对应这个公式**：
- 遍历所有 i 和 j
- 系数相乘：`a[i] * b[j]`
- 次数相加：`i + j`
- 累加到结果：`ans[i+j] += ...`

---

## 六、输出部分：从高次到低次

```cpp
for (int i = 199; i >= 0; i--)
    if (ans[i]) cout << ans[i];
```

**为什么从高到低？**
- 题目要求"根据次数由高到低顺序输出"
- 多项式通常也习惯这样写：`3x² + 2x + 1`

**为什么用 199？**
- 两个多项式最高都是 99 次（因为数组大小 100）
- 乘积最高：99 + 99 = 198
- 用 200 确保覆盖所有

## 七、整体数据流

```
输入行: "x+1 x-1"
    ↓
分割: "x+1" 和 "x-1"
    ↓
parse("x+1"):  a[1]=1, a[0]=1
parse("x-1"):  b[1]=1, b[0]=-1
    ↓
乘法:
  ans[2] = 1×1 = 1
  ans[1] = 1×(-1) + 1×1 = 0
  ans[0] = 1×(-1) = -1
    ↓
输出: ans[2]=1, ans[1]=0(跳过), ans[0]=-1
    ↓
结果: "1 -1"
```

## 九、为什么这样设计？

| 设计选择 | 原因 |
|----------|------|
| 用数组存储 | 乘法时 i+j 直接得到次数，无需匹配 |
| 固定大小 100 | 题目保证 n < 50，100 足够 |
| 忽略零系数 | 输出时跳过 ans[i]==0 |
| 字符串解析 | 处理复杂输入格式的唯一方式 |

## 十、一句话总结

**把一个多项式当成"次数→系数"的映射表（数组），乘法就是两个映射表按规则合并，简单、快速、代码少。**

## 完整代码（先贴出来方便对照）

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<int> parse(string s) {
    vector<int> p(100);
    int i = 0, n = s.size();
    while (i < n) {
        int sign = 1, coef = 0, exp = 0;
        if (s[i] == '+') i++;
        else if (s[i] == '-') sign = -1, i++;
        
        bool hasNum = false;
        while (i < n && isdigit(s[i])) coef = coef*10 + s[i++]-'0', hasNum = true;
        if (!hasNum) coef = 1;
        coef *= sign;
        
        if (i < n && s[i] == 'x') {
            i++;
            if (i < n && s[i] == '^') {
                i++;
                while (i < n && isdigit(s[i])) exp = exp*10 + s[i++]-'0';
            } else exp = 1;
        }
        p[exp] += coef;
    }
    return p;
}

int main() {
    string line;
    while (getline(cin, line)) {
        if (line.empty()) continue;
        int pos = line.find(' ');
        vector<int> a = parse(line.substr(0, pos));
        vector<int> b = parse(line.substr(pos + 1));
        vector<int> ans(200);
        
        for (int i = 0; i < 100; i++)
            for (int j = 0; j < 100; j++)
                ans[i+j] += a[i] * b[j];
        
        bool first = true;
        for (int i = 199; i >= 0; i--)
            if (ans[i]) {
                if (!first) cout << " ";
                cout << ans[i];
                first = false;
            }
        cout << endl;
    }
}
```

### 2.2 主循环 - 遍历每个项

```cpp

```
- 循环解析每个项（如 `+3x^2`、`-x`、`+1`）

---

### 2.3 读取符号

```cpp
        int sign = 1, coef = 0, exp = 0;
        if (s[i] == '+') i++;
        else if (s[i] == '-') sign = -1, i++;
```
- `sign`：当前项的符号，默认为 1（正）
- `coef`：系数，默认为 0
- `exp`：次数，默认为 0
- 如果遇到 `+`，跳过，符号保持正
- 如果遇到 `-`，符号变为 -1，跳过

**示例**：
```
"+3x^2" → sign=1, i指向 '3'
"-x"    → sign=-1, i指向 'x'
"x+1"   → 第一个项没有符号，sign=1
```

---

### 2.4 读取系数

```cpp
        bool hasNum = false;
        while (i < n && isdigit(s[i])) {
            coef = coef * 10 + (s[i] - '0');
            hasNum = true;
            i++;
        }
```
- `isdigit(s[i])`：判断字符是否是数字（'0'~'9'）
- 循环读取连续的数字字符，组成系数
- `hasNum`：标记是否读取到了数字

**示例**：
```
"123x"  → coef = 123, hasNum = true
"x"     → 没有数字，coef = 0, hasNum = false
```

---

### 2.5 处理系数省略

```cpp
        if (!hasNum) coef = 1;
        coef *= sign;
```
- 如果没有读取到数字（如 `x` 或 `-x`），系数默认为 1
- 乘以符号，得到最终系数

**示例**：
```
"x"   → coef = 1
"-x"  → coef = -1
"+x"  → coef = 1
"2x"  → coef = 2
```

---

### 2.6 读取指数

```cpp
        if (i < n && s[i] == 'x') {
            i++;
            if (i < n && s[i] == '^') {
                i++;
                while (i < n && isdigit(s[i])) {
                    exp = exp * 10 + (s[i] - '0');
                    i++;
                }
            } else {
                exp = 1;
            }
        }
```
- 如果当前字符是 `x`，说明这一项包含变量
- 跳过 `x`
- 如果后面是 `^`，跳过 `^`，读取指数数字
- 如果没有 `^`，指数默认为 1（`x` 表示 `x^1`）
- 如果不是 `x`（常数项），exp 保持为 0

**示例**：
```
"x^2"   → i指向x，跳过x，遇到^，读取2 → exp=2
"x"     → i指向x，跳过x，没有^ → exp=1
"1"     → 没有x → exp=0
```

---

### 2.7 存储系数

```
- 将系数累加到对应次数
- 例如 `3x^2` 会执行 `p[2] += 3`
- 合并同类项（如果同一次数有多个项）

    return p;
}
```

```

## 6. 样例执行过程

```
