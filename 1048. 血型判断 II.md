## 一、问题背景

### 血型遗传规则

| 血型 | 基因型 | 说明 |
| :--- | :--- | :--- |
| **A型** | AA 或 AO | 显性基因 A |
| **B型** | BB 或 BO | 显性基因 B |
| **AB型** | AB | A和B共显性 |
| **O型** | OO | 隐性基因 O |

### 遗传规则
- 孩子从父母各继承一个等位基因
- A 和 B 是显性，O 是隐性
- 显性基因会掩盖隐性基因的表现

## 三、函数详解

### 1. `get_genotypes()` —— 获取血型的基因型

```cpp
vector<string> get_genotypes(const string& blood) {
    if (blood == "A") return {"AA", "AO"};     // A型可能：AA或AO
    if (blood == "B") return {"BB", "BO"};     // B型可能：BB或BO
    if (blood == "AB") return {"AB"};          // AB型只能是AB
    if (blood == "O") return {"OO"};           // O型只能是OO
    return {};
}
```

#### 为什么 A 型有两种基因型？

| 父方等位基因 | 母方等位基因 | 孩子血型 |
| :--- | :--- | :--- |
| A + A | → | A型（AA）|
| A + O | → | A型（AO）|

**显性基因 A 会掩盖隐性基因 O**，所以 AO 表现型是 A 型。

---

### 2. `get_blood_type()` —— 基因型转血型

```cpp
string get_blood_type(char g1, char g2) {
    string s = "";
    s += min(g1, g2);  // 保证字母顺序
    s += max(g1, g2);
    
    if (s == "AA" || s == "AO") return "A";
    if (s == "BB" || s == "BO") return "B";
    if (s == "AB") return "AB";
    if (s == "OO") return "O";
    return "";
}
```

#### 为什么要 `min` 和 `max`？

```
// 输入可能是 (A, O) 或 (O, A)
// 统一排序为 "AO"，方便比较
```

#### 遗传规则验证

| 基因组合 | 排序后 | 血型 | 说明 |
| :--- | :--- | :--- | :--- |
| A + A | AA | A | 纯合A型 |
| A + O | AO | A | A显性，O隐性 |
| B + B | BB | B | 纯合B型 |
| B + O | BO | B | B显性，O隐性 |
| A + B | AB | AB | A和B共显性 |
| O + O | OO | O | 纯合隐性 |

---

### 3. `get_child_bloods()` —— 计算孩子可能血型

```cpp
set<string> get_child_bloods(const string& p1_geno, const string& p2_geno) {
    set<string> res;  // 使用 set 自动去重并排序
    
    for (char a : p1_geno) {      // 遍历父方的两个等位基因
        for (char b : p2_geno) {  // 遍历母方的两个等位基因
            res.insert(get_blood_type(a, b));
        }
    }
    
    return res;
}
```

#### 示例：父亲 AO × 母亲 BO

```
父亲基因型: A O
母亲基因型: B O

组合：
A + B = AB
A + O = A
O + B = B
O + O = O

结果：{A, B, AB, O}（所有血型都可能！）
```

### 4. `print_blood_set()` —— 格式化输出

```cpp
void print_blood_set(const set<string>& s) {
    if (s.empty()) {
        cout << "impossible";
    } else if (s.size() == 1) {
        cout << *s.begin();
    } else {
        cout << "{";
        bool first = true;
        for (const string& b : s) {
            if (!first) cout << ",";
            cout << b;
            first = false;
        }
        cout << "}";
    }
}
```

#### 输出格式示例

| 集合大小 | 输出示例 |
| :--- | :--- |
| 0（空集） | `impossible` |
| 1 | `A` |
| 2 | `{A,B}` |
| 3 | `{A,B,AB}` |
| 4 | `{A,B,AB,O}` |

## 四、主函数详解

### 输入格式
```
父亲血型 母亲血型 孩子血型
```
其中 `?` 表示未知

### 三种情况

1. **孩子未知**：`A B ?` → 输出孩子可能的血型
2. **父亲未知**：`? B A` → 输出父亲可能的血型
3. **母亲未知**：`A ? AB` → 输出母亲可能的血型

---

### 情况1：孩子未知（`c == "?"`）

```cpp
if (c == "?") {

    auto f_genos = get_genotypes(f);
    auto m_genos = get_genotypes(m);
    
    // 遍历所有可能的父母基因型组合
    for (const auto& fg : f_genos) {
        for (const auto& mg : m_genos) {
            auto cb = get_child_bloods(fg, mg);
            possible_c.insert(cb.begin(), cb.end());
        }
    }
    
    cout << f << " " << m << " ";
    print_blood_set(possible_c);
    cout << endl;
}
```

#### 示例：父亲 A 型，母亲 B 型

```
父亲 A 型 → 基因型 {AA, AO}
母亲 B 型 → 基因型 {BB, BO}

枚举所有组合：
AA × BB → {AB}
AA × BO → {A, AB}
AO × BB → {B, AB}
AO × BO → {A, B, AB, O}

并集：{A, B, AB, O}（所有血型都有可能）
```
