# EOJ-1018 分数加法 - 完整解析

---

## 一、题目理解

### 问题描述

给定 n 个分数，计算它们的和，结果以最简分数 `n/m` 形式输出。

### 输入格式

- 第一行：整数 n，表示分数个数
- 接下来 n 行：每行一个分数，格式为 `n/m`

### 输出格式

- 一行：最简分数 `n/m`

### 数据范围

- 所有数据在 int 范围内
- 不存在负分数

---

## 二、核心数学原理

### 分数加法公式

```
a/b + c/d = (a×d + c×b) / (b×d)
```

### 约分

```
化简分数：分子和分母同时除以它们的最大公约数（gcd）
```

### 示例

```
1/3 + 1/2 = (1×2 + 1×3) / (3×2) = 5/6
```

---

## 三、解题思路

```
1. 读入 n
2. 初始结果：0/1
3. 对于每个分数：
   a. 读取分数 n/m
   b. 通分相加：分子 = 结果分子 × m + n × 结果分母
   c. 分母 = 结果分母 × m
   d. 约分
4. 输出结果
```

---

## 四、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int gcd(int a, int b) {
    while (b != 0) {
        int t = a % b;
        a = b;
        b = t;
    }
    return a;
}

int main() {
    int n;
    cin >> n;

    int num = 0;   // 分子
    int den = 1;   // 分母

    for (int i = 0; i < n; i++) {
        string s;
        cin >> s;

        // 分割分子和分母
        int pos = s.find('/');
        int a = stoi(s.substr(0, pos));
        int b = stoi(s.substr(pos + 1));

        // 通分相加：num/den + a/b = (num*b + a*den) / (den*b)
        num = num * b + a * den;
        den = den * b;

        // 约分
        int g = gcd(abs(num), abs(den));
        num /= g;
        den /= g;
    }

    cout << num << "/" << den << endl;

    return 0;
}
```

读取一个分数字符串，如 `"1/3"`。

### 6. 分割分子和分母

```cpp
        int pos = s.find('/');
        int a = stoi(s.substr(0, pos));
        int b = stoi(s.substr(pos + 1));
```

| 代码 | 含义 |
|------|------|
| `s.find('/')` | 查找 `/` 的位置 |
| `s.substr(0, pos)` | 从开头到 `/` 前 → 分子 |
| `s.substr(pos + 1)` | 从 `/` 后到末尾 → 分母 |
| `stoi()` | 字符串转整数 |

**示例：** `"1/3"`
```
pos = 1
a = stoi("1") = 1
b = stoi("3") = 3
```

---

### 7. 通分相加

```cpp
        num = num * b + a * den;
        den = den * b;
```

**公式：**
```
当前结果 = num/den
新分数 = a/b

相加 = num/den + a/b = (num×b + a×den) / (den×b)
```

**示例：** `1/2 + 1/3`

```
num=1, den=2, a=1, b=3
num = 1×3 + 1×2 = 5
den = 2×3 = 6
结果 = 5/6 ✓
```

---

### 8. 约分

```cpp
        int g = gcd(abs(num), abs(den));
        num /= g;
        den /= g;
```

- 计算最大公约数
- 分子分母同时除以 gcd
- `abs()` 防止负数（虽然题目没有负分数）

**示例：** `2/4`
```
g = gcd(2, 4) = 2
num = 2/2 = 1
den = 4/2 = 2
结果 = 1/2 ✓
```

---

### 9. 输出

```cpp
    cout << num << "/" << den << endl;
```

输出最简分数。

## 十、处理整数结果

如果结果是整数（如 `3`），输出应该是 `3/1`。

代码中 `cout << num << "/" << den` 会自动输出 `3/1`，符合要求。

## 十一、总结

| 要点 | 说明 |
|------|------|
| **核心** | 通分相加，然后约分 |
| **通分公式** | `num/den + a/b = (num×b + a×den) / (den×b)` |
| **约分** | 用 gcd 约分到最简 |
| **数据范围** | int 范围内，中间乘法用 long long 更安全 |
| **输入格式** | `n/m`，用 `find('/')` 分割 |

**核心思想：用通分法累加分数，每次相加后用 gcd 约分。**
