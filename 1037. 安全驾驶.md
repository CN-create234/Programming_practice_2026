## 题目场景

### 物理场景
想象一条**笔直的单车道测试道路**，所有车都**同向**行驶（都朝目的地开）。

- 有 **n + 1** 辆车
- 其中 **1 辆是最后一辆车**（在最后面），还有 **n 辆前车**（在前面）
- 所有车**同时出发**
- 每辆车有自己的**初始位置**和**恒定速度**

### 关键规则
> **若后面的小车追上前面的小车，为了安全则必须降速到与前车相同的速度。**

- 如果后车比前车快，后车会追上前车
- 一旦追上，后车必须**减速**到前车的速度
- 减速后，两车以相同速度一起行驶
- 如果前面还有更慢的车，可能会连锁减速

## 具体例子

### 例子1：理解基本规则
假设：
- 最后一辆车：速度 100 km/h，在位置 0
- 前车：速度 60 km/h，在最后一辆车前方 120 km
- 目的地：距离最后一辆车 300 km

```
位置 0           120           300
|---------------|-------------|
最后一辆车      前车         目的地
速度 100        速度 60
```

**过程**：
1. 两车同时出发
2. 最后一辆车速度 100 > 前车速度 60，会追上前车
3. 追上时间：120 / (100-60) = 3 小时
4. 追上时位置：100 × 3 = 300 km（正好到目的地！）
5. 所以最后一辆车全程无需减速，刚好到目的地时追上

**结果**：最后一辆车可以保持 100 km/h ✅

---

### 例子2：速度太快会怎样？
如果最后一辆车速度 120 km/h：
```
追上时间：120 / (120-60) = 2 小时
追上位置：120 × 2 = 240 km（还没到目的地！）
```
- 在 240 km 处追上前车
- 被迫减速到 60 km/h
- **违反了"全程匀速"的要求** ❌

所以速度 120 不行。

---

### 例子3：有多辆车
```
目的地距离：100 km
前车1：位置 80 km，速度 100 km/h
前车2：位置 70 km，速度 10 km/h
```

- 前车2在 70 km 处，速度很慢（10 km/h）
- 最后一辆车要想不减速，**必须不能追上前车2**
- 即使前车1允许更高速度，前车2也限制了速度

**答案**：最后一辆车速度 ≤ 33.33 km/h

## 题目真正的意思

### "最后一辆小车不想中途降速，希望全程匀速行驶"
- 最后一辆车**从起点到终点**速度始终保持不变
- 不能因为追上前车而减速

### "找出最后一辆小车在满足条件（全程匀速且保证安全）的情况下最大可能的速度"
- 求一个**最大的速度 v**
- 使得最后一辆车以速度 v 行驶时：
  1. 全程匀速（不减速）
  2. 保证安全（不撞上任何前车）

### "保证安全"的含义
**可以追上前车，但不能追上的时候前车还没到达目的地。**

- 如果追上前车时，前车已经到终点了，那就安全（因为前车不在路上了）
- 如果追上前车时，前车还在路上，那就必须减速（不允许）

## 数学模型推导

### 对每一辆前车 i

设：
- `d`：最后一辆车到目的地的距离
- `k_i`：前车 i 在最后一辆车前方的距离
- `s_i`：前车 i 的速度
- `v`：最后一辆车的速度（未知）

### 前车到达目的地的时间
```
T_i = (d - k_i) / s_i
```
- 前车 i 离目的地的距离：`d - k_i`
- 除以速度 `s_i`，得到到达时间

### 最后一辆车追上前车的时间
如果 `v > s_i`：
```
t_i = k_i / (v - s_i)
```
- 初始距离差：`k_i`
- 速度差：`v - s_i`
- 追上时间：距离差 / 速度差

### 安全条件
要保证安全，**要么追不上，要么追上的时候前车已经到达**：

**情况1：追不上**
```
v <= s_i
```

**情况2：追得上，但前车已到达**
```
t_i >= T_i   （追上的时候，前车已经到达）
k_i / (v - s_i) >= (d - k_i) / s_i
```

### 整理情况2
```
k_i / (v - s_i) >= (d - k_i) / s_i
k_i * s_i >= (v - s_i)(d - k_i)
k_i * s_i >= v(d - k_i) - s_i(d - k_i)
k_i * s_i + s_i(d - k_i) >= v(d - k_i)
s_i * (k_i + d - k_i) >= v(d - k_i)
s_i * d >= v(d - k_i)
v <= s_i * d / (d - k_i)
```

### 最终约束
结合两种情况：
```
v <= s_i * d / (d - k_i)  对所有 i
```

### 求最大值
```
v_max = min(s_i * d / (d - k_i))
```

## 直观理解

### 公式的物理意义
```
limit = s_i * d / (d - k_i)
```

这个公式可以理解为：
- 如果前车速度是 `s_i`，距离终点 `d - k_i`
- 前车到达终点需要时间：`(d - k_i) / s_i`
- 最后一辆车必须在这个时间之后才能到达前车的位置
- 所以速度不能超过 `k_i / ((d - k_i) / s_i) = s_i * k_i / (d - k_i)`？
- 等等，这里有点绕...

### 更直观的理解

换个角度：最后一辆车要保证在任何前车到达目的地之前，不追上前车。

对于前车 i：
- 前车到达目的地的时间是 `T_i = (d - k_i) / s_i`
- 最后一辆车在时间 `T_i` 时，行驶了 `v * T_i = v * (d - k_i) / s_i`
- 此时前车在位置 `d`（已经到达）
- 为了不追上前车，最后一辆车在时间 `T_i` 时，位置必须 ≤ `d`
- 即：`v * (d - k_i) / s_i ≤ d`
- 整理：`v ≤ s_i * d / (d - k_i)`

**这样理解更清晰！**

| 要点 | 说明 |
|------|------|
| 全程匀速 | 最后一辆车速度不变 |
| 保证安全 | 不能追上路上的前车 |
| 允许追上已到终点的前车 | 前车到终点后就不在路上了 |

## 变量命名建议

你的代码中使用了 `d` 和 `D`，容易混淆。建议使用更清晰的命名：

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int totalDist;  // 最后一辆车到目的地的距离
    cin >> totalDist;

    int n;
    cin >> n;

    double ans = 1e18;

    for (int i = 0; i < n; i++) {
        int frontDist, speed;  // 前车距离和速度
        cin >> frontDist >> speed;

        // 速度限制公式：limit = speed * totalDist / (totalDist - frontDist)
        double limit = (double)speed * totalDist / (totalDist - frontDist);
        ans = min(ans, limit);
    }

    cout << fixed << setprecision(6) << ans << endl;

    return 0;
}
```

这样更清晰，不容易混淆。
