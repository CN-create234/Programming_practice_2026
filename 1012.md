# 1012. Problem A+B (Big Integer)大整数加法 - 逐行详细解释

单点时限: 2.0 sec  
内存限制: 256 MB  

给出两个正整数 A 和 B，计算 A + B。

注意：A、B 的位数不超过 500 位。

---

## 输入格式

测试用例包含若干行。每行包含两个正整数 A 和 B。

---

## 输出格式

对于每一行输入，输出一行 A + B 的结果。

---

### 样例

#### 输入
2 3  
1231231231823192 123123123123123123  
1000000000000000001 1  

#### 输出
5  
1354354354946315  
1000000000000000002

## 一、完整代码

```cpp
#include <bits/stdc++.h>
using namespace std;

string addBigInteger(string a, string b) {
    // 反转字符串，让个位在索引 0
    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());

    string result = "";
    int carry = 0;
    int maxLen = max(a.size(), b.size());

    for (int i = 0; i < maxLen; i++) {
        int digitA = (i < (int)a.size()) ? a[i] - '0' : 0;
        int digitB = (i < (int)b.size()) ? b[i] - '0' : 0;

        int sum = digitA + digitB + carry;
        result += (sum % 10) + '0';
        carry = sum / 10;
    }

    // 处理最后的进位
    if (carry > 0) {
        result += carry + '0';
    }

    // 反转回来
    reverse(result.begin(), result.end());
    return result;
}

int main() {
    string a, b;
    while (cin >> a >> b) {
        cout << addBigInteger(a, b) << endl;
    }
    return 0;
}
```

### 第三部分：反转字符串

```cpp
    // 反转字符串，让个位在索引 0
    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());
```

| 代码 | 含义 |
|------|------|
| `reverse(a.begin(), a.end())` | 将字符串 `a` 反转 |
| `reverse(b.begin(), b.end())` | 将字符串 `b` 反转 |

**为什么需要反转？**

```
原始字符串："123"
索引：       0  1  2
字符：       '1''2''3'
含义：       百 十 个

反转后：     "321"
索引：       0  1  2
字符：       '3''2''1'
含义：       个 十 百
```

反转后，索引 0 对应个位，方便从低位开始逐位相加。

### 第五部分：逐位相加（核心）

```cpp
    for (int i = 0; i < maxLen; i++) {
        int digitA = (i < (int)a.size()) ? a[i] - '0' : 0;
        int digitB = (i < (int)b.size()) ? b[i] - '0' : 0;

        int sum = digitA + digitB + carry;
        result += (sum % 10) + '0';
        carry = sum / 10;
    }
```

**逐行解释：**

| 行 | 代码 | 含义 |
|----|------|------|
| 1 | `for (int i = 0; i < maxLen; i++)` | 从低位到高位遍历 |
| 2 | `(i < (int)a.size()) ? a[i] - '0' : 0` | 如果 `i` 在 `a` 范围内，取该位数字；否则取 0 |
| 3 | `(i < (int)b.size()) ? b[i] - '0' : 0` | 如果 `i` 在 `b` 范围内，取该位数字；否则取 0 |
| 4 | `int sum = digitA + digitB + carry` | 当前位相加（含进位） |
| 5 | `result += (sum % 10) + '0'` | 当前位结果（数字转字符）追加到 result |
| 6 | `carry = sum / 10` | 计算进位 |

**示例：** `a = "123"`, `b = "45"`

反转后：`a="321"`, `b="54"`

```
i=0: digitA=3, digitB=4, carry=0
     sum=7, result="7", carry=0

i=1: digitA=2, digitB=5, carry=0
     sum=7, result="77", carry=0

i=2: digitA=1, digitB=0, carry=0
     sum=1, result="771", carry=0
```

### 第六部分：处理最后的进位

```cpp
    // 处理最后的进位
    if (carry > 0) {
        result += carry + '0';
    }
```

| 代码 | 含义 |
|------|------|
| `if (carry > 0)` | 如果最后还有进位 |
| `result += carry + '0'` | 将进位追加到结果末尾 |

**示例：** `a = "999"`, `b = "1"`

```
反转后：a="999", b="1"

i=0: 9+1=10 → result="0", carry=1
i=1: 9+0+1=10 → result="00", carry=1
i=2: 9+0+1=10 → result="000", carry=1
循环结束，carry=1
result += '1' → result="0001"
```

### 第七部分：反转返回

```cpp
    // 反转回来
    reverse(result.begin(), result.end());
    return result;
```

| 代码 | 含义 |
|------|------|
| `reverse(result.begin(), result.end())` | 反转结果字符串 |
| `return result` | 返回结果 |

**为什么需要再次反转？**

`result` 中存储的顺序是：**个位 → 最高位**（低位在前）

需要反转成：**最高位 → 个位**（高位在前）

**示例：** `result = "771"`（个位7，十位7，百位1）

```
反转后：result = "177"（百位1，十位7，个位7）
```

## 五、使用优化（去前导零）

```cpp
string addBigInteger(string a, string b) {
    // 去掉前导零（可选）
    while (a.size() > 1 && a[0] == '0') a.erase(0, 1);
    while (b.size() > 1 && b[0] == '0') b.erase(0, 1);

    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());

    string result = "";
    int carry = 0;
    int maxLen = max(a.size(), b.size());

    for (int i = 0; i < maxLen; i++) {
        int digitA = (i < (int)a.size()) ? a[i] - '0' : 0;
        int digitB = (i < (int)b.size()) ? b[i] - '0' : 0;
        int sum = digitA + digitB + carry;
        result += (sum % 10) + '0';
        carry = sum / 10;
    }

    if (carry) result += carry + '0';

    reverse(result.begin(), result.end());
    return result;
}
```

## 六、总结

| 步骤 | 说明 |
|------|------|
| **反转** | 让个位在索引 0，方便从低位相加 |
| **逐位相加** | 从索引 0 开始，逐位相加并处理进位 |
| **位数补零** | 短数高位补 0 |
| **处理进位** | 循环结束后检查 carry |
| **再次反转** | 恢复高位在前顺序 |

**核心思想：** 用字符串模拟竖式加法，从个位开始逐位相加并处理进位。
